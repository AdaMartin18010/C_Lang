## 🏆 贡献者认可计划

### 徽章体系

贡献者可以通过不同类型的贡献获得徽章：

| 徽章 | 名称 | 获得条件 |
|:---:|:---|:---|
| 📝 | **文档大师** | 贡献 5+ 篇高质量文档 |
| 💻 | **代码工匠** | 贡献 20+ 个可运行代码示例 |
| 🔧 | **Bug猎手** | 提交 10+ 个 Bug 报告并被确认 |
| 🌐 | **翻译使者** | 完成 3+ 篇文档翻译 |
| 🎨 | **思维设计师** | 创建 3+ 个思维表征工具 |
| 🏗️ | **架构师** | 成功提案并实现 1+ 个新模块 |
| 🔬 | **学术先锋** | 贡献形式化/理论深度内容 |
| 🛡️ | **安全卫士** | 贡献安全标准相关内容 |
| 🌟 | **社区之星** | 协助审核 10+ 个 PR |
| 💡 | **创新者** | 提出并实现创新功能 |

### 晋升路径

```
🌱 新手贡献者 → 🌿 活跃贡献者 → 🌳 核心贡献者 → 👑 维护者
     (1-3)          (4-10)           (11-25)         (26+)
```

**各等级权益**：

| 等级 | PR审核权 | Issue标签管理 | 发布权限 | 决策参与 |
|-----|:-------:|:------------:|:-------:|:-------:|
| 🌱 新手 | - | - | - | - |
| 🌿 活跃 | - | - | - | 建议权 |
| 🌳 核心 | ✅ | ✅ | - | 投票权 |
| 👑 维护者 | ✅ | ✅ | ✅ | 决策权 |

## 🚀 如何获得认可

想要出现在荣誉榜上？以下是一些建议：

### 快速开始

1. **从简单的开始**
   - 修复拼写错误
   - 更新失效链接
   - 改进文档格式

2. **分享你的专长**
   - 添加你熟悉的领域内容
   - 改进现有代码示例
   - 创建思维表征工具

3. **参与社区**
   - 协助审核 PR
   - 回答其他用户的问题
   - 提出建设性建议

### 提升贡献质量

- 遵循 [内容风格指南](CONTENT_STYLE_GUIDE.md)
- 使用 PR 模板提交变更
- 确保代码示例可运行
- 积极参与讨论

### 长期贡献

- 认领并维护特定模块
- 参与季度内容评审
- 帮助新人入门
- 推动社区建设
