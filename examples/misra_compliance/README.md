# MISRA C:2023 代码对照示例

可编译的 C 代码示例，展示 MISRA C:2023 关键规则的**违规写法**与**合规写法**。

## 文件索引

| 文件 | 覆盖规则类别 | 规则数 | 关键规则 |
|------|------------|--------|---------|
| `type_safety.c` | 类型转换与表达式 | 5 | 10.1, 10.3, 10.4, 10.8, 12.2 |
| `memory_safety.c` | 指针与数组 | 7 | 9.1, 18.1, 18.2, 18.3, 18.6, 18.7, 22.1 |
| `control_flow.c` | 控制流与循环 | 7 | 14.1, 14.4, 15.1, 15.5, 15.6, 15.7, 16.1, 17.2 |
| `function_safety.c` | 函数与资源 | 7 | 8.2, 17.1, 17.5, 17.7, 21.3, 21.6, 22.5 |

## 编译运行

```bash
make
./type_safety
./memory_safety
./control_flow
./function_safety
```

## 使用建议

1. **违规代码**用于理解规则禁止的行为模式
2. **合规代码**展示 MISRA 推荐的惯用写法
3. 所有示例均可在启用 `-Wconversion` 时编译通过（合规版本）
4. 建议配合静态分析工具（如 PC-lint, Helix QAC, Cppcheck）验证

## 规则分类速查

| 类别 | 规则范围 | 核心思想 |
|------|---------|---------|
| 类型安全 | Rule 10.x | 避免隐式转换，显式控制类型变化 |
| 初始化 | Rule 9.x | 使用前初始化，避免未定义值 |
| 指针/数组 | Rule 18.x | 边界检查，避免越界和悬空指针 |
| 控制流 | Rule 14-16.x | 清晰可预测的控制结构 |
| 函数 | Rule 17.x | 参数约束，避免递归，使用返回值 |
| 资源管理 | Rule 21-22.x | 显式分配/释放，避免泄漏 |

> 完整 220 条规则请参考 `knowledge/01_Core_Knowledge_System/09_Safety_Standards/MISRA_C_2023/`
