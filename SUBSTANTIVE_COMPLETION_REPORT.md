## 🛠️ 实践工具链

### 完整分析流程

```bash
# 编译链分析
gcc -E hello.c -o hello.i        # 预处理
gcc -S hello.i -o hello.s        # 编译
gcc -c hello.s -o hello.o        # 汇编
gcc hello.o -o hello             # 链接

# ELF分析
readelf -a hello                 # 完整ELF信息
readelf -d hello                 # 动态段
objdump -d hello                 # 反汇编
nm hello                         # 符号表

# 动态链接分析
ldd hello                        # 依赖库
LD_DEBUG=libs ./hello            # 库加载过程
LD_DEBUG=bindings ./hello        # 符号绑定

# 加载分析
strace -e execve,mmap ./hello    # 系统调用
cat /proc/$$/maps                # 内存映射

# 性能分析
perf record ./hello              # 性能记录
perf report                      # 性能报告
perf top                         # 实时分析

# eBPF追踪
sudo bpftrace -e 'kprobe:do_execve { printf("exec: %s\n", str(arg0)); }'
```
