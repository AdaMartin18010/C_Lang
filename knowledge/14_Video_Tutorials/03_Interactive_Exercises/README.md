## 模块概述

交互式练习平台是视频教程的配套实践系统，提供即时的代码编写、编译、评测反馈。每个练习都与视频课程对应，帮助学习者巩固知识、检验理解、提升编程能力。

### 平台特性

- **即时反馈**: 提交后立即获得结果
- **渐进难度**: 从填空题到完整项目
- **多维度评测**: 正确性、性能、代码质量
- **学习路径**: 智能推荐下一个练习
- **社区讨论**: 每题都有讨论区

## 平台技术架构

### 2.1 系统架构

```
前端 (React)
├── 代码编辑器 (Monaco Editor)
├── 题目展示组件
├── 提交与评测界面
└── 学习进度仪表盘

后端 (Go/Rust)
├── API Gateway
├── 题目管理服务
├── 评测队列服务
├── 沙箱执行环境
└── 用户数据服务

执行环境 (Docker/K8s)
├── 编译容器 (GCC/Clang)
├── 运行沙箱 ( seccomp-bpf )
├── 资源限制 (cgroup)
└── 结果收集
```

### 2.2 评测流程

```
用户提交代码
      │
      ▼
代码静态检查 (风格/安全)
      │
      ▼
编译 (gcc/clang -Wall -Wextra)
      │
      ├── 编译失败 → 返回错误信息
      │
      ▼
运行测试用例 (多组输入)
      │
      ├── 运行时错误 → 返回错误类型
      ├── 结果错误 → 返回差异对比
      │
      ▼
性能测试 (时间和内存)
      │
      ▼
代码质量评分 (复杂度/注释)
      │
      ▼
生成详细报告
```

### 2.3 安全措施

- **沙箱隔离**: 每个提交在独立容器中运行
- **资源限制**: CPU时间5秒，内存256MB
- **系统调用过滤**: 只允许安全的syscall
- **网络隔离**: 禁止外部网络访问
- **代码扫描**: 静态分析检测危险代码

## 练习设计规范

### 3.1 题目结构

```yaml
title: "练习标题"
id: "01-02"
difficulty: 2
type: "implementation"
topics:
  - "变量"
  - "数据类型"
time_limit: 1  # 秒
memory_limit: 64  # MB
description: |
  题目描述，使用Markdown格式
  可以包含代码块：
  ```c
  int example();
  ```
input_format: "输入格式说明"
output_format: "输出格式说明"
sample_input: |
  5
sample_output: |
  25
constraints:
  - "1 <= n <= 1000"
test_cases:
  - input: "5"
    output: "25"
    is_sample: true
  - input: "10"
    output: "100"
    is_hidden: true
```

### 3.2 测试用例设计

**原则**:
- 边界值: 最小、最大、临界值
- 随机值: 正常范围内的随机测试
- 特殊值: 0、负数、空值等
- 压力值: 大规模数据测试性能

**示例**: 数组求和
```
Test 1: [1, 2, 3] → 6 (基本功能)
Test 2: [] → 0 (空数组)
Test 3: [INT_MAX-1, 1] → INT_MAX (溢出边界)
Test 4: [1]*1000000 → 1000000 (性能测试)
```

## 练习示例

### 示例: 实现strcpy

**题目描述**:
实现标准库函数strcpy，将源字符串复制到目标缓冲区。

**函数原型**:
```c
char *my_strcpy(char *dest, const char *src);
```

**要求**:
- 包含终止符'\0'
- 返回目标指针
- 假设dest有足够的空间

**测试用例**:

```c
// Test 1
char dest[20];
my_strcpy(dest, "Hello");
// dest应该包含"Hello\0"

// Test 2
char dest[1];
my_strcpy(dest, "");
// dest应该只包含"\0"

// Test 3
char dest[100];
my_strcpy(dest, "A very long string...");
// 正确复制长字符串
```

**参考答案**:
```c
char *my_strcpy(char *dest, const char *src) {
    char *d = dest;
    while ((*d++ = *src++) != '\0');
    return dest;
}
```
