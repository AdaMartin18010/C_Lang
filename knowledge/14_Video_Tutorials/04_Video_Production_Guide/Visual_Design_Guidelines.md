## 1. 设计原则

### 1.1 核心原则

```
C语言视频教程视觉设计三大原则:

1. 清晰性 (Clarity)
   - 代码必须清晰可读
   - 重点突出，层次分明
   - 避免视觉干扰

2. 一致性 (Consistency)
   - 统一的设计语言
   - 可预测的视觉模式
   - 品牌识别度

3. 专业性 (Professionalism)
   - 符合技术教育定位
   - 现代但不花哨
   - 可信度和权威性
```

### 1.2 风格定位

- **风格**: 现代科技感 + 教育专业性
- **调性**: 清晰、精准、深入
- **情感**: 可信赖、启发性、专业

## 3. 字体系统

### 3.1 字体选择

**标题字体**:

- 中文: 思源黑体 (Noto Sans CJK SC)
- 英文: Inter / Roboto
- 用途: 标题、重点强调

**正文字体**:

- 中文: 思源黑体 Regular
- 英文: Inter Regular
- 用途: 说明文字、注释

**代码字体**:

- Fira Code / JetBrains Mono
- 用途: 代码展示、终端输出
- 特性: 等宽、连字支持

## 6. 动画规范

### 6.1 动画原则

- **目的性**: 每个动画都有明确目的
- **微妙性**: 不过度使用，不分散注意力
- **一致性**: 相机动作用相同时长
- **性能**: 60fps流畅运行

### 6.2 标准动画时长

| 类型 | 时长 | 缓动函数 |
|-----|------|---------|
| 微交互 | 150ms | ease-out |
| 标准过渡 | 300ms | ease-in-out |
| 页面切换 | 500ms | cubic-bezier(0.4, 0, 0.2, 1) |
| 复杂动画 | 800ms | custom |

### 6.3 常用动画效果

**代码逐行显示**:

```css
@keyframes lineReveal {
  from {
    opacity: 0;
    transform: translateX(-20px);
  }
  to {
    opacity: 1;
    transform: translateX(0);
  }
}

.code-line {
  animation: lineReveal 0.3s ease-out forwards;
}
```

**高亮闪烁**:

```css
@keyframes highlightPulse {
  0%, 100% { background-color: transparent; }
  50% { background-color: rgba(0, 212, 170, 0.3); }
}

.highlight {
  animation: highlightPulse 1s ease-in-out 3;
}
```

**光标闪烁**:

```css
@keyframes blink {
  0%, 50% { opacity: 1; }
  51%, 100% { opacity: 0; }
}

.cursor {
  animation: blink 1s step-end infinite;
}
```

使用品牌色的不同透明度:

- 柱宽: 40px
- 间距: 16px
- 圆角: 4px (顶部)
- 数值标签: 顶部居中

**时间线/流程图**:

- 节点: 圆形，直径 24px
- 连线: 2px，实线或虚线
- 标签: 节点右侧或上方

**内存布局图**:

- 不同区域使用不同背景色
- 箭头指示数据流向
