## 1. 制作流程概览

```
完整的视频制作流程:

阶段1: 前期准备 (2-3天)
├── 确定主题和学习目标
├── 编写详细脚本
├── 准备代码示例和演示
└── 收集参考资料

阶段2: 录制 (1-2天)
├── 设置录制环境
├── 录制屏幕和音频
├── 拍摄真人出镜 (可选)
└── 备份原始素材

阶段3: 后期制作 (2-3天)
├── 粗剪和整理
├── 添加字幕和标注
├── 插入动画和特效
├── 配乐和音效
└── 调色和输出

阶段4: 审核发布 (1天)
├── 内容审核
├── 技术审核
├── 最终输出
└── 发布和分发
```

## 2. 录制设备推荐

### 2.1 屏幕录制

| 软件 | 平台 | 特点 | 推荐场景 |
|-----|------|------|---------|
| OBS Studio | Win/Mac/Linux | 免费开源，功能强大 | 专业录制 |
| Camtasia | Win/Mac | 录制+编辑一体 | 快速制作 |
| ScreenFlow | Mac | 界面友好 | Mac用户 |
| Loom | 浏览器 | 即时分享 | 快速演示 |

**OBS Studio 设置建议**:

```
视频设置:
- 分辨率: 1920x1080 (1080p) 或 2560x1440 (2K)
- 帧率: 30fps (录制) / 60fps (游戏)
- 编码: H.264 (NVENC if NVIDIA GPU)

音频设置:
- 采样率: 48kHz
- 声道: 立体声
- 码率: 160kbps

输出设置:
- 格式: MKV (录制) -> MP4 (发布)
- 录像路径: SSD 快速存储
```

### 2.2 音频设备

**麦克风选择**:

| 级别 | 推荐产品 | 价格区间 | 适用场景 |
|-----|---------|---------|---------|
| 入门级 | Blue Snowball, 舒尔MV7 | ￥300-800 | 初学者 |
| 进阶级 | Audio-Technica AT2020, 舒尔SM7B | ￥1000-3000 | 常规制作 |
| 专业级 | Neumann TLM 103, 舒尔SM58 | ￥5000+ | 专业工作室 |

**录音环境处理**:

- 使用吸音棉减少回声
- 避免硬墙面反射
- 关闭空调/风扇等噪音源
- 使用防喷罩减少爆破音

### 2.3 摄像设备 (真人出镜)

**推荐配置**:

- 摄像头: Logitech C920/C922 (入门级), Sony A6400 (专业级)
- 分辨率: 1080p 最低
- 帧率: 30fps
- 照明: 三点布光 (主光、补光、轮廓光)

## 3. 录制技巧

### 3.1 屏幕录制最佳实践

**代码展示**:

```
字体设置:
- 字体: Fira Code, JetBrains Mono, Cascadia Code
- 字号: 18-24px (确保移动端可读)
- 行高: 1.5-1.8

编辑器配置:
- 主题: 暗色主题 (减少眼睛疲劳)
- 行号: 显示
- 缩进指示: 显示
- 滚动条: 始终显示

演示技巧:
1. 使用字体放大快捷键 (Ctrl++/Cmd++)
2. 高亮当前行
3. 平滑滚动，避免快速跳动
4. 鼠标轨迹可见 (但不要太显眼)
```

**终端录制**:

```
终端美化:
- 使用 zsh + oh-my-zsh
- 配置 Powerlevel10k 主题
- 字体: Nerd Font (支持图标)
- 背景半透明 (显示壁纸增加层次)

演示技巧:
1. 使用 asciinema 录制可交互终端
2. 清除/简化提示符
3. 避免显示个人路径信息
```

### 3.2 声音录制

**语音技巧**:

- 语速: 适中偏慢 (比日常慢20%)
- 停顿: 关键概念后停顿2秒
- 语调: 有起伏，避免单调
- 音量: 保持一致

**降噪处理**:

```
录制时:
- 保持麦克风距离 15-20cm
- 使用防喷罩
- 避免衣物摩擦声

后期处理:
- 使用 Audacity 降噪
- 标准化音量 (-3dB 峰值)
- 压缩动态范围
```

### 3.3 真人出镜技巧

**镜头表现**:

- 看镜头 (与观众建立联系)
- 保持自然微笑
- 适当手势增强表达
- 避免过多小动作

**背景设置**:

- 简洁干净
- 与主题相关 (书架、设备)
- 避免杂乱和反光
- 良好照明

## 4. 后期制作

### 4.1 剪辑软件

| 软件 | 平台 | 学习曲线 | 特点 |
|-----|------|---------|------|
| DaVinci Resolve | Win/Mac/Linux | 中等 | 免费，专业级调色 |

| iMovie | Mac/iOS | 简单 | 入门友好 |

### 4.2 剪辑流程

```
标准剪辑流程:

1. 素材整理
   ├── 按镜头/场景分类
   ├── 命名规范: 日期_场景_镜次_版本
   └── 删除废片

2. 粗剪 (Assembly Cut)
   ├── 按脚本顺序排列
   ├── 剪掉明显错误
   └── 确定大致时长

3. 精剪 (Fine Cut)
   ├── 调整节奏和转场
   ├── 添加B-roll (补充画面)
   ├── 修剪停顿和口误
   └── 确保逻辑连贯

4. 音频处理
   ├── 降噪和标准化
   ├── 添加背景音乐
   ├── 音量和淡入淡出
   └── 添加音效

5. 视觉特效
   ├── 添加字幕
   ├── 插入图表和动画
   ├── 调色和风格化
   └── 添加片头片尾

6. 导出
   ├── 预览检查
   ├── 渲染输出
   └── 多平台适配
```

### 4.3 字幕制作

**字幕类型**:

1. **开放式字幕** (硬字幕) - 直接嵌入视频
2. **闭合式字幕** (软字幕) - 可开关的CC字幕

**制作工具**:

- ArcTime Pro (推荐)
- Aegisub
- Adobe Premiere 内置
- YouTube 自动生成 + 校对

**字幕规范**:

```
样式设置:
- 字体: 思源黑体 / Noto Sans CJK
- 字号: 根据分辨率调整 (1080p建议36-48px)
- 颜色: 白色 + 黑色描边/阴影
- 位置: 底部居中，距底边 10%

时间轴:
- 单句长度: 2-6秒
- 字符数: 每行不超过15个中文
- 提前出现: 0.2秒
- 延后消失: 0.2秒
```
