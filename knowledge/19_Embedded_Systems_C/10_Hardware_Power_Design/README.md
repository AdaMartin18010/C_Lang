## 2. LDO 深度分析

### 2.1 LDO形式化模型

**定义 2.1 (LDO传递函数)**

```
LDO核心方程:
  Vout = Vref × (1 + R1/R2)

压差 (Dropout Voltage):
  Vdrop = Vin - Vout = Iout × RDS(on)

功率耗散:
  Pd = (Vin - Vout) × Iout + Vin × Iq

效率:
  η ≈ Vout / Vin (当 Iout >> Iq)

热计算:
  Tj = Ta + Pd × Rth(ja) < Tj_max (125°C)
```

## 3. Buck 转换器设计

### 3.1 Buck稳态方程

**定义 3.1 (Buck转换器)**

```
理想关系:
  Vout = D × Vin
  其中 D = Ton/Tsw (占空比)

电感选择:
  L = (Vin - Vout) × D / (r × Iout × fsw)
  其中 r = ΔIL/Iout = 0.2-0.4 (纹波电流比)

输出纹波:
  ΔVout = ΔIL/(8×C×fsw) + ΔIL×ESR

CCM条件:
  Iout > (Vin-Vout)×D/(2×L×fsw)
```

### 3.2 控制环路稳定性

```
功率级传递函数:
  Gvd(s) = Vin × (1 + s/ωesr) / (1 + s/(Q×ω0) + (s/ω0)²)

  ω0 = 1/√(LC)          (谐振频率)
  Q = Rload×√(C/L)      (品质因数)

Type II补偿:
  Gc(s) = K×(1+s/ωz)/(s×(1+s/ωp))

  设计目标:
    fc = fsw/10          (穿越频率)
    PM > 45°             (相位裕度)
    GM > 6dB             (增益裕度)
```

## 4. 电源完整性分析

### 4.1 PDN目标阻抗

**定义 4.1 (目标阻抗)**

```
Ztarget = ΔVmax / ΔImax

示例:
  Vout = 1.2V, 5%纹波 → ΔVmax = 60mV
  ΔImax = 5A
  Ztarget = 12mΩ

PDN元件频率范围:
  VRM:      DC-1kHz    (稳压器)
  Bulk:     1k-100kHz  (大电容10-470μF)
  Ceramic:  100k-10MHz (陶瓷1-10μF)
  PCB:      10-100MHz  (平面电容)
  On-chip:  >100MHz    (片上解耦)
```
