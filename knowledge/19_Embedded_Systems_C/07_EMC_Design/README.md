# EMC (电磁兼容) 设计深度解析

> **核心目标**: 抑制电磁干扰 (EMI)、确保电磁敏感性 (EMS)
> **三大支柱**: 滤波、屏蔽、接地
> **法规标准**: CISPR、IEC、FCC、CE
> **最后更新**: 2026-03-29

---

### 1.2 干扰传播模型

**定义 1.1 (EMI传播模型)**

```
EMI问题 = 干扰源 × 耦合路径 × 敏感设备

干扰源强度 (dBμV):
  V_source = 20 × log10(V / 1μV)

耦合路径损耗 (dB):
  传导: Attenuation = 20 × log10(Z_filter / Z_source)
  辐射: Attenuation = 20 × log10(λ / (2πd)) + Shielding_effectiveness

敏感设备裕量 (dB):
  Margin = V_immunity - V_coupled

  其中 V_coupled = V_source - Attenuation

设计准则:
  Margin > 6dB (安全裕量)

  即: V_immunity - (V_source - Attenuation) > 6dB

  解得: Attenuation > V_source - V_immunity + 6dB
```

### 2.2 滤波器插入损耗计算

**定义 2.1 (插入损耗)**

```
插入损耗 (IL) = 20 × log10(V_without_filter / V_with_filter)

LC低通滤波器 (截止频率):
  f_c = 1 / (2π√(LC))

  对于 f >> f_c:
    Attenuation ≈ 40 × log10(f / f_c) dB/decade

  即每十倍频程40dB衰减 (二阶滤波器)

设计示例:
  目标: 抑制100MHz干扰，需要60dB衰减

  设 f_c = 10MHz
  100MHz是f_c的10倍 (1 decade)
  二阶滤波器可提供40dB衰减 → 不足

  解决方案:
    1. 使用三阶滤波器 (60dB/decade)
    2. 降低f_c到1MHz (100MHz是2 decade)
       40dB × 2 = 80dB > 60dB

元件选择约束:
  电感:
    - 自谐振频率 (SRF) > 5 × f_interference
    - 额定电流 > 1.5 × I_max
    - DCR引起的压降 < 5% V_nominal

  电容:
    - ESL引起的自谐振 > 0.5 × f_interference
    - 耐压 > 2 × V_max
    - 温度特性满足工作范围
```

## 3. 接地与屏蔽设计

### 3.1 接地策略对比矩阵

| 接地方式 | 原理 | 优点 | 缺点 | 适用场景 |
|----------|------|------|------|----------|
| **单点接地** | 所有地线汇聚于一点 | 无地环路 | 高频阻抗大 | 低频 (<1MHz) |
| **多点接地** | 就近接地 | 高频阻抗低 | 可能形成环路 | 高频 (>10MHz) |
| **混合接地** | 低频单点+高频多点 | 兼顾两者 | 设计复杂 | 宽频带系统 |
| **浮地** | 不直接接地 | 阻断地环路 | 静电积累 | 敏感测量 |

### 3.2 屏蔽效能计算

**定义 3.1 (屏蔽效能)**

```
屏蔽效能 (SE) = 20 × log10(E_incident / E_transmitted) dB

屏蔽机制:
  1. 反射损耗 (R): 阻抗失配导致反射
  2. 吸收损耗 (A): 趋肤效应导致能量耗散
  3. 多次反射损耗 (B): 薄屏蔽体内的内部反射

总屏蔽效能:
  SE = R + A + B (dB)

吸收损耗计算:
  A = 8.69 × t / δ dB

  其中:
    t = 屏蔽体厚度 (m)
    δ = 趋肤深度 = √(2 / (ωμσ))

    ω = 2πf (角频率)
    μ = μ0μr (磁导率)
    σ = 电导率

反射损耗 (电场):
  R_E = 168 - 10×log10(μr/σr × f³ × r²) dB

  其中:
    μr = 相对磁导率
    σr = 相对电导率
    f = 频率 (MHz)
    r = 源到屏蔽体距离 (m)

材料选择:
  铜/铝: 高电导率 → 高反射损耗 → 适合电场屏蔽
  钢/μ合金: 高磁导率 → 高吸收损耗 → 适合磁场屏蔽
  多层屏蔽: 铜-钢-铜组合，兼顾高低频
```

## 4. 信号完整性与EMC

### 4.1 传输线理论

**定义 4.1 (传输线参数)**

```
特性阻抗:
  Z0 = √(L / C)

  其中:
    L = 单位长度电感 (H/m)
    C = 单位长度电容 (F/m)

微带线 (Microstrip) 近似公式:
  Z0 ≈ 87 / √(εr + 1.41) × ln(5.98h / (0.8w + t))

  其中:
    εr = 介电常数
    h = 介质厚度
    w = 线宽
    t = 铜厚

带状线 (Stripline) 近似公式:
  Z0 ≈ 60 / √εr × ln(4h / (0.67π(0.8w + t)))

反射系数:
  Γ = (ZL - Z0) / (ZL + Z0)

  反射损耗 = -20 × log10(|Γ|) dB

  完全匹配 (ZL = Z0): Γ = 0, 无反射
  开路 (ZL = ∞): Γ = 1, 全反射
  短路 (ZL = 0): Γ = -1, 全反射反相

EMC影响:
  阻抗不连续 → 反射 → 驻波 → 辐射增加
  端接策略:
    - 源端端接: 串联电阻 Rs = Z0 - Z_driver
    - 负载端端接: 并联电阻 R = Z0 (直流功耗大)
    - AC端接: 电阻串联电容 (隔直)
```

### 4.2 层叠设计决策矩阵

| 层数 | 结构 | 优势 | 成本 | EMC性能 |
|------|------|------|------|---------|
| **2层** | 信号-地/信号-电源 | 最低成本 | 低 | 差 (无完整平面) |
| **4层** | 信号-GND-PWR-信号 | 完整参考平面 | 中 | 良 |
| **6层** | 信号-GND-signal-PWR-GND-signal | 多信号层 | 中高 | 优 |
| **8层+** | 复杂层叠 | 高密度/多电源 | 高 | 最优 |
