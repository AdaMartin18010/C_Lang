## 2. FOC (磁场定向控制) 形式化定义

### 2.1 FOC核心变换

**定义 2.1 (Clark变换 - 3相到2相)**
```
Clark变换 (ABC → αβ):
  [i_α]   [ 2/3   -1/3   -1/3 ] [i_A]
  [i_β] = [  0    1/√3  -1/√3 ] [i_B]
                                 [i_C]

等幅值变换 (保持幅值不变):
  i_α = i_A
  i_β = (i_A + 2*i_B) / √3

约束: i_A + i_B + i_C = 0

逆Clark变换 (αβ → ABC):
  i_A = i_α
  i_B = (-i_α + √3*i_β) / 2
  i_C = (-i_α - √3*i_β) / 2
```

**定义 2.2 (Park变换 - 静止到旋转)**
```
Park变换 (αβ → dq):
  [i_d]   [ cos(θ)   sin(θ) ] [i_α]
  [i_q] = [-sin(θ)   cos(θ) ] [i_β]

其中 θ = 转子电角度

d轴: 与转子磁链对齐 (产生磁通)
q轴: 超前d轴90° (产生转矩)

转矩方程:
  T_e = (3/2) * p * [ψ_f * i_q + (L_d - L_q) * i_d * i_q]
  
  其中:
    p = 极对数
    ψ_f = 永磁磁链
    L_d, L_q = d/q轴电感

对于SPMSM (表贴式):
  L_d = L_q = L_s
  T_e = (3/2) * p * ψ_f * i_q
  (转矩与i_q成正比)

对于IPMSM (内置式):
  L_d < L_q
  可利用磁阻转矩: T_e = (3/2)*p*[ψ_f*i_q + (L_d-L_q)*i_d*i_q]
  采用MTPA (最大转矩电流比) 控制
```

## 3. SVPWM (空间矢量脉宽调制)

### 3.1 SVPWM原理

**定义 3.1 (电压空间矢量)**
```
基本电压矢量:
  V0 = 000 (零矢量)
  V1 = 100, V2 = 110, V3 = 010
  V4 = 011, V5 = 001, V6 = 101
  V7 = 111 (零矢量)

扇区划分 (6个扇区):
  Sector I:   0°   - 60°   (V4, V6)
  Sector II:  60°  - 120°  (V2, V6)
  Sector III: 120° - 180°  (V2, V3)
  Sector IV:  180° - 240°  (V1, V3)
  Sector V:   240° - 300°  (V1, V5)
  Sector VI:  300° - 360°  (V4, V5)

调制比 (调制深度):
  m = |V_ref| / (2/3 * V_dc)
  
  线性调制范围: 0 ≤ m ≤ 1
  过调制: m > 1 (引入低次谐波)

占空比计算 (以Sector I为例):
  T1 = m * T_s * sin(60° - θ)
  T2 = m * T_s * sin(θ)
  T0 = T_s - T1 - T2
  
  其中 T_s = PWM周期

相电压波形: 马鞍波 (比正弦调制提高15%电压利用率)
```

### 3.2 SVPWM实现算法

```c
/* SVPWM形式化实现 */

typedef struct {
    float vd;       /* d轴电压 */
    float vq;       /* q轴电压 */
    float vdc;      /* 直流母线电压 */
    float theta;    /* 电角度 */
} SVPWM_Input;

typedef struct {
    float ta, tb, tc;   /* 三相占空比 [0, 1] */
    uint8_t sector;     /* 扇区 1-6 */
} SVPWM_Output;

void svpwm_calculate(SVPWM_Input *in, SVPWM_Output *out) {
    /* 1. 反Park变换 (Vd, Vq) -> (Vα, Vβ) */
    float valpha = in->vd * cos(in->theta) - in->vq * sin(in->theta);
    float vbeta  = in->vd * sin(in->theta) + in->vq * cos(in->theta);
    
    /* 2. 计算三相电压 */
    float va = valpha;
    float vb = -0.5f * valpha + 0.8660254f * vbeta;
    float vc = -0.5f * valpha - 0.8660254f * vbeta;
    
    /* 3. 加入零序分量 (三次谐波注入) */
    float vmax = fmaxf(va, fmaxf(vb, vc));
    float vmin = fminf(va, fminf(vb, vc));
    float voffset = -0.5f * (vmax + vmin);
    
    /* 4. 归一化到PWM周期 */
    float v_norm = 1.0f / (in->vdc * 0.5773503f);  /* 2/sqrt(3) */
    out->ta = 0.5f + (va + voffset) * v_norm;
    out->tb = 0.5f + (vb + voffset) * v_norm;
    out->tc = 0.5f + (vc + voffset) * v_norm;
    
    /* 5. 限幅 */
    out->ta = fmaxf(0.0f, fminf(1.0f, out->ta));
    out->tb = fmaxf(0.0f, fminf(1.0f, out->tb));
    out->tc = fmaxf(0.0f, fminf(1.0f, out->tc));
}

/* SVPWM vs SPWM 优势:
 * 
 * 1. 电压利用率高15% (最大调制比1.15 vs 1.0)
 * 2. 谐波含量低
 * 3. 更适合数字实现
 * 4. 易于过调制实现
 */
```

## 4. PI调节器数字化

### 4.1 离散PI形式化

**定义 4.1 (离散PI控制器)**
```
连续域:
  u(t) = Kp * e(t) + Ki * ∫e(t)dt

离散化 (后向欧拉):
  u[k] = Kp * e[k] + Ki * T_s * Σe[i]

增量式 (抗饱和):
  Δu[k] = Kp * (e[k] - e[k-1]) + Ki * T_s * e[k]
  u[k] = u[k-1] + Δu[k]

积分分离 (大误差时取消积分):
  if |e[k]| > E_threshold:
      Ki_eff = 0
  else:
      Ki_eff = Ki

输出限幅:
  u[k] = sat(u[k], U_min, U_max)
  
  条件积分 (抗积分饱和):
    if u[k] == U_max and e[k] > 0:
        不积分 (积分项保持不变)
    if u[k] == U_min and e[k] < 0:
        不积分
```

### 4.2 参数整定方法

| 方法 | 公式 | 适用场景 | 特点 |
|------|------|----------|------|
| **Ziegler-Nichols** | Kp=0.6*Ku, Ki=2*Kp/Tu | 通用 | 经验法则 |
| **Lambda整定** | Kp=τ/(K*λ), Ki=1/τ | 过程控制 | 鲁棒性好 |
| **模态整定** | 根据阻尼比设计 | 伺服系统 | 响应可控 |
| **试凑法** | 经验调整 | 简单系统 | 直观但耗时 |
