> **测试方法**: 单元测试、集成测试、硬件在环、边界扫描

### 1.2 调试接口对比矩阵

| 特性 | JTAG | SWD | SWO | 并行跟踪 |
|------|------|-----|-----|----------|
| **信号线** | 5 | 2 | 1 | 1-4 |
| **速度** | 10-20MHz | 50+MHz | 1-50Mbps | 100+MHz |
| **功能** | 全功能 | 调试+跟踪 | 仅跟踪 | 高速跟踪 |
| **引脚需求** | 多 | 少 | 可复用SWD | 多 |
| **边界扫描** | 支持 | 不支持 | - | - |
| **推荐使用** | 开发/测试 | 生产/空间受限 | 轻量跟踪 | 深度分析 |

### 2.1 断点类型决策树

```
断点类型选择决策树
│
├── 代码断点?
│   ├── 是
│   │   ├── 闪存地址 (Flash)

├── 数据断点?
│   ├── 是
│   │   ├── 读访问监控

│   │   │
```

### 4.2 CoreDump机制

```c
/* CoreDump (故障现场保存) 形式化实现 */

typedef struct {
    /* 核心寄存器 */
    uint32_t r0, r1, r2, r3, r12;
    uint32_t lr;        /* 链接寄存器 */
    uint32_t pc;        /* 程序计数器 */
    uint32_t xpsr;      /* 程序状态寄存器 */

    /* 故障特定寄存器 */
    uint32_t cfsr;      /* 可配置故障状态 */
    uint32_t hfsr;      /* HardFault状态 */
    uint32_t mmfar;     /* MemManage地址 */
    uint32_t bfar;      /* BusFault地址 */

    /* 上下文信息 */
    uint32_t exc_return;
    uint32_t msp;       /* 主堆栈指针 */
    uint32_t psp;       /* 进程堆栈指针 */

    /* 时间戳 */
    uint32_t timestamp;
    uint32_t reset_count;

    /* 校验 */
    uint32_t magic;
    uint32_t checksum;
} CoreDumpInfo;

#define COREDUMP_MAGIC  0xDEADBEEF
#define COREDUMP_ADDR   0x2000FFF0  /* 保留的RAM区域 */

void save_coredump(void) {
    CoreDumpInfo *dump = (CoreDumpInfo *)COREDUMP_ADDR;

    /* 读取寄存器 */
    __asm volatile (
        "mrs %0, msp\n"
        "mrs %1, psp\n"
        "mrs %2, xpsr\n"
        : "=r"(dump->msp), "=r"(dump->psp), "=r"(dump->xpsr)
    );

    dump->cfsr = SCB->CFSR;
    dump->hfsr = SCB->HFSR;
    dump->mmfar = SCB->MMFAR;
    dump->bfar = SCB->BFAR;

    dump->magic = COREDUMP_MAGIC;
    dump->timestamp = get_tick_count();

    /* 计算校验 */
    dump->checksum = calculate_checksum(dump, sizeof(CoreDumpInfo) - 4);
}

/* HardFault处理 */
void HardFault_Handler(void) {
    /* 保存现场 */
    save_coredump();

    /* 可选: 尝试恢复 */
    /* 或: 触发复位 */
    NVIC_SystemReset();
}

/* 启动时检查 */
void check_coredump(void) {
    CoreDumpInfo *dump = (CoreDumpInfo *)COREDUMP_ADDR;

    if (dump->magic == COREDUMP_MAGIC) {
        if (verify_checksum(dump)) {
            /* 有效的CoreDump */
            log_error("Previous crash detected:");
            log_error("  PC = 0x%08X", dump->pc);
            log_error("  LR = 0x%08X", dump->lr);
            log_error("  CFSR = 0x%08X", dump->cfsr);

            /* 解析故障原因 */
            analyze_fault(dump);

            /* 发送到上位机/云端 */
            upload_coredump(dump);
        }

        /* 清除标记 */
        dump->magic = 0;
    }
}
```
