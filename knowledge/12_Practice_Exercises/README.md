# 12 实践练习与项目 (Practice Exercises & Projects)

> **难度**: L1-L6 | **预估学习时间**: 50-100小时（全部完成）
> **目标**: 通过实际编程练习巩固C语言知识

---

## 模块概述

本模块提供**分级练习题**和**实际项目案例**，帮助学习者从理论走向实践。每个练习都配有参考答案和详细解析。

### 适用人群

| 级别 | 适合人群 | 推荐练习 |
|:-----|:---------|:---------|
| **初级** | C语言新手 | 基础语法练习、简单算法 |
| **中级** | 有编程经验 | 数据结构、系统编程 |
| **高级** | 专业开发者 | 性能优化、并发编程、嵌入式 |
| **专家** | 系统程序员 | 编译器、操作系统、形式化验证 |

## 学习建议

### 路径1：基础到进阶

```text
基础练习 → 指针专项 → 数据结构项目 → 算法挑战 → 系统编程
```

### 路径2：嵌入式方向

```text
基础练习 → 指针专项 → 嵌入式挑战 → 性能优化 → 并发并行
```

### 路径3：系统编程方向

```text
指针专项 → 数据结构项目 → 系统编程 → 并发并行 → 迷你项目
```

### 路径4：安全专家方向

```text
基础练习 → 安全编码 → 调试谜题 → 并发并行 → 迷你项目
```

### 路径5：现代C/Zig开发者

```text
基础练习 → 指针专项 → Zig-C迁移 → 并发并行 → 迷你项目
```

### 路径6：全栈挑战（推荐）

```text
基础练习 → 指针专项 → 数据结构项目 → 算法挑战 → 系统编程
→ 并发并行 → 安全编码 → 调试谜题 → 迷你项目
```

## 评估标准

| 等级 | 要求 |
|:-----|:-----|
| **合格** | 代码正确运行，无明显bug |
| **良好** | 代码风格规范，有适当注释 |
| **优秀** | 考虑边界情况，有单元测试 |
