# MIT 6.S081 实验

本目录提供MIT 6.S081（原6.828）操作系统课程实验的完整指南。该课程使用xv6教学操作系统，帮助学生深入理解操作系统内核原理。

## 课程概述

MIT 6.S081 是一门经典的本科生操作系统课程，通过实现一个真实的RISC-V操作系统（xv6）来学习OS核心概念。
