> **目标**: 通过阅读优秀开源项目源码，学习工业级C编程技巧

---

## 项目列表

| 项目 | 核心特点 | 文档 |
|:-----|:---------|:-----|
| **Redis** | 高性能内存数据库、紧凑数据结构、渐进式rehash | [数据结构解读](01_Redis_Data_Structures.md) |

## 学习路径

```
1. 数据结构设计
   └── Redis SDS、Dict、SkipList

2. 存储引擎原理
   └── SQLite B-Tree、Pager、VDBE

3. 高性能网络编程
   └── Nginx epoll、事件循环、连接池

4. 版本控制系统设计
   └── Git对象模型、打包算法、分支管理
```
