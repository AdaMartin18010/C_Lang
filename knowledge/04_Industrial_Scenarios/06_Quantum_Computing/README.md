## 📖 技术概述

本目录涵盖**量子计算**核心技术——量子模拟、量子-经典接口、量子纠错和Shor算法。
量子计算代表了计算范式的根本性转变，利用量子叠加和纠缠实现经典计算机无法企及的计算能力，在密码学、药物发现、优化问题等领域具有革命性潜力。

### 核心知识点

| 主题 | 描述 | 应用场景 |
|:-----|:-----|:---------|
| 量子模拟 | 经典计算机模拟量子系统 | 算法验证、教育、小规模原型 |
| 量子-经典接口 | 编译器、脉冲控制、测量处理 | 量子计算机控制系统 |
| 表面码纠错 | 容错量子计算的核心纠错码 | 大规模量子计算机 |
| Shor算法 | 量子因数分解算法 | 密码分析、安全研究 |

### 01_Quantum_Simulation_C.md 详细内容

- **复数运算基础**：C99 complex.h、复数加减乘除、兼容非C99环境的实现
- **量子态表示**：态矢量结构、振幅数组、希尔伯特空间维度(2^n)
- **量子门实现**：
  - 单量子比特门：Pauli-X/Y/Z、Hadamard、S、T门
  - 旋转门：RX/RY/RZ参数化实现
  - 双量子比特门：CNOT、受控门
- **测量操作**：概率计算、坍缩态、随机采样
- **Grover搜索算法**：Oracle标记、扩散算子、迭代次数计算
- **性能优化**：AVX2 SIMD复数运算、并行处理
- **常见陷阱**：状态空间指数爆炸、浮点精度损失

### 01_Quantum_Classical_Interface.md 详细内容

- **量子编译器架构**：
  - 高级语言 → OpenQASM → 脉冲序列 → AWG输出

- **OpenQASM支持**：
  - 标准门集：X/Y/Z/H/S/T、旋转门、CNOT
  - 电路表示：门指令数组、时间调度
  - 简化解析器实现
- **脉冲序列生成**：
  - 脉冲形状：高斯、DRAG、方波
  - AWG波形：IQ调制、相位控制
  - 门到脉冲映射：频率、振幅、持续时间
- **测量处理**：
  - IQ解调：数字混频、积分
  - 阈值判定：单阈值/多阈值分类
  - 校准：质心计算、分离度分析
  - 统计：保真度估计、错误率
- **实时反馈**：测量结果驱动的条件执行

## ⚠️ 学习建议

### 常见陷阱

1. **状态空间爆炸**：n量子比特需要2^n复数，n=30需8GB内存
2. **忽略退相干**：真实量子比特有有限的相干时间
3. **过度简化**：忽视纠错开销和物理实现约束
4. **数学与物理脱节**：需要同时理解数学形式和物理实现

### 实践建议

- 从小规模(2-4量子比特)开始，逐步扩展
- 使用Qiskit等框架验证C实现结果
- 关注量子计算领域的最新突破(Google、IBM、中科院等)
- 参与开源量子软件项目
