# CMU 15-213 实验

本目录提供CMU 15-213（计算机系统导论）课程实验的完整指南。这门课程被称为"CSAPP"（Computer Systems: A Programmer's Perspective），是理解计算机系统的经典课程。

## 课程概述

CMU 15-213 从程序员的角度深入讲解计算机系统，涵盖数据表示、汇编、内存层次结构、链接、异常控制流等核心主题。

## 核心技能

- **位级操作**: 理解数据的底层表示
- **汇编阅读**: 读懂x86-64汇编代码
- **安全思维**: 理解常见安全漏洞原理
- **性能优化**: 缓存友好的代码设计
- **系统编程**: Unix系统调用与并发
