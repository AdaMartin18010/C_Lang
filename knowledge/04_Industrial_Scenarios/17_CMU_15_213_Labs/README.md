# CMU 15-213 实验

本目录提供CMU 15-213（计算机系统导论）课程实验的完整指南。这门课程被称为"CSAPP"（Computer Systems: A Programmer's Perspective），是理解计算机系统的经典课程。

## 课程概述

CMU 15-213 从程序员的角度深入讲解计算机系统，涵盖数据表示、汇编、内存层次结构、链接、异常控制流等核心主题。

## 实验列表

| 实验 | 文件 | 主题 |
|------|------|------|
| 概述 | `00_CMU_Labs_Overview.md` | 实验介绍与准备 |
| Data Lab | `01_Data_Lab.md` | 位操作与整数/浮点数表示 |
| Bomb Lab | `02_Bomb_Lab.md` | 汇编代码逆向工程 |
| Attack Lab | `03_Attack_Lab.md` | 缓冲区溢出攻击与防御 |
| Cache Lab | `04_Cache_Lab.md` | 缓存模拟与优化 |
| Shell Lab | `05_Shell_Lab.md` | Unix Shell实现 |
| Malloc Lab | `06_Malloc_Lab.md` | 动态内存分配器 |
| Proxy Lab | `07_Proxy_Lab.md` | Web代理服务器 |

## 实验难度梯度

```
Data Lab → Bomb Lab → Attack Lab → Cache Lab → Shell Lab → Malloc Lab → Proxy Lab
  入门       进阶        进阶        中级        中级        高级        高级
```

## 核心技能

- **位级操作**: 理解数据的底层表示
- **汇编阅读**: 读懂x86-64汇编代码
- **安全思维**: 理解常见安全漏洞原理
- **性能优化**: 缓存友好的代码设计
- **系统编程**: Unix系统调用与并发

- 与 `02_Formal_Semantics_and_Physics` 的硬件知识互补
