## 动力学基础

描述机器人运动与力的关系。

### 动力学方程

```
M(q)q̈ + C(q, q̇)q̇ + G(q) = τ
```

- M: 质量矩阵
- C: 科氏力和离心力
- G: 重力
- τ: 关节力矩
