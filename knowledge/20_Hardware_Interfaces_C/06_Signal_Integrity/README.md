## 1. 传输线理论基础

### 1.1 传输线方程

**定义 1.1 (电报方程)**

```
电报方程:
  ∂V/∂z = -L×∂I/∂t - R×I
  ∂I/∂z = -C×∂V/∂t - G×V

特性阻抗:
  Z0 = √((R + jωL)/(G + jωC))

  无损传输线 (R=G=0):
    Z0 = √(L/C)

传播速度:
  vp = 1/√(LC) = c/√εr

  其中 c = 光速 (3×10^8 m/s)
  εr = 介电常数 (FR-4 ≈ 4.2)

时延:
  td = length / vp

  微带线: 约 150ps/inch
  带状线: 约 170ps/inch

波长:
  λ = vp / f

  集总vs分布系统判据:
    当 length < λ/10 时，可视为集总系统
    当 length > λ/10 时，必须作为传输线处理

    示例: 1GHz信号，FR-4
    λ = c/(f×√εr) = 3e8/(1e9×2.05) ≈ 146mm
    λ/10 = 14.6mm

    >14.6mm走线必须阻抗控制
```

### 1.2 阻抗计算公式

| 类型 | 公式 | 参数说明 |
|------|------|----------|
| **微带线** | Z0 ≈ 87/√(εr+1.41)×ln(5.98h/(0.8w+t)) | h=介质厚,w=线宽,t=铜厚 |
| **带状线** | Z0 ≈ 60/√εr×ln(4h/(0.67π(0.8w+t))) | h=到参考平面距离 |
| **差分对** | Zdiff = 2×Z0×(1-0.48exp(-0.96s/h)) | s=间距 |
| **共面波导** | Z0 ≈ (η0/√εeff)×(K(k')/K(k)) | K=椭圆积分 |

## 2. 反射与端接

### 2.1 反射理论

**定义 2.1 (反射系数)**

```
反射系数:
  Γ = (ZL - Z0) / (ZL + Z0)

传输系数:
  T = 2×ZL / (ZL + Z0) = 1 + Γ

特殊情况:
  开路 (ZL = ∞): Γ = 1, 全反射
  短路 (ZL = 0): Γ = -1, 全反射反相
  匹配 (ZL = Z0): Γ = 0, 无反射

多次反射:
  源端反射系数: Γs = (Zs - Z0)/(Zs + Z0)
  负载端反射系数: ΓL = (ZL - Z0)/(ZL + Z0)

  稳态电压:
    V∞ = Vsource × ZL / (Zs + ZL)

  振铃幅度由Γs和ΓL决定

信号完整性判据:
  |Γ| < 0.1  (反射 < 10%)
  对应阻抗容差: Z0 × 0.9 < ZL < Z0 × 1.1
```

### 2.2 端接策略决策树

```
端接策略选择
│
├── 点对点 (单一驱动, 单一负载)
│   ├── 单向信号
│   │   ├── 源端端接 (推荐)
│   │   │   ├── 串联电阻 Rs = Z0 - Zdriver
│   │   │   ├── 优点: 低功耗, 无直流负载
│   │   │   └── 缺点: 源端半幅度波形
│   │   │
│   │   └── 负载端端接
│   │       ├── 并联电阻 R = Z0 到Vtt
│   │       ├── 优点:  clean waveform at load
│   │       └── 缺点: 直流功耗, 低电平抬高
│   │
│   └── 双向信号
│       └── 两端端接
│           ├── Thevenin端接 (R1//R2)
│           └── AC端接 (R+C)
│
├── 多点 (单一驱动, 多负载)
│   ├── Daisy chain (菊花链)
│   │   └── 末端端接
│   │       └── 短截线长度 < trise×vp/6
│   │
│   └── Star (星型)
│       └── 不推荐
│           └── 分支造成阻抗不连续
│
└── 多驱动 (总线)
    └── 中间端接
        └── 需要开关控制使能
```

## 3. 串扰分析

### 3.1 串扰机制

**定义 3.1 (串扰系数)**

```
容性耦合:
  Ic = Cc × dV/dt

  近端串扰 (NEXT): + polarity
  远端串扰 (FEXT): - polarity

感性耦合:
  VL = Lm × dI/dt

  近端串扰: - polarity
  远端串扰: + polarity

总串扰:
  近端: NEXT = (Cc×Z0 + Lm/Z0)/(4×td×trise)
  远端: FEXT = (Cc×Z0 - Lm/Z0)×length/(2×trise)

对于均匀介质 ( stripline ):
  Cc×Z0 = Lm/Z0
  远端串扰 ≈ 0

3W规则:
  间距 ≥ 3×线宽, 串扰 < 5%

防护地线:
  两侧地线可减少串扰50-70%
  地线需打过孔到参考平面
```

### 3.2 串扰减少策略矩阵

| 技术 | 效果 | 代价 | 适用场景 |
|------|------|------|----------|
| **增加间距** | 1/d²减少 | 面积增加 | 通用 |
| **减少平行长度** | 线性减少 | 布线复杂 | 通用 |
| **地平面隔离** | 50-70% | 需要过孔 | 敏感信号 |
| **差分对** | 共模抑制 | 2倍布线 | 高速串行 |
| **降低边沿速率** | 线性减少 | 延迟增加 | 非关键时序 |

## 4. 高速接口设计

### 4.1 DDR4 设计规则

```
DDR4 信号完整性关键参数

拓扑:
  Fly-by (菊花链) 地址/命令/控制
  T型 (双rank) 或 点对点 (单rank) 数据

布线规则:
  ┌─────────────────────────────────────────────────────┐
  │ 参数          │ 规格                                │
  ├─────────────────────────────────────────────────────┤
  │ 数据速率      │ 1600-3200 MT/s (DDR4)              │
  │ 位宽          │ 64-bit (ECC:72)                    │
  │ 走线阻抗    │ 单端40Ω, 差分80Ω                   │
  │ 长度匹配    │ Byte内±10mil, 组间±100mil         │
  │ 间距        │ 3W (线宽3倍)                        │
  │ 过孔        │ 最多2个, 背钻减少stub              │
  └─────────────────────────────────────────────────────┘

等长匹配:
  DQ-DQS: ±0.5ps (同一byte)
  DQS-CLK: ±100ps (读时序)
  Address-CLK: ±100ps

ODT (On-Die Termination):
  RZQ = 240Ω ± 1%
  ODT值: RZQ/2, RZQ/4, RZQ/6, RZQ/8

  写操作: 接收端开启ODT
  读操作: PHY端开启ODT
```

### 4.2 PCIe 设计要点

```
PCIe Gen3/4/5 设计规则

物理层:
  Gen3: 8 GT/s, 128b/130b编码
  Gen4: 16 GT/s
  Gen5: 32 GT/s

通道损耗预算:
  ┌───────────────────────────────────────┐
  │ 代际   │ 总损耗   │ 插槽损耗 │ CEM损耗│
  ├───────────────────────────────────────┤
  │ Gen3   │ 22 dB   │ 9.5 dB   │ 20 dB  │
  │ Gen4   │ 28 dB   │ 16 dB    │ 28 dB  │
  │ Gen5   │ 36 dB   │ 20 dB    │ 36 dB  │
  └───────────────────────────────────────┘

布线要求:
  - 差分阻抗: 85Ω ± 10%
  - 对内延迟: < 1ps (约5mil)
  - 过孔stub: 背钻至 < 10mil
  - 交流耦合: 100-220nF, 靠近接收端

参考时钟:
  - 100MHz ± 300ppm
  - 扩频: -0.5% down spread
  - 抖动: < 1ps RMS (Gen4)
```
