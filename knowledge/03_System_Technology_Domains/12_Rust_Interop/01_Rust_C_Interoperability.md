# Rust 与 C 互操作性深度指南

> **层级**: L4 (方法论层)  
> **目标**: 掌握 Rust 与 C 代码互操作的技术

---

## 1. FFI 基础

### 1.1 类型映射

Rust 与 C 的类型对应关系：
- int -> c_int
- void* -> *mut c_void
- struct -> #[repr(C)] struct

### 1.2 外部函数声明

使用 extern "C" 声明 C 函数，并提供安全包装。

## 2. 内存管理

处理 C 分配和 Rust 分配的内存，确保正确释放。

## 3. 复杂类型转换

结构体映射、不透明指针、回调函数处理。
