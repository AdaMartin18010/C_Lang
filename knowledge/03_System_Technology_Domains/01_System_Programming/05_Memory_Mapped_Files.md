# 内存映射文件 (Memory-Mapped Files)

> **层级定位**: 03 System Technology Domains / 01 System Programming

## 概述

内存映射文件 (`mmap`) 允许将文件内容直接映射到进程的虚拟地址空间，实现文件 I/O 和共享内存的高效操作。
