## 安全组件

### TPM (可信平台模块)

提供硬件级别的安全功能：

- 安全密钥存储
- 平台完整性测量
- 远程证明

### HSM (硬件安全模块)

企业级密钥管理：

- FIPS 140-2认证
- 高性能加密操作
- 密钥生命周期管理
