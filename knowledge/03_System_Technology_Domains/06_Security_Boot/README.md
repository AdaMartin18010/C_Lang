## 安全启动流程

```
Boot ROM → BL1 → BL2 → BL31 → BL32 → BL33
   ↓       ↓     ↓      ↓      ↓      ↓
 固化    信任   加载   运行时  TEE   UEFI
 代码    锚点   固件   服务    固件  /Linux
```
