# C到硬件映射

本目录深入探讨如何将C语言代码映射到硬件实现，是连接软件编程与硬件设计的桥梁。

## 内容概览

| 文件 | 描述 |
|------|------|
| `01_C_for_Synthesis.md` | 可综合C代码的编写规范与限制 |
| `02_Pointer_to_Memory_Interface.md` | 指针到内存接口的映射机制 |
| `03_Loops_to_Pipelines.md` | 循环结构到硬件流水线的转换 |
| `04_Functions_to_Modules.md` | 函数到硬件模块的映射策略 |

## 核心概念

- **可综合性**: 理解哪些C语言结构可以转换为硬件
- **内存接口**: 指针和数组在硬件中的实现方式
- **流水线并行**: 将顺序循环转换为并行硬件流水线
- **模块化设计**: 函数如何映射为独立的硬件模块
