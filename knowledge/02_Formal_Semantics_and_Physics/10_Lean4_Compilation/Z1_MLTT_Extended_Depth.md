## 扩展A：判断理论的深层性质

### 定理A.1 (上下文置换性)

**陈述**: 若Γ₁, x:A, y:B, Γ₂ ⊢ J且x ∉ FV(B)，则Γ₁, y:B, x:A, Γ₂ ⊢ J。

**证明**:
对Γ₁, x:A, y:B, Γ₂ ⊢ J的推导进行结构归纳。

基本情况：

- 变量规则：直接由上下文查找，置换不影响。
- 其他原子规则：同理。

归纳步骤：

- Π-形成：假设在Γ₁, x:A, y:B, Γ₂ ⊢ A type和Γ₁, x:A, y:B, Γ₂, z:A ⊢ B type。
  由归纳假设，可置换为Γ₁, y:B, x:A, Γ₂ ⊢ A type。
  对于B，注意z是新鲜的，且x ∉ FV(B)保证不影响。

结论：上下文置换保持判断有效性。∎

### 定理A.2 (弱化保守性)

**陈述**: Γ ⊢ J当且仅当Γ, x:A ⊢ J（x不在J中自由出现）。

**证明**:
(⇒) 方向：由弱化引理直接得。
(⇐) 方向：对Γ, x:A ⊢ J的推导归纳。

- 若x从未使用，则删除x的假设不影响推导。
- 由于x ∉ FV(J)，最终结论J不依赖x。

结论：无关假设可安全删除。∎

## 扩展B：Π-类型的深层性质

### 定理B.1 (Π-类型的函子性)

**陈述**: Π-类型在依赖关系下保持函数结构。

**形式化**:
给定f : A → A\'和g : (a:A) → B a → B\' (f a)，
存在mapΠ : (Πa:A. B a) → (Πa\':A\'. B\' a\')。

**构造**:

```
mapΠ h := λa\'. g (inv f a\') (h (inv f a\'))
```

其中inv f是f的逆（假设f是同构）。

**证明**:
需验证类型正确性：

- h : Πa:A. B a
- 对任意a\' : A\'，inv f a\' : A
- h (inv f a\') : B (inv f a\')
- g (inv f a\') : B (inv f a\') → B\' a\'
- 组合得B\' a\'

结论：Π-类型在依赖关系下是协变函子。∎

### 定理B.2 (函数外延性的推导)

**陈述**: 在MLTT+函数外延性公理(FunExt)中，(∀x. f x = g x) → f = g。

**证明**:
假设h : ∀x. f x = g x。

构造路径：

```
λi. λx. (h x) i
```

验证：

- 在i = i0：(h x) i0 = f x（由h的定义）
- 在i = i1：(h x) i1 = g x
- 因此λi. λx. (h x) i : f = g

结论：函数外延性成立。∎

## 扩展C：W-类型的通用性质

### 定理C.1 (W-类型的初始代数)

**陈述**: W(x:A).B(x)是多项式函子P(X) = Σ(x:A).(B(x) → X)的初始代数。

**证明**:
需证：

1. W有代数结构：sup : P(W) → W
2. 对任意代数α : P(C) → C，存在唯一同态fold α : W → C

**构造fold**:

```
fold α (sup (a, f)) := α (a, λb. fold α (f b))
```

**唯一性证明**:
设h : W → C是另一个同态。
对w : W进行归纳：

- w = sup (a, f)
- h w = h (sup (a, f)) = α (a, λb. h (f b))
- 由归纳假设，h (f b) = fold α (f b)
- 因此h w = fold α w

结论：W是初始代数。∎

### 定理C.2 (W-类型的递归原理)

**陈述**: 存在归纳原理：

```
W-rec : (P : W → Type) →
        ((a:A)(f:B(a)→W)(ih:(b:B(a))→P(f b)) → P(sup a f)) →
        (w:W) → P w
```

**证明**:
构造：

```
W-rec P h (sup a f) := h a f (λb. W-rec P h (f b))
```

验证类型正确性：

- h的第二个参数需要ih : (b:B(a)) → P(f b)
- 递归调用W-rec P h (f b)恰好提供P(f b)

结论：递归原理有效。∎

## 扩展D：证明擦除的语义保持

### 定理D.1 (擦除的完备性)

**陈述**: 擦除函数ε是满射到可计算子类型的。

**证明**:
对任意可计算类型T，存在Lean类型A使得ε(A) = T。

构造：

- 若T是原始类型（Nat, Bool等），取A = T
- 若T = T₁ → T₂，取A = ε⁻¹(T₁) → ε⁻¹(T₂)
- 若T是积类型，取A = Σ类型

结论：擦除覆盖所有可计算类型。∎

### 定理D.2 (擦除的最小性)

**陈述**: 擦除删除所有证明内容，保留最小计算信息。

**证明**:
对任意M : Prop，ε(M) = Unit。

由证明无关性，所有M的证明映射到Unit的唯一元素。
因此擦除是到可计算域的最小映射。∎

## 扩展E：归一化定理

### 定理E.1 (强归一化)

**陈述**: MLTT中所有良类型项都有有限归约序列到范式。

**证明概要**:
使用Tait-Girard可计算性方法。

定义可计算性谓词Comp(A) ⊆ A：

- 对Nat：Comp(Nat) = {t : Nat | t →* n for some n}
- 对Π(x:A).B：Comp(Π(x:A).B) = {t | ∀a ∈ Comp(A), t a ∈ Comp(B[a/x])}

**引理**: 若Γ ⊢ t : A，则t ∈ Comp(A)。

**证明**: 对Γ ⊢ t : A进行归纳。

结论：强归一化成立。∎

## 扩展F：与集合论的关系

### 定理F.1 (集合论模型)

**陈述**: MLTT可在ZFC+大基数中解释。

**构造**:

- Type n ↦ V_{κ_n}，其中κ_n是第n个不可达基数
- Π(x:A).B(x) ↦ ∏_{a∈A} B(a)
- Σ(x:A).B(x) ↦ ∑_{a∈A} B(a)

**验证**:
检查所有类型构造在V_{κ_n}中封闭。

结论：MLTT相对于ZFC+大基数一致。∎

## 扩展G： decidability结果

### 定理G.1 (类型检查可判定)

**陈述**: 给定Γ, t, A，判定Γ ⊢ t : A是可判定的。

**证明**:
算法：

1. 对t进行语法分析
2. 根据语法结构应用类型规则
3. 检查约束可满足性

由强归一化，类型检查终止。
由合流性，结果唯一。

结论：类型检查可判定。∎

### 定理G.2 (相等性判定)

**陈述**: 给定Γ ⊢ a, b : A，判定Γ ⊢ a = b : A是可判定的。

**证明**:
算法：

1. 将a和b归约到范式
2. 比较范式是否语法相等

由强归一化，归约终止。
由范式唯一性，判定正确。

结论：相等性可判定。∎

**证明**:

结论：MLTT的范畴语义是LCCC。∎
