## 三、定理组合规则

### 3.1 顺序组合

**规则3.1 (顺序组合)**:

```
T₁ : A → B    T₂ : B → C
─────────────────────────
T₂ ∘ T₁ : A → C
```

**例子**:

```
类型保持性 : 归约前→归约后
进展性 : 可归约→值
────────────────────────────
类型安全 = 类型保持性 ∘ 进展性
```

### 3.2 并行组合

**规则3.2 (并行组合)**:

```
T₁ : A₁ → B₁    T₂ : A₂ → B₂
─────────────────────────────────
T₁ × T₂ : (A₁, A₂) → (B₁, B₂)
```

**例子**:

```
闭包转换正确性 : 高阶→一阶
尾调用优化正确性 : 递归→循环
────────────────────────────────
函数式优化 = 闭包转换 × 尾调用优化
```

### 3.3 条件组合

**规则3.3 (条件组合)**:

```
T₁ : P → Q    T₂ : Q → R
─────────────────────────
if P then T₂ ∘ T₁ : P → R
```

**例子**:

```
可内联(M) → 内联正确性
不可内联(M) → 直接调用正确性
───────────────────────────────
函数调用优化 = if 可内联 then 内联 else 直接调用
```

### 3.4 归纳组合

**规则3.4 (归纳组合)**:

```
Base : P(0)
Step : ∀n. P(n) → P(n+1)
──────────────────────────
Induction : ∀n. P(n)
```

**例子**:

```
基本情况 : 空项良类型
归纳步骤 : 若子项良类型，则复合项良类型
──────────────────────────────────────────
所有项类型安全
```

## 四、证明技术模式

### 4.1 结构归纳模式

**适用**: 语法结构、类型规则

**模式**:

```
Goal: ∀M : Term. P(M)

Proof:
  Base Cases:
    - M = Var(x): 证明 P(Var)
    - M = Const(c): 证明 P(Const)

  Inductive Cases:
    - M = λx.M': IH(M') ⊢ P(λx.M')
    - M = M₁ M₂: IH(M₁), IH(M₂) ⊢ P(M₁ M₂)
```

**应用定理**: 替换引理、类型保持性

### 4.2 良基归纳模式

**适用**: 递归定义、归约关系

**模式**:

```
Goal: ∀M. P(M)

Proof:
  对(M, <)进行良基归纳
  其中 < 是良基关系(如归约关系)

  Case: M是极小元
    - 证明P(M)直接成立

  Case: M不是极小元
    - ∀M' < M. P(M')  (归纳假设)
    - 证明 P(M)
```

**应用定理**: 强规范化、终止性

### 4.3 逻辑关系模式

**适用**: 高阶函数、多态

**模式**:

```
定义逻辑关系:
  V⟦τ⟧ : Value × Value → Prop
  E⟦τ⟧ : Term × Term → Prop

证明:
  1. 基本引理: ⊢ M : τ ⟹ M E⟦τ⟧ M
  2. 兼容性: M E⟦τ⟧ M' ⟹ C[M] E⟦τ'⟧ C[M']
  3. 得出性质P
```

**应用定理**: 规范化、参数性

### 4.4 Galois连接模式

**适用**: 抽象解释、优化

**模式**:

```
给定: α : Concrete → Abstract
      γ : Abstract → Concrete

证明 α ⊣ γ:
  1. α(γ(a)) ⊑ a  (压缩)
  2. c ⊑ γ(α(c))  (扩展)

应用: 优化正确性、分析安全性
```

**应用定理**: 数据流分析、抽象解释
