## 软件科学基础

### 3.1 范畴论语义

**定理3.1 (笛卡尔闭范畴)**:
```
简单类型λ演算 ↔ CCC
```

### 3.2 单子理论

**定理3.2 (Moggi)**:
```
计算效应 = 单子
```

**对应**:
| 效应 | 单子 |
|:-----|:-----|
| 状态 | State |
| 异常 | Maybe/Either |
| 输入输出 | IO |
| 非确定性 | List |
