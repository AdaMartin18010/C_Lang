# Martin-Löf 类型论与 Lean 4 内核

> **层级**: L6 (元理论层)

## 一、判断与命题

Martin-Löf 确立了判断的四种形式：

1. A type
2. A = B type
3. a : A
4. a = b : A

这与经典逻辑的命题-真值二元对立形成根本区别。

## 二、归纳类型

W-类型形式化归纳构造：
`
W_{x:A} B(x)
`

Lean 4 的 inductive 是 W-类型的语法糖。

## 三、宇宙层级

避免罗素悖论的方案：
`
Type_0 : Type_1 : Type_2 : ...
`

## 四、证明擦除

Prop vs Type 分离：

- Type：有计算内容
- Prop：只有存在性（编译时擦除）

## 五、对 C 编译的影响

1. 宇宙层级完全擦除
2. 证明项完全移除
3. 归纳类型扁平化为 C 结构体
