## 关系代数形式化

### 1.1 关系模型

**定义1.1 (关系)**:
```
关系 R ⊆ D₁ × D₂ × ... × Dₙ
其中Dᵢ是域
```

**定理1.1 (关系代数完备性)**:
```
{σ, π, ∪, -, ×} 是关系完备的操作集
```

### 1.2 关系演算

**元组关系演算**:
```
{ t | φ(t) }
其中φ是逻辑公式
```

**定理1.2 (等价性)**:
```
关系代数表达能力 ≡ 安全关系演算
```

## 索引与查询优化

### 3.1 B+树索引

**定理3.1 (B+树查询复杂度)**:
```
查找: O(log_B N)
插入: O(log_B N)
范围查询: O(log_B N + K)
```

### 3.2 查询优化

**定理3.2 (连接顺序优化)**:
```
左深树计划空间: O(n!)
 bush树计划空间: Ω(n!)
```

**动态规划算法**: 最优子结构保证

## 事务与并发

### 4.1 ACID属性

**原子性**: 全做或全不做
**一致性**: 约束保持
**隔离性**: 并发透明
**持久性**: 提交即保存

### 4.2 隔离级别

| 级别 | 脏读 | 不可重复读 | 幻读 |
|:-----|:----:|:----------:|:----:|
| READ UNCOMMITTED | ✓ | ✓ | ✓ |
| READ COMMITTED | ✗ | ✓ | ✓ |
| REPEATABLE READ | ✗ | ✗ | ✓ |
| SERIALIZABLE | ✗ | ✗ | ✗ |

### 4.3 两阶段锁定

**定理4.1 (2PL可串行化)**:
```
严格两阶段锁定产生可串行化调度
```
