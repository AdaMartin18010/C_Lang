## 五、跨课程关联网络

```

    │       │
    │       ├──→ 编译器 (CS143, 15-411)
    │       │       │
    │       │       └──→ 代码生成 (Z3-Z5)
    │       │
    │       └──→ 类型论 (Z1, Z2, Z10)
    │               │
    │               └──→ 形式化验证 (CS320)
    │
    ├──→ 系统 (CS61C, 6.004)
    │       │
    │       ├──→ 操作系统 (CS162)
    │       │       └──→ 运行时 (RUNTIME)
    │       │
    │       └──→ 分布式 (6.824)
    │               └──→ 一致性理论
    │
    └──→ 理论 (6.042, CS170)
            │
            └──→ 算法 (6.046)
```
