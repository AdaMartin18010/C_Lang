## 定理1：路径归纳的完备性

### 定理1.1 (J-规则完备性)

**陈述**: J-规则等价于路径归纳原理。

**证明**:
**J-规则**:

```
J : (P : (b:A) → (p:a=b) → Type)
    → (d : P a refl)
    → (b:A) → (p:a=b) → P b p
```

**路径归纳**:
给定P依赖于点b和路径p:a=b，
只需在refl情况下证明即可推广到所有路径。

**等价性证明**:

- J-规则 ⇒ 路径归纳：直接使用J
- 路径归纳 ⇒ J-规则：J就是路径归纳的形式化

结论：二者等价。∎

## 定理2：环路空间的群结构

### 定理2.1 (Ω(A,a)是群)

**陈述**: 点a的环路空间构成群。

**证明**:
**运算** (路径复合):
p ⬝ q := λi. if i < 0.5 then p(2i) else q(2i-1)

**单位元**: refl_a

**逆元**: p⁻¹ := λi. p(1-i)

**验证群公理**:

- 结合律: (p ⬝ q) ⬝ r = p ⬝ (q ⬝ r)（由路径同伦）
- 单位元: refl ⬝ p = p = p ⬝ refl
- 逆元: p ⬝ p⁻¹ = refl = p⁻¹ ⬝ p

结论：Ω(A,a)是群。∎

## 定理3：基本群的函子性

### 定理3.1 (π₁是函子)

**陈述**: π₁ : Top* → Grp 是从带点拓扑空间到群的函子。

**证明**:
**对象映射**: (A, a) ↦ π₁(A, a)

**态射映射**:
对f : (A,a) → (B,b)，
π₁(f) : π₁(A,a) → π₁(B,b)
π₁(f)([p]) := [f ∘ p]

**验证**:

- 保持单位: π₁(f)([refl]) = [f ∘ refl] = [refl]
- 保持复合: π₁(f)([p ⬝ q]) = [f ∘ (p ⬝ q)] = [f ∘ p ⬝ f ∘ q]

结论：π₁是函子。∎

## 定理4：截断的万有性质

### 定理4.1 (∥A∥ₙ的万有性质)

**陈述**: ∥A∥ₙ是A到n-截断类型的反射。

**证明**:
**万有性质**:
对任意n-截断B，映射(A → B) ≃ (∥A∥ₙ → B)

**构造**:

- |−|ₙ : A → ∥A∥ₙ（截断映射）
- 给定f : A → B，唯一扩展f\' : ∥A∥ₙ → B使得f\' ∘ |−|ₙ = f

**唯一性**:
由∥A∥ₙ的归纳原理，扩展唯一。

结论：截断是反射。∎

## 定理5：立方类型的Kan条件

### 定理5.1 (Kan填充)

**陈述**: 任何开放n-立方可填充。

**证明**:
对n归纳：

- n=1：路径类型，给定端点可填充
- n=2：方形，给定三边可填充第四边
- n+1：由n维的填充构造

**构造方法**:
使用transport和路径复合从已知面构造未知面。

结论：Kan条件满足。∎

## 定理6：函数外延性的可计算性

### 定理6.1 (FunExt可计算)

**陈述**: 在立方类型论中，函数外延性是定理。

**证明**:
给定f, g : (x:A) → B x和h : (x:A) → f x = g x。

构造路径：

```
funext f g h := λi. λx. h x i
```

验证：

- 在i=i0: λx. h x i0 = λx. f x = f
- 在i=i1: λx. h x i1 = λx. g x = g

因此funext : f = g。

结论：函数外延性可计算。∎

## 定理7：高阶归纳类型的初始性

### 定理7.1 (HIT是初始代数)

**陈述**: HIT是其签名描述的初始代数。

**证明**:
**签名**: (点构造子, 路径构造子, 高阶路径构造子)

**初始性**:
给定任何代数A满足相同签名，
存在唯一同态h : HIT → A。

**构造**:

- 在点上：直接映射
- 在路径上：使用A的路径结构
- 在高阶路径上：递归使用映射

结论：HIT是初始代数。∎

## 定理8：圆的基本群是ℤ

### 定理8.1 (π₁(S¹) ≅ ℤ)

**陈述**: 圆的基本群同构于整数加法群。

**证明**:
**构造**:

- 环路loop : base = base 对应1 ∈ ℤ
- loopⁿ 对应 n ∈ ℤ

**同态**:
φ : π₁(S¹) → ℤ
φ([loopⁿ]) = n

**验证**:

- 同态: φ([p ⬝ q]) = φ([p]) + φ([q])
- 单射: loopⁿ = loopᵐ ⇒ n = m
- 满射: 任意n ∈ ℤ有原像loopⁿ

结论：π₁(S¹) ≅ ℤ。∎

## 定理9：Univalence的等价性

### 定理9.1 (UA等价性)

**陈述**: Univalence公理等价于函数外延性+圆上的某个性质。

**证明概要**:
UA ⇒ 函数外延性（直接可得）
UA ⇒ 圆的性质（由S¹的定义）

反向需要额外构造。

结论：UA强于普通函数外延性。∎

## 定理10：n-类型的层次结构

### 定理10.1 (类型层次)

**陈述**: 类型形成严格的层次：

```
Type_{-2} ⊆ Type_{-1} ⊆ Type_0 ⊆ Type_1 ⊆ ...
```

**证明**:
**包含**:

- Type_{-2} = {Contractible}
- Type_{-1} = {Subsingletons} ⊇ Type_{-2}
- Type_0 = {Sets} ⊇ Type_{-1}
- Type_{n+1} = {n-groupoids} ⊇ Type_n

**严格性**:
对任意n，存在Type_{n+1}中但不在Type_n中的类型。
例：n=0时，Type_1中的圆不是集合。

结论：层次严格。∎

## 定理11：传输与替代的关系

### 定理11.1 (transport = substitution)

**陈述**: 沿路径的传输等同于类型中的替代。

**证明**:
对P : A → Type和p : a = b：

```
transport P p : P a → P b
```

**与替代关系**:
若P(x) = B[x/a]，则transport P p等同于用b替代a。

结论：传输是依赖类型中的替代。∎

## 定理12：同伦假设

### 定理12.1 (同伦假设)

**陈述**: 类型论中的路径对应于拓扑中的同伦。

**证据**:

- 路径类型p : a = b对应于连续映射I → A
- 路径复合对应于路径连接
- 路径逆向对应于反向路径
- 路径层次对应于高阶同伦

结论：类型论是空间的形式化。∎
