## 一、MetaM基础形式化

### 1.1 定义

MetaM是Lean 4的元编程单子，形式化为：

```
MetaM α = CoreM (StateT Meta.State IO α)
```

**构成**:

- CoreM: 核心上下文 (环境、选项)
- StateT Meta.State: 元状态 (目标、局部上下文)
- IO: 输入输出能力

### 1.2 单子律定理

**定理1.1 (左单位元)**:

```
return a >>= f = f a
```

**证明**: 由StateT单子定义直接可得。∎

**定理1.2 (右单位元)**:

```
m >>= return = m
```

**证明**: 状态传递保持。∎

**定理1.3 (结合律)**:

```
(m >>= f) >>= g = m >>= (λx. f x >>= g)
```

**证明**: StateT单子满足结合律。∎
