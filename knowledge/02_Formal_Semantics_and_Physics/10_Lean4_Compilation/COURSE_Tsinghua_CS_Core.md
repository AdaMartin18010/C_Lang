## 数据结构形式化

### 1.1 红黑树

**定理1.1 (红黑树平衡性)**:
```
从根到叶的所有路径长度差 ≤ 2倍
```

**定理1.2 (操作复杂度)**:
```
插入、删除、查找: O(log n)
```

### 1.2 B+树

**定理1.3 (B+树高度)**:
```
高度 = O(log_m n) 其中m是阶数
```
