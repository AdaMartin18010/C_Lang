## 【哲学·相等性的演变】

### 从命题到构造

**命题相等** (经典逻辑):

- a = b 是真或假
- 无计算内容

**判断相等** (Martin-Löf):

- a = b : A 是判断
- 证明是构造

**路径相等** (HoTT):

- a = b 是空间中的路径
- 路径本身有结构

### 不可区分原理

**经典**: 相等对象在所有性质上不可区分
**HoTT**: 路径保持所有构造

## 【科学·HoTT形式化】

### 1. 路径类型

**定义**: 对于a, b : A，路径类型 a =_A b

**构造子**:

```
refl : ∀ (a : A), a = a
```

**消去** (路径归纳/J-消去):

```
J : (P : (b : A) → (p : a = b) → Type)
    → (d : P a refl)
    → (b : A) → (p : a = b)
    → P b p
```

### 2. 高阶路径

**定义**: 路径的路径形成高阶群胚结构

```
Ω(A, a) := a = a                    -- 环路空间
Ω²(A, a) := refl_a = refl_a         -- 2维环路
Ωⁿ(A, a) := Ωⁿ⁻¹(Ω(A, a), refl)    -- n维环路
```

**同伦群**:

```
π₁(A, a) := ∥Ω(A, a)∥₀              -- 基本群
πₙ(A, a) := ∥Ωⁿ(A, a)∥₀            -- 高阶同伦群
```

### 3. 截断(Truncation)

**定义**: n-截断使类型成为n-群胚

```
∥A∥₋₂ := ⊤                    -- 可收缩
∥A∋₋₁ := ¬¬A                  -- 命题截断
∥A∥₀  := A / (λ x y => ∥x = y∥₋₁)  -- 集合截断
∥A∥ₙ₊₁ := A / (λ x y => ∥x = y∥ₙ)
```

**性质**:

- ∥A∥₀ 是集合（无高阶路径）
- ∥A∥₋₁ 是命题（至多单元素）

## 【工程·立方类型论】

### 1. De Morgan立方类型论

**区间I**: 特殊类型，端点i0, i1

**路径类型**:

```
Path A a b := (i : I) → A [i0 ↦ a, i1 ↦ b]
```

**路径抽象**:

```
λ i. t : Path A t[i0/i] t[i1/i]
```

**路径应用**:

```
p i : A，其中 i : I
```

### 2. 填充操作

**Kan条件**: 任何开放立方可填充

**示例 (方形)**:

```
给定:
  p : a = b
  q : a = c
  r : b = d

填充给出:
  s : c = d
  且 square : p ⬝ s = q ⬝ r
```

### 3. 可计算函数外延性

**定理**: funext可计算

**证明**:

```
funext : (f g : (x : A) → B x) → ((x : A) → f x = g x) → f = g
funext f g p := λ i x. p x i
```

这是真正的λ项，不是公理！

## 【可视化·同伦层级】

```
类型A的层级结构:

Level ∞: 高阶群胚
    ↓
Level 1: 群胚 (1-截断)
    ↓
Level 0: 集合 (0-截断)
    ↓
Level -1: 命题 (-1-截断)
    ↓
Level -2: 可收缩 (单点)

截断操作 ∥A∥ₙ 将A坍缩到第n层
```

## 【定理·路径归纳】

### J-规则的正确性

**定理**: J-规则与路径归纳等价。

**证明**:

```
J P d b p 定义了在路径p上的归纳

步骤1: 基本情况 p = refl
  J P d a refl = d
  由refl的定义

步骤2: 归纳步骤
  任意路径p可变形为refl（因为区间可缩）
  由同伦延拓性质

结论: J规则有效 ∎
```
