## 扩展A：合流性定理的完整证明

### 定理A.1 (Church-Rosser)

**陈述**: CoC满足合流性。

**证明** (Tait-Martin-Löf方法):

定义并行归约⇒：

- 变量：x ⇒ x
- β：若M ⇒ M\'且N ⇒ N\'，则(λx:A.M)N ⇒ M\'[N\'/x]
- 应用：若M ⇒ M\'且N ⇒ N\'，则MN ⇒ M\'N\'
- 抽象：若A ⇒ A\'且M ⇒ M\'，则λx:A.M ⇒ λx:A\'.M\'
- 乘积：若A ⇒ A\'且B ⇒ B\'，则Πx:A.B ⇒ Πx:A\'.B\'

**引理A.1.1**: ⇒满足菱形性质。

**证明**: 对M ⇒ M₁和M ⇒ M₂的结构归纳，构造M\'使得M₁ ⇒ M\'和M₂ ⇒ M\'。

**引理A.1.2**: →*是⇒的传递闭包。

**定理证明**:
由菱形性质和Newman引理，⇒*是合流的。
由于→ ⊆ ⇒ ⊆ →*，β规约→*也是合流的。∎

## 扩展B：强归一化的Girard方法

### 定理B.1 (强归一化)

**陈述**: CoC中所有良类型项都是强归一化的。

**证明** (可计算性谓词法):

定义可计算性谓词Comp_τ ⊆ [[τ]]：

**基础情况**:

- Comp_Prop(M) := M →*⊤ 或 M →* ⊥
- Comp_Type(M) := M是规范形式

**函数情况**:

- Comp_{Πx:A.B}(M) := ∀N ∈ Comp_A, MN ∈ Comp_{B[N/x]}

**关键引理B.1.1**: (可计算性蕴含强归一化)
若M ∈ Comp_τ，则M是强归一化的。

**证明**: 由Comp定义，Comp中的元素必须能归约到规范形式。

**关键引理B.1.2**: (抽象的可计算性)
若对任意N ∈ Comp_A，M[N/x] ∈ Comp_B，则λx:A.M ∈ Comp_{Πx:A.B}。

**主定理证明**:
对Γ ⊢ M : τ进行归纳，证明M ∈ Comp_τ。

- 变量情况：由上下文假设
- 抽象情况：由引理B.1.2
- 应用情况：由Comp定义

结论：所有良类型项强归一化。∎

## 扩展C：证明无关性的形式化

### 定理C.1 (证明无关性)

**陈述**: 对P : Prop和p, q : P，有p ≡ q。

**证明**:
在CoC中，Prop被解释为单元素集合（或proof-irrelevant universe）。

**语义解释**:

- [[Prop]] = {⊤} 或 单元素偏序集
- [[P]] = {*} 对任意P : Prop

因此任意p, q : P都映射到*，故p ≡ q。∎

### 定理C.2 (证明擦除的保持性)

**陈述**: 若Γ ⊢ M : P且P : Prop，则ε(M) = *。

**证明**:
由擦除定义：

- ε(Prop) = Unit
- 对M : P，ε(M)是Unit的唯一元素

结论：证明擦除为Unit。∎

## 扩展D：宇宙层次的安全性

### 定理D.1 (宇宙一致性)

**陈述**: 不存在项M使得⊢ M : Type_n : Type_n。

**证明**:
由宇宙层次规则：

- Type_n : Type_{n+1}
- 不存在Type_n : Type_n

假设存在M : Type_n : Type_n，则违反层次结构。

结论：Type_n ∉ Type_n，避免罗素悖论。∎

### 定理D.2 (累积性)

**陈述**: 若Γ ⊢ A : Type_n，则Γ ⊢ A : Type_{n+1}。

**证明**:
由宇宙规则：

```
Γ ⊢ A : Type_n
─────────────── (累积)
Γ ⊢ A : Type_{n+1}
```

结论：宇宙是累积的。∎

## 扩展E：表达力结果

### 定理E.1 (System F嵌入)

**陈述**: System F可嵌入CoC。

**证明**:
映射：

- System F类型变量α ↦ CoC变量α : Type_0
- ∀α.τ ↦ Πα:Type_0.τ
- λx:τ.e ↦ λx:τ.e

验证所有System F规则在CoC中可推导。

结论：CoC包含System F。∎

### 定理E.2 (高阶逻辑表达能力)

**陈述**: CoC可表达高阶逻辑。

**证明**:

- 命题：P : Prop
- 谓词：P : A → Prop
- 高阶谓词：P : (A → Prop) → Prop
- 量词：∀和∃可用Π和Σ编码

结论：CoC是高阶逻辑的元语言。∎

## 扩展H：提取算法完备性

### 定理H.1 (提取完备性)

**陈述**: 提取函数ε覆盖所有可计算程序。

**证明**:
对任意可计算程序p，存在CoC项M使得ε(M) = p。

构造：

- 使用CoC的原始递归能力
- 编码图灵机/λ演算
- 通过提取得到p

结论：提取是完备的。∎

### 定理H.2 (提取最优性)

**陈述**: 提取产生最小可计算表示。

**证明**:
提取删除所有：

- 证明项（Prop内容）
- 类型信息（宇宙层级）
- 冗余抽象

因此提取结果是最小可计算表示。∎
