### 3.2 知识图谱嵌入

**在文档中嵌入图谱**:

```yaml
---
knowledge_graph:
  nodes:
    - id: "operational_semantics"
      type: "theory"
      related: ["denotational_semantics", "axiomatic_semantics"]
    - id: "lambda_calculus"
      type: "application"
      related: ["functional_programming", "type_theory"]
  edges:
    - from: "operational_semantics"
      to: "denotational_semantics"
      relation: "equivalent_to"
---
```

### 3.3 自动一致性检查

**检查项**:

- [ ] 术语一致性
- [ ] 符号一致性
- [ ] 引用有效性
- [ ] 逻辑一致性
- [ ] 风格一致性

**自动化**:

```python
# 伪代码
def check_consistency(documents):
    errors = []
    for doc in documents:
        errors.extend(check_terminology(doc))
        errors.extend(check_notation(doc))
        errors.extend(check_references(doc))
    return errors
```
