# 语义学符号标准 (Semantic Notation Standard)

> **版本**: 1.0
> **适用范围**: 所有Lean 4编译知识库语义文档
> **状态**: 强制标准

---

## 1. 基本原则

### 1.1 一致性原则

所有语义文档必须使用本标准定义的符号，禁止混用其他符号系统。

### 1.2 清晰性原则

优先使用国际通用的标准符号，避免自定义符号。

### 1.3 可扩展性原则

符号系统应支持简单类型、多态类型、依赖类型等多种类型系统。

## 2. 操作语义 (Operational Semantics) 符号

### 2.1 归约关系

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `M → N` | `M \to N` | 一步归约 | M 一步归约为 N |
| `M →* N` | `M \to^* N` | 多步归约 | M 零步或多步归约为 N |
| `M →+ N` | `M \to^+ N` | 正多步归约 | M 一步或多步归约为 N |
| `M ⇓ v` | `M \Downarrow v` | 求值到值 | M 求值得到值 v |
| `M ↑` | `M \uparrow` | 发散 | M 不终止 |
| `M ↛` | `M \not\to` | 不可归约 | M 是正规形式 |

### 2.2 配置

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `⟨M, σ⟩` | `\langle M, \sigma \rangle` | 配置 | 项 M 在状态 σ 下 |
| `⟨M, σ⟩ → ⟨M', σ'⟩` | - | 配置转换 | 配置的一步转换 |

**注意**: 配置也可以用元组形式 `(M, σ)`，但推荐使用尖括号以区别于普通元组。

### 2.3 求值上下文

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `E[·]` | `E[\cdot]` | 求值上下文 | 带洞上下文 |
| `E[M]` | `E[M]` | 填充上下文 | 将 M 填入上下文 E |

### 2.4 语法范畴

| 符号 | 含义 | 说明 |
|:-----|:-----|:-----|
| `M, N, L` | 项 (Terms) | 程序的语法表示 |
| `V, v` | 值 (Values) | 正规形式的结果 |
| `E` | 表达式 (Expressions) | 广义项 |

## 3. 指称语义 (Denotational Semantics) 符号

### 3.1 语义括号

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `⟦M⟧` | `\llbracket M \rrbracket` | M 的指称 | 项 M 的语义解释 |
| `⟦M⟧ρ` | `\llbracket M \rrbracket_\rho` | 在环境 ρ 下的指称 | 含环境的语义解释 |
| `⟦τ⟧` | `\llbracket \tau \rrbracket` | 类型 τ 的指称 | 类型的语义域 |

**废弃符号**: `C⟦M⟧` (旧文档中的形式，统一改为 `⟦M⟧`)

### 3.2 环境与状态

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `ρ` | `\rho` | 环境/变量赋值 | 变量到值的映射 |
| `σ` | `\sigma` | 状态/存储 | 存储位置到值的映射 |
| `γ` | `\gamma` | 类型环境 | 类型变量到类型的映射 |
| `ρ[x↦v]` | `\rho[x \mapsto v]` | 环境更新 | 将 x 映射到 v 的新环境 |

**术语统一**:

- 使用 "State" (状态) 而不是 "Store" (存储)
- 使用 "Environment" (环境) 或 "估值映射"

### 3.3 语义域

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `D` | `D` | 语义域 | 指称的集合 |
| `V` | `V` | 值域 | 值的集合 |
| `⊥` | `\bot` | 底元素 | 未定义/发散 |
| `⊤` | `\top` | 顶元素 | 最大元素 |
| `⊑` | `\sqsubseteq` | 偏序 | 域上的序关系 |
| `⨆` | `\bigsqcup` | 最小上界 | 最小上确界 |

### 3.4 连续函数

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `[D → D]` | `[D \to D]` | 连续函数空间 | 从 D 到 D 的连续函数 |
| `fix f` | `\mathbf{fix}\, f` | 不动点 | 函数 f 的最小不动点 |

## 4. 公理语义 (Axiomatic Semantics) 符号

### 4.1 Hoare三元组

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `{P} C {Q}` | `\{P\}\, C \, \{Q\}` | Hoare三元组 | 若 P 成立，执行 C 后 Q 成立 |
| `⊢ {P} C {Q}` | `\vdash \{P\}\, C \, \{Q\}` | 语法可推导 | 可证明的Hoare三元组 |
| `⊨ {P} C {Q}` | `\vDash \{P\}\, C \, \{Q\}` | 语义有效 | 语义上成立的Hoare三元组 |

### 4.2 最弱前置条件与最强后置条件

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `wp(C, Q)` | `\mathit{wp}(C, Q)` | 最弱前置条件 | 使 C 执行后 Q 成立的最弱条件 |
| `sp(C, P)` | `\mathit{sp}(C, P)` | 最强后置条件 | C 执行后从 P 可达的最强条件 |

### 4.3 断言

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `P, Q, R` | - | 断言 | 描述程序状态的谓词 |
| `φ, ψ` | `\phi, \psi` | 逻辑公式 | 一般逻辑公式 |
| `P[x↦E]` | `P[x \mapsto E]` | 替换 | 将 P 中的 x 替换为 E |
| `P ⇒ Q` | `P \Rightarrow Q` | 逻辑蕴含 | P 蕴含 Q |
| `P ∧ Q` | `P \land Q` | 逻辑合取 | P 且 Q |
| `P ∨ Q` | `P \lor Q` | 逻辑析取 | P 或 Q |
| `¬P` | `\neg P` | 逻辑否定 | 非 P |
| `∀x.P` | `\forall x. P` | 全称量词 | 对所有 x, P |
| `∃x.P` | `\exists x. P` | 存在量词 | 存在 x 使 P |

### 4.4 状态满足

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `σ ⊨ P` | `\sigma \vDash P` | 状态满足 | 状态 σ 满足断言 P |
| `⊨ P` | `\vDash P` | 有效 | P 在所有状态下成立 |

## 5. 类型系统符号

### 5.1 类型与项

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `τ, σ, ρ` | `\tau, \sigma, \rho` | 类型 | 类型变量或类型表达式 |
| `M, N, L` | - | 项 | 程序/表达式 |
| `x, y, z` | - | 变量 | 项变量 |
| `c` | - | 常量 | 常量值 |

### 5.2 类型判断

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `Γ ⊢ M : τ` | `\Gamma \vdash M : \tau` | 类型判断 | 在上下文 Γ 下，M 具有类型 τ |
| `M : τ` | `M : \tau` | 类型标注 | M 的类型是 τ |
| `τ₁ → τ₂` | `\tau_1 \to \tau_2` | 函数类型 | 从 τ₁ 到 τ₂ 的函数 |
| `Πx:τ.σ` | `\Pi x:\tau.\sigma` | 依赖积类型 | 依赖函数类型 |
| `Σx:τ.σ` | `\Sigma x:\tau.\sigma` | 依赖和类型 | 依赖对类型 |
| `τ₁ × τ₂` | `\tau_1 \times \tau_2` | 积类型 | 对类型 |
| `τ₁ + τ₂` | `\tau_1 + \tau_2` | 和类型 | 变体/联合类型 |

### 5.3 类型上下文

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `Γ` | `\Gamma` | 类型上下文 | 变量到类型的映射 |
| `Γ, x:τ` | `\Gamma, x:\tau` | 上下文扩展 | 添加 x:τ 到 Γ |
| `∅` | `\emptyset` | 空上下文 | 空类型环境 |

## 6. 范畴论语义符号

### 6.1 范畴结构

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `C, D` | `\mathcal{C}, \mathcal{D}` | 范畴 | 范畴名称 |
| `ob(C)` | - | 对象类 | 范畴 C 的对象 |
| `hom(A, B)` | `\hom(A, B)` | 态射集 | 从 A 到 B 的态射 |
| `f: A → B` | `f: A \to B` | 态射 | 从对象 A 到 B 的态射 |
| `g ∘ f` | `g \circ f` | 态射复合 | f 然后 g |
| `id_A` | `\mathit{id}_A` | 恒等态射 | A 上的恒等态射 |

### 6.2 特殊构造

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `A × B` | `A \times B` | 范畴积 | 对象 A 和 B 的积 |
| `A + B` | `A + B` | 范畴余积 | 对象 A 和 B 的余积 |
| `[A ⇒ B]` | `[A \Rightarrow B]` | 指数对象 | 从 A 到 B 的函数对象 |
| `F: C → D` | `F: \mathcal{C} \to \mathcal{D}` | 函子 | 从范畴 C 到 D 的函子 |
| `α: F ⇒ G` | `\alpha: F \Rightarrow G` | 自然变换 | 函子 F 到 G 的自然变换 |
| `F ⊣ G` | `F \dashv G` | 伴随 | F 左伴随于 G |

## 7. 逻辑关系符号

### 7.1 逻辑谓词

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `V⟦τ⟧` | `V\llbracket\tau\rrbracket` | 值关系 | 类型 τ 的值关系 |
| `E⟦τ⟧` | `E\llbracket\tau\rrbracket` | 表达式关系 | 类型 τ 的表达式关系 |
| `G⟦Γ⟧` | `G\llbracket\Gamma\rrbracket` | 环境关系 | 上下文 Γ 的环境关系 |
| `(v, v') ∈ V⟦τ⟧` | - | 值相关 | 值 v 和 v' 在关系 V⟦τ⟧ 中 |

### 7.2 步进索引

| 符号 | LaTeX | 含义 | 说明 |
|:-----|:------|:-----|:-----|
| `≼_k` | `\preceq_k` | k步近似 | 在 k 步内行为相同 |
| `≃` | `\simeq` | 上下文等价 | 在所有上下文中行为相同 |

## 8. 废弃符号对照表

以下符号在旧文档中使用，已废弃，统一替换为新符号：

| 废弃符号 | 新符号 | 说明 |
|:---------|:-------|:-----|
| `C⟦M⟧` | `⟦M⟧` | 指称语义统一使用双括号 |
| `Store` | `State` | 统一使用 State |
| `σ'` | `σ'` | 保留，但明确 State 类型 |
| `⇓_n` | `⇓` 或 `⇓^{≤n}` | 步数限制使用上标说明 |
| `M ≡ N` | `M = N` (语法相等) 或 `M ≃ N` (语义相等) | 区分语法和语义相等 |

## 9. 命名约定

### 9.1 变量命名

| 类别 | 命名 | 例子 |
|:-----|:-----|:-----|
| 项变量 | 小写字母 | `x, y, z` |
| 项 | 大写字母 | `M, N, L` |
| 类型变量 | 希腊字母 | `τ, σ, ρ` |
| 环境/状态 | 希腊字母 | `ρ, σ, γ` |
| 范畴 | 花体字母 | `C, D` |
| 函子 | 大写字母 | `F, G` |
| 断言 | 大写字母 | `P, Q, R` |

### 9.2 字体约定

| 元素 | 字体 | LaTeX | 例子 |
|:-----|:-----|:------|:-----|
| 数学符号 | 斜体 | `$x$` | *x* |
| 常量/函数 | 直立 | `\mathit{fix}` | **fix** |
| 关键字 | 粗体 | `\mathbf{if}` | **if** |
| 范畴 | 花体 | `\mathcal{C}` | 𝒞 |

## 10. 使用示例

### 示例1: 操作语义

```
(λx.M) V → M[V/x]                          (β规约)

M → M'
───────── (上下文)
E[M] → E[M']

⟨x:=E, σ⟩ → ⟨skip, σ[x↦⟦E⟧σ]⟩             (赋值)
```

### 示例2: 指称语义

```
⟦x⟧ρ = ρ(x)
⟦λx.M⟧ρ = λv. ⟦M⟧ρ[x↦v]
⟦M N⟧ρ = ⟦M⟧ρ (⟦N⟧ρ)
```

### 示例3: 公理语义

```
{Q[x↦E]} x:=E {Q}                           (赋值规则)

{P} C₁ {R}   {R} C₂ {Q}
─────────────────────────                    (顺序规则)
{P} C₁; C₂ {Q}

wp(x:=E, Q) = Q[x↦E]
```

### 示例4: 类型系统

```
Γ ⊢ M : τ → σ    Γ ⊢ N : τ
───────────────────────────                   (函数应用)
Γ ⊢ M N : σ

Γ, x:τ ⊢ M : σ
───────────────────                            (函数抽象)
Γ ⊢ λx.M : τ → σ
```

### 示例5: 范畴论

```
F: C → D  是函子

F(id_A) = id_{F(A)}                          (保持恒等)
F(g ∘ f) = F(g) ∘ F(f)                       (保持复合)

F ⊣ G 表示 F 左伴随于 G
```

## 11. 扩展规则

### 11.1 新增符号流程

1. **提案**: 提交符号提案，说明用途和理由
2. **审查**: 审查是否与现有符号冲突
3. **实验**: 在单个文档中试用
4. **标准化**: 通过后更新本标准
5. **推广**: 更新所有文档

### 11.2 符号冲突处理

当引入新理论与现有符号冲突时：

1. 优先使用国际标准符号
2. 使用命名空间或下标区分 (如 `→_β` 和 `→_σ`)
3. 在文档开头明确说明符号重载
