## RISC-V指令集形式化

### 1.1 指令语义

**定义1.1 (指令)**:
```
Instr = (Opcode, Rd, Rs1, Rs2, Imm)
```

**执行语义**:
```
execute(add rd, rs1, rs2) = 
    Reg[rd] ← Reg[rs1] + Reg[rs2]
```

### 1.2 程序正确性

**定理1.1 (RISC-V程序正确性)**:
```
若程序P在仿真器中正确运行，
则在实际硬件上产生相同结果
```

## 流水线处理器

### 2.1 流水线阶段

```
IF → ID → EX → MEM → WB
```

### 2.2 数据冒险

**定理2.1 (前递正确性)**:
```
前递机制消除RAW冒险，无需停顿
```

**定理2.2 (停顿必要性)**:
```
负载-使用冒险需要至少1周期停顿
```

### 2.3 控制冒险

**定理2.3 (分支预测)**:
```
静态预测准确率 ≈ 50%
动态预测准确率 > 90%
```

## 内存层次结构

### 3.1 缓存形式化

**定义3.1 (缓存)**:
```
Cache = (S, E, B, m)
其中S=组数, E=路数, B=块大小
```

**定理3.1 (缓存命中时间)**:
```
命中时间 = O(1)
```

**定理3.2 (缓存未命中惩罚)**:
```
未命中惩罚 = 内存访问时间 + 传输时间
```

### 3.2 缓存一致性

**MESI协议**:
- M: Modified
- E: Exclusive
- S: Shared
- I: Invalid

**定理3.3 (MESI一致性)**:
```
MESI协议保证缓存一致性
```

## 并行处理

### 4.1 OpenMP模型

**定理4.1 (Amdahl定律)**:
```
Speedup = 1 / ((1-P) + P/N)
其中P是并行比例，N是处理器数
```

### 4.2 数据竞争

**定理4.2 (数据竞争检测)**:
```
存在数据竞争 ⟹ 程序非确定性
```
