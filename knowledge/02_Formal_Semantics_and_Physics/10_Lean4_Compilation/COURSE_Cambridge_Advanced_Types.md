## 归纳定义形式化

### 2.1 归纳类型

**定义2.1 (归纳类型)**:
```
inductive T : Type
| c₁ : A₁ → T
| c₂ : A₂ → T → T
```

**定理2.1 (归纳原理)**:
```
∀P: T → Prop.
(∀a. P(c₁ a)) → (∀a t. P t → P(c₂ a t)) → ∀t. P t
```

### 2.2 归纳-递归定义

**定理2.2 (Dybjer集合)**:
```
归纳-递归定义在Martin-Löf类型论中一致
```

## 同构与等价

### 3.1 结构恒等原理

**定理3.1 (同构即相等)**:
```
在Univalent Foundations中，
(A ≅ B) ≃ (A = B)
```

### 3.2 传输原理

**定理3.2 (运输)**:
```
transport : (P : A → Type) → (x = y) → P x → P y
```
