## 一、三种语义学的独立定义

### 1.1 操作语义 (Operational Semantics)

**定义**: 通过重写规则定义程序执行

**小步语义**:

```
(λx.M) V → M[V/x]           (β规约)
M → M\'  ⟹  M N → M\' N     (相容性)
```

**大步语义**:

```
⟨M, σ⟩ ⇓ ⟨V, σ\'⟩
表示在状态σ下M求值为V，新状态σ\'
```

### 1.2 指称语义 (Denotational Semantics)

**定义**: 将程序映射到数学对象

**基本映射**:

```
C⟦n⟧ = n                                    (数值)
C⟦x⟧ = lookup(ρ, x)                         (变量)
C⟦λx.M⟧ = λv. C⟦M⟧[ρ[x↦v]]                  (抽象)
C⟦M N⟧ = C⟦M⟧(C⟦N⟧)                        (应用)
```

### 1.3 公理语义 (Axiomatic Semantics)

**定义**: 通过前后条件描述程序行为

**Hoare三元组**:

```
{φ} C {ψ}
表示若前条件φ成立，执行C后后条件ψ成立
```

**推理规则**:

```
{φ[x↦E]} x:=E {φ}                          (赋值)
{φ} C₁ {χ}  {χ} C₂ {ψ}                     (顺序)
  ─────────────────────
      {φ} C₁;C₂ {ψ}
```

## 二、核心关联定理

### 定理2.1 (操作语义与指称语义的等价)

**陈述**: 操作语义求值等价于指称语义的计算

**形式化**:

```
M →* V    ⟺    C⟦M⟧ = C⟦V⟧
```

**证明**:

**方向1 (→):** 对归约步骤数归纳

- 基本情况: 0步，M = V，显然成立
- 归纳步骤: M → M\' →* V
  由归纳假设C⟦M\'⟧ = C⟦V⟧
  需证C⟦M⟧ = C⟦M\'⟧
  对每种规约规则验证语义不变

**方向2 (←):** 由指称语义的组合性

- C⟦M⟧ = C⟦V⟧ 意味着 M 和 V 有相同指称
- 由操作语义的合流性，M 必可归约到 V

**结论**: 两种语义等价 ∎

---

### 定理2.2 (指称语义到公理语义的转换)

**陈述**: 指称语义可导出最弱前置条件

**形式化**:

```
wp(C, ψ) = {σ | C⟦C⟧(σ) ⊨ ψ}
```

**证明**:

**wp的定义**: 使得C执行后ψ成立的最大前条件集合

**由指称语义构造**:

- C⟦C⟧: State → State 是程序的指称
- ψ ⊆ State 是后条件
- wp(C, ψ) = C⟦C⟧⁻¹(ψ)

**验证Hoare规则**:

- 赋值: wp(x:=E, ψ) = ψ[x↦E]
- 顺序: wp(C₁;C₂, ψ) = wp(C₁, wp(C₂, ψ))

**结论**: wp可由指称语义构造 ∎

---

### 定理2.3 (公理语义的完备性与可靠性)

**陈述**: Hoare逻辑对指称语义是可靠且完备的

**可靠性**:

```
⊢ {φ} C {ψ}  ⟹  ⊨ {φ} C {ψ}
语法可推导 ⟹ 语义有效
```

**完备性**:

```
⊨ {φ} C {ψ}  ⟹  ⊢ {φ} C {ψ}
语义有效 ⟹ 语法可推导
```

**证明概要**:

**可靠性**: 对每个Hoare规则，验证语义保持

- 赋值规则: 直接由语义
- 条件规则: 由逻辑的case分析
- 循环规则: 由归纳原理

**完备性**: 构造性证明

- 对每个命令C，证明 ⊢ {wp(C,ψ)} C {ψ}
- 若 ⊨ {φ} C {ψ}，则 φ → wp(C,ψ)
- 因此 ⊢ {φ} C {ψ}

**结论**: Hoare逻辑可靠且完备 ∎

## 三、统一视角：Galois连接

### 3.1 语义的格结构

**状态集合的幂集格**: (P(State), ⊆)

**程序指称**: C⟦C⟧: State → State

**谓词**: φ, ψ ∈ P(State)

### 3.2 Galois连接框架

```
指称语义: D(C) = C⟦C⟧
公理语义: A(C)(φ) = wp(C, φ)

Galois连接:
D(C) ⊣ A(C)
```

**定理3.1 (Galois连接刻画)**:

```
C⟦C⟧(σ) ∈ ψ  ⟺  σ ∈ wp(C, ψ)
```

这正好是Galois连接的定义。

## 四、逻辑关系桥梁

### 4.1 逻辑关系定义

连接语法与语义的桥梁：

```
V⟦τ⟧ ⊆ Value × Value    (值关系)
E⟦τ⟧ ⊆ Expr × Expr      (表达式关系)
```

### 4.2 基本引理

**引理4.1 (基本引理)**:

```
若 Γ ⊢ M : τ 且 (γ, γ\') ∈ G⟦Γ⟧，
则 (M[γ], M[γ\']) ∈ E⟦τ⟧
```

**证明**: 对类型推导归纳

## 七、未解决问题与未来方向

### 7.1 当前形式化缺口

1. **指称语义的构造性**: 需要domain theory处理递归
2. **高阶程序的公理语义**: 一阶Hoare逻辑不足
3. **并发程序的统一语义**: 需要进程代数扩展

### 7.2 研究方向

1. **分离逻辑**: 扩展公理语义处理内存
2. **Game Semantics**: 交互式指称语义
3. **微多态**: 细化类型连接精度与可判定性
