# 存储电路

本目录介绍计算机存储系统的硬件基础，从基本的触发器到复杂的缓存阵列设计。

## 内容概览

| 文件 | 描述 |
|------|------|
| `01_SR_Latch_and_D_FlipFlop.md` | SR锁存器与D触发器原理 |
| `02_Register_File.md` | 寄存器堆的设计与实现 |
| `03_SRAM_Cell.md` | 静态随机存取存储器单元 |
| `04_Cache_Array.md` | 缓存阵列的硬件结构 |

## 核心概念

- **时序逻辑**: 锁存器与触发器的工作原理
- **寄存器堆**: 多端口寄存器文件的设计
- **SRAM**: 6T单元结构与读写操作
- **缓存硬件**: 缓存阵列的物理组织方式

## 前置知识

需要掌握 `01_Digital_Logic_Foundations` 中的数字逻辑基础。
