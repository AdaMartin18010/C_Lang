# 数字逻辑基础

本目录介绍数字电路的物理基础，从晶体管级别到逻辑门实现的完整知识链。

## 内容概览

| 文件 | 描述 |
|------|------|
| `01_Transistor_Level.md` | 晶体管级电路设计与CMOS技术 |
| `02_Logic_Gates_Implementation.md` | 逻辑门的物理实现方式 |
| `03_Combinational_Circuits.md` | 组合逻辑电路分析与设计 |

## 核心概念

- **CMOS技术**: 互补金属氧化物半导体的工作原理
- **逻辑门**: 与门、或门、非门等的晶体管级实现
- **组合逻辑**: 无记忆元件的数字电路设计

## 学习路径

这是物理机器层的基础部分，建议按文件序号顺序学习。
