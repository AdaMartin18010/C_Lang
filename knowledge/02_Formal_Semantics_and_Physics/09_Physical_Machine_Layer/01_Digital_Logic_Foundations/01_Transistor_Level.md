## 2. MOS晶体管工作原理

### 2.1 半导体物理基础

#### 2.1.1 硅晶体结构与载流子

硅(Si)是四价元素，每个硅原子与四个相邻原子形成共价键。在绝对零度时，硅是绝缘体；在室温下，部分共价键断裂产生自由电子和空穴。

```
硅晶体结构示意图:

        Si      Si      Si      Si      Si
         \\      |      /        \\      |
          ●─────●─────●          ●─────●
         /      |      \\        /      |
        Si      Si      Si      Si      Si
         |     / \\     |        |     /
        ●─────●   ●─────●      ●─────●
         |     \\ /     |        |     \\
        Si      Si      Si      Si      Si
         \\      |      /        \\      |
          ●─────●─────●          ●─────●
         /      |      \\        /      |
        Si      Si      Si      Si      Si

共价键电子: ●
自由电子:   ○ (带负电 -1.6×10⁻¹⁹ C)
空穴:       ⊕ (带正电 +1.6×10⁻¹⁹ C)
```

**本征载流子浓度公式:**

```
                    nᵢ = √(N_c × N_v) × exp(-E_g / 2kT)

其中:
  nᵢ = 本征载流子浓度 (硅在300K时约1.5×10¹⁰ cm⁻³)
  N_c = 导带有效态密度
  N_v = 价带有效态密度
  E_g = 禁带宽度 (硅为1.12eV)
  k = 玻尔兹曼常数 (8.617×10⁻⁵ eV/K)
  T = 绝对温度 (K)
```

#### 2.1.2 N型与P型半导体

**N型半导体 (电子为多数载流子):**

```
N型硅掺杂示意图 (掺入五价磷P):

        Si      Si      Si      Si
         \\      |      /        \\
          ●─────●─────●─────[P]─────●
         /      |      \\      |    /
        Si      Si      Si     ⊖   Si
         |     / \\     |      +    |
        ●─────●   ●─────●   多余电子
         |     \\ /     |      ↓    |
        Si      Si      Si     ○   Si
         \\      |      /        \\
          ●─────●─────●─────────────●

磷原子提供5个价电子，其中4个参与共价键，
第5个电子成为自由电子 (⊖)，带负电。

施主浓度: N_D ≈ 自由电子浓度 n_n
空穴浓度: p_n = nᵢ² / n_n    (少数载流子)
```

**P型半导体 (空穴为多数载流子):**

```
P型硅掺杂示意图 (掺入三价硼B):

        Si      Si      Si      Si
         \\      |      /        \\
          ●─────●─────●─────[B]─────●
         /      |      \\      |    /
        Si      Si      Si     ⊕   Si
         |     / \\     |      +    |
        ●─────●   ⊙─────●   空穴位置
         |     \\ /     |      ↑    |
        Si      Si      Si     ○   Si
         \\      |      /        \\
          ●─────●─────●─────────────●

硼原子提供3个价电子，形成共价键时缺少1个电子，
产生空穴 (⊕)，可接受电子，带正电。

受主浓度: N_A ≈ 空穴浓度 p_p
电子浓度: n_p = nᵢ² / p_p    (少数载流子)
```

### 2.3 NMOS晶体管工作原理

#### 2.3.1 截止状态 (V_GS < V_th)

```
NMOS截止状态能带图与电荷分布:

                    能带图                          电荷分布

    能量 ↑           E_c ───┬────────┬───         +++++++++++++
         │            (导带) │        │          栅极正电荷
         │                 │   势垒  │              │
         │    E_i ─────────┼────────┼────      ───┴─── 耗尽层
         │   (本征)        │   ↓↓↓  │         - - - - -
         │                 │        │         受主离子
    E_F ─┼─────────────────┴────────┴────     P型衬底
         │            E_v (价带)              (中性区)
         │
         └────────────────────────────────▶
                    位置 (S → 衬底 → D)

描述:
- 栅压低于阈值电压，表面未反型
- 源漏之间没有导电沟道
- 只有极小的漏电流 (亚阈值泄漏)

等效电路:
        D
        │
    ━━━┫┣━━━  (开路，近似无穷大电阻)
        │
        S
        │
        B
```

#### 2.3.2 导通状态 (V_GS > V_th)

```
NMOS导通状态能带图与沟道形成:

                    能带图                          截面图

    能量 ↑           E_c ─┬──┬──┬──┬──┬──┬──┬──
         │            (导带)│  │  │  │  │  │  │
         │    E_i ──────────┘  │  │  │  │  └──┘← 反型层
         │   (本征)            │  │  │  │    (电子积累)
         │    E_F ─────────────┴──┴──┴──┴──────
         │            E_v (价带)
         │
         └────────────────────────────────▶
                         位置

表面反型条件:
              表面势 ψ_s = 2φ_F

其中 φ_F = (kT/q) × ln(N_A/n_i) 是费米势

截面图:
                    栅极 (+V_G)
                       │
              ═════════╧═════════
              +++++++++++++++++++  (栅极正电荷)
              ───────────────────
              ║  SiO₂  ║
              ═══════════════════
              ║░░░░░░░░░░░░░░░░░║  ← N型反型层 (电子)
        源N⁺ ═╬░░░░沟道区░░░░░░░╬═ 漏N⁺
              ║░░░░(导电)░░░░░░░║
              ╠══════════════════╣
              ║    P型衬底      ║
              ║  (负偏置 B)     ║
              ╚══════════════════╝

电子从源极通过沟道流向漏极，形成漏极电流 I_D
```

#### 2.3.3 I-V特性详解

```
NMOS晶体管I-V特性曲线:

    I_D ↑
        │
   V_GS=│     V_GS=    V_GS=    V_GS=
   1.8V │     1.5V     1.2V     0.9V
        │      │        │        │
        │    ╭─╮      ╭─╮      ╭─╮
        │   ╱   ╲    ╱   ╲    ╱   ╲
        │  ╱     ╲  ╱     ╲  ╱     ╲
        │ ╱       ╲╱       ╲╱       ╲
        │╱                               (饱和区)
        ├─────────────────────────────────────▶
        │                                  V_DS
        │    │      │      │
        └────┴──────┴──────┴────────────────▶
            0.2    0.4    0.6   (V_GS-V_th)

                    线性区 → 饱和区

工作区域划分:
┌─────────────────────────────────────────────────────────────────┐
│ 区域          条件                          电流公式            │
├─────────────────────────────────────────────────────────────────┤
│ 截止区        V_GS < V_th                   I_D ≈ 0             │
│                                                                  │
│ 线性区        V_GS > V_th,                  I_D = μ_n C_ox (W/L) │
│               V_DS < V_GS - V_th                × [(V_GS-V_th)V_DS - V_DS²/2]│
│                                                                  │
│ 饱和区        V_GS > V_th,                  I_D = (μ_n C_ox/2)(W/L)│
│               V_DS > V_GS - V_th                × (V_GS - V_th)²  │
│                                              × (1 + λV_DS)       │
└─────────────────────────────────────────────────────────────────┘

参数说明:
  μ_n = 电子迁移率 (~500 cm²/V·s for Si)
  C_ox = 栅氧单位面积电容 (ε_ox/t_ox)
  ε_ox = SiO₂介电常数 (3.9ε₀)
  W/L = 宽长比 (晶体管尺寸设计参数)
  λ = 沟道长度调制系数
```

```
        注意: PMOS电压为负，电流方向从源到漏

电压极性约定:
  V_SG = V_S - V_G  (源栅电压，正值)
  V_SD = V_S - V_D  (源漏电压，正值)
  V_th,p = -|V_th|  (PMOS阈值电压，负值)

导通条件:
  PMOS导通: V_SG > |V_th,p| 即 V_G < V_S - |V_th,p|
  通常 V_S = VDD，所以 V_G < VDD - |V_th,p|
```

## 3. CMOS工艺详解

### 3.1 CMOS的革命性意义

```
CMOS (Complementary MOS) - 互补金属氧化物半导体

历史背景:
┌─────────────────────────────────────────────────────────────────┐
│ 年代      工艺技术           特点                                │
├─────────────────────────────────────────────────────────────────┤
│ 1960s     PMOS-only          速度慢，功耗大，工艺简单            │
│ 1970s     NMOS-only          速度快，但静态功耗大                │
│ 1980s+    CMOS               低功耗，高密度，成为主流            │
│ 2000s+    SOI CMOS           更好的性能，更低的泄漏              │
│ 2010s+    FinFET/3D          更好的栅控，抑制短沟道效应          │
└─────────────────────────────────────────────────────────────────┘

CMOS核心优势:
1. 静态功耗极低 - 只有泄漏电流，无直流通路
2. 噪声容限大 - 接近理想逻辑电平
3. 抗干扰能力强 - 电源电压波动容忍度高
4. 集成度高 - 适合大规模集成电路
```

### 3.2 CMOS结构：阱工艺

```
- PMOS做在N阱中，NMOS做在P衬底上

- P衬底接最低电位 (GND)

- 在N型衬底上制作P阱
- NMOS在P阱中，PMOS在N衬底上
- P阱接GND，N衬底接VDD
- 现代工艺多采用双阱或三阱结构
```

### 4.3 开关延迟分析

```
晶体管开关延迟模型:

简化RC延迟模型:

        VDD
         │
    ━━━━━┫┣━━━━┳━━━━━━━━━━━┓
      PMOS     │           │
         │     │    C_L    │
         ├─────┤   ═══     │
    ─────┫┣────┘           │
      NMOS    GND         GND
         │
        GND

        反相器带负载电容C_L

延迟计算公式:

上升时间 (t_r): 输出从10% VDD上升到90% VDD
  t_r ≈ 2.2 × R_p × C_L

下降时间 (t_f): 输出从90% VDD下降到10% VDD
  t_f ≈ 2.2 × R_n × C_L

传播延迟 (t_p): 输入到输出的平均延迟
  t_p = (t_pHL + t_pLH) / 2

其中:
  t_pHL = 0.69 × R_n × C_L   (输出由高变低)
  t_pLH = 0.69 × R_p × C_L   (输出由低变高)

示例计算:
┌────────────────────────────────────────────────────────────────┐
│ 条件: 65nm工艺, W_n/L = 2, W_p/L = 4, C_L = 10 fF             │
│                                                                │
│ NMOS导通电阻:                                                  │
│   R_n = 1 / (μ_n C_ox × 2 × 1) ≈ 400 Ω                       │
│                                                                │
│ PMOS导通电阻:                                                  │
│   R_p = 1 / (μ_p C_ox × 4 × 1) ≈ 500 Ω                       │
│                                                                │
│ 传播延迟:                                                      │
│   t_pHL = 0.69 × 400 × 10×10⁻¹⁵ = 2.76 ps                    │
│   t_pLH = 0.69 × 500 × 10×10⁻¹⁵ = 3.45 ps                    │
│   t_p = (2.76 + 3.45) / 2 = 3.1 ps                           │
└────────────────────────────────────────────────────────────────┘
```

## 5. 功耗分析

### 5.1 功耗组成

```
CMOS电路功耗组成:

总功耗 = 动态功耗 + 静态功耗

P_total = P_dynamic + P_static
        = P_switching + P_shortcircuit + P_leakage

┌────────────────────────────────────────────────────────────────┐
│                                                                │
│   P_total                                                      │
│     │                                                          │
│     ├──► P_switching  (开关功耗/充放电功耗)                     │
│     │     占主导地位 (70-90%)                                   │
│     │                                                          │
│     ├──► P_shortcircuit (短路功耗)                              │
│     │     输入跳变期间的瞬态功耗 (5-20%)                        │
│     │                                                          │
│     └──► P_leakage (泄漏功耗)                                   │
│           静态功耗，技术节点越小越严重 (10-30%)                  │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### 5.2 动态功耗详解

```
开关功耗 (Switching Power):

物理机制:
- 输出节点从0→1时，从电源抽取能量给负载电容充电
- 输出节点从1→0时，负载电容储存的能量通过NMOS耗散到地
- 每个完整开关周期消耗的能量: E = C_L × V_DD²

公式推导:

能量存储在电容中:
  E_C = ½ × C_L × V_DD²

电源提供的总能量:
  E_supply = C_L × V_DD²

NMOS管耗散的能量:
  E_dissipated = E_supply - E_C = ½ × C_L × V_DD²

一个完整周期 (0→1→0) 消耗的总能量:
  E_cycle = C_L × V_DD²

开关功耗 (频率为f时):
  P_switching = α × C_L × V_DD² × f

其中:
  α = 活动因子 (0 ≤ α ≤ 1)，表示开关活动概率
  C_L = 负载电容
  V_DD = 电源电压
  f = 时钟频率

开关功耗波形示意图:

    V_DD ─┐
          │    ╭────╮      ╭────╮      ╭────╮
          │   ╱      ╲    ╱      ╲    ╱      ╲
          │  ╱        ╲──╱        ╲──╱        ╲──
        ──┴─╱────────────────────────────────────────▶ t

    电流 ─┐
          │    ╭╮      ╭╮      ╭╮      ╭╮
          │   ╱  ╲    ╱  ╲    ╱  ╲    ╱  ╲
          │──╱    ╲──╱    ╲──╱    ╲──╱    ╲──────▶ t
          │
          └─ 充电电流(从电源)   放电电流(到地)

功耗降低策略:
┌────────────────────────────────────────────────────────────────┐
│ 策略              效果                    实现方式             │
├────────────────────────────────────────────────────────────────┤
│ 降低电压          P ∝ V²                电压调节、DVFS         │
│ 降低频率          P ∝ f                 动态频率调节           │
│ 降低电容          P ∝ C                 缩小尺寸、减少连线     │
│ 降低活动因子      P ∝ α                 时钟门控、电源门控     │
└────────────────────────────────────────────────────────────────┘
```

### 5.3 短路功耗

```
短路功耗 (Short-Circuit Power):

产生机制:
- 输入信号跳变期间，PMOS和NMOS同时导通
- 形成从VDD到GND的直接通路
- 产生瞬态电流尖峰

短路电流示意图:

    V_in ─┐
          │       ╭────╮
          │      ╱      ╲
          │─────╱        ╲──────────
          │
          │              ╭────╮
          │   ╭─────────╱      ╲
          │──╱                      ╲────
          │
          └──────────────────────────────▶ t

    短路  ─┐
    电流   │      ╭──╮
          │     ╱    ╲
          │────╱      ╲────
          │
          └──────────────────────────────▶ t

          ════════════════════════
          ←─ 两管同时导通时间 ──→

短路功耗公式:
  P_shortcircuit = α × t_sc × V_DD × I_peak × f

其中:
  t_sc = 短路持续时间 (输入跳变时间)
  I_peak = 峰值短路电流

减小短路功耗的方法:
1. 减小输入跳变时间 (快速边沿)
2. 降低电源电压
3. 优化晶体管尺寸比
```

### 5.4 静态泄漏功耗

```
泄漏功耗 (Leakage Power):

泄漏电流来源:

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   I_leakage                                                     │
│     │                                                           │
│     ├──► I_subthreshold (亚阈值泄漏) ← 最严重，指数依赖V_th      │
│     │                                                           │
│     ├──► I_gate (栅极隧穿泄漏) ← 栅氧很薄时显著                  │
│     │                                                           │
│     ├──► I_junction (结反向偏置泄漏)                             │
│     │                                                           │
│     └──► I_GIDL (栅致漏极泄漏)                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

亚阈值泄漏详解:

亚阈值电流公式:
              I_D,sub = I_0 × exp[(V_GS - V_th)/(n×V_T)] × [1 - exp(-V_DS/V_T)]

其中:
  I_0 = 工艺相关常数
  n = 亚阈值斜率因子 (1.3-1.5)
  V_T = kT/q ≈ 26 mV (室温)

亚阈值斜率:
  S = n × V_T × ln(10) ≈ 60-100 mV/decade

表示V_GS每变化S，I_D变化10倍

亚阈值泄漏与阈值电压的关系:

    I_sub ↑
          │  ╱
          │ ╱
          │╱
          ├─────────────▶
          │          V_th
          │
          └─────────────────────────▶
              0.1V   0.3V   0.5V

泄漏随阈值电压降低而指数增长!

技术节点演进中的泄漏问题:

┌────────────────────────────────────────────────────────────────┐
│ 技术节点    亚阈值泄漏占比    栅泄漏占比      总泄漏/总功耗      │
├────────────────────────────────────────────────────────────────┤
│ 180 nm      5%               <1%            ~5%                │
│ 90 nm       15%              1%             ~15%               │
│ 65 nm       25%              5%             ~25%               │
│ 45 nm       35%              15%            ~40%               │
│ 32 nm       40%              25%            ~50%               │
│ 16 nm       45%              30%            ~55%               │
│ 7 nm        50%              35%            ~60%               │
└────────────────────────────────────────────────────────────────┘

降低泄漏功耗的技术:
1. 多阈值电压 (MTCMOS) - 关键路径用低V_th，其他用高V_th
2. 电源门控 (Power Gating) - 闲置模块断电
3. 体偏置 (Body Bias) - 调节V_th
```

### 7.2 逻辑门的构建原理

```
从反相器到通用逻辑门的扩展:

原理: 并联PMOS实现 NAND/NOR 功能
      串联NMOS实现 NAND/NOR 功能

PMOS网络特点:
- PMOS导通时输出高电平
- 并联PMOS: 任一输入为0，输出为1 (OR行为)
- 串联PMOS: 所有输入为0，输出才为1 (AND行为)

NMOS网络特点:
- NMOS导通时输出低电平
- 并联NMOS: 任一输入为1，输出为0 (NOR行为)
- 串联NMOS: 所有输入为1，输出才为0 (NAND行为)

CMOS逻辑门设计规则:
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  PMOS网络 (上拉网络)    NMOS网络 (下拉网络)                     │
│  ───────────────────    ───────────────────                     │
│                                                                 │
│  输出 = 1 的路径          输出 = 0 的路径                       │
│                                                                 │
│  ● 并联 = OR (任一导通)   ● 并联 = NOR (任一导通拉低)           │
│  ● 串联 = AND (全部导通)  ● 串联 = NAND (全部导通拉低)          │
│                                                                 │
│  对偶关系: PMOS网络与NMOS网络互为对偶                            │
│  PMOS并联 ↔ NMOS串联                                            │
│  PMOS串联 ↔ NMOS并联                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

示例: 2输入NAND门

        VDD
         │
    ━━━━━┳━━━━━
         │
    ━━━━━┻━━━━━ Y = !(A·B)
         │
    ━━━━━┫┣━━━━━
         │
    ━━━━━┫┣━━━━━
         │
        GND

PMOS: 并联 (A=0或B=0时，至少一个导通，Y=1)
NMOS: 串联 (A=1且B=1时，全部导通，Y=0)
```

### 7.3 与C语言/汇编的关联

```
从晶体管到C程序的完整映射链:

C语言层:
────────
  int a, b, c;
  c = a & b;    // 按位与操作

汇编层 (RISC-V):
────────────────
  lw   t0, a      # 加载a到t0
  lw   t1, b      # 加载b到t1
  and  t2, t0, t1 # t2 = t0 & t1
  sw   t2, c      # 存储结果到c

微架构层 (ALU实现):
───────────────────
  ┌─────────────────────────────────────────┐
  │            32位ALU                       │
  │  ┌─────────┐    ┌─────────┐    ┌─────┐ │
  │  │ 操作数A │───▶│  AND阵列 │───▶│ 结果│ │
  │  │ (32bit) │    │(32位宽) │    │     │ │
  │  └─────────┘    └────▲────┘    └─────┘ │
  │  ┌─────────┐         │                │
  │  │ 操作数B │─────────┘                │
  │  │ (32bit) │                          │
  │  └─────────┘                          │
  └─────────────────────────────────────────┘

门级实现:
─────────
  每个AND操作 = 1个AND门
  32位AND = 32个AND门并行

       A[0] ──┬──▶┌───┐
              └──▶│AND│──▶ Result[0]
       B[0] ─────▶└───┘

       A[1] ──┬──▶┌───┐
              └──▶│AND│──▶ Result[1]
       B[1] ─────▶└───┘

       ... (共32组)

晶体管级实现:
─────────────
  每个AND门 = NAND门 + 反相器
  NAND门 = 2个并联PMOS + 2个串联NMOS
  反相器 = 1个PMOS + 1个NMOS

  所以每个AND需要:
  - 3个PMOS
  - 3个NMOS

  32位AND总共需要:
  - 192个晶体管 (32 × 6)

物理层:
───────
  192个晶体管 =
  - 192 × (W × L) 的硅面积
  - 约 1-10 微米的芯片面积 (先进工艺)
  - 皮秒级的开关延迟
  - 纳瓦级的功耗

完整映射关系表:
┌─────────────────────────────────────────────────────────────────┐
│ C语言              汇编               门电路    晶体管数         │
├─────────────────────────────────────────────────────────────────┤
│ a & b              AND              AND门     6/位              │
│ a | b              OR               OR门      6/位              │
│ ~a                 NOT              反相器    2/位              │
│ a ^ b              XOR              XOR门     12/位             │
│ a + b              ADD              加法器    ~30/位            │
│ a < b              SLT              比较器    ~20/位            │
│ if (a)             BEQ/BNE          多路选择器 ~8/位            │
└─────────────────────────────────────────────────────────────────┘

理解这一映射的重要性:
1. 理解C语言操作的实际硬件成本
2. 理解为什么某些操作更快 (如移位 vs 乘法)
3. 理解编译器优化的硬件基础
4. 理解嵌入式系统的功耗和面积限制
```

```verilog
//-------------------------

    input  wire gate,      // 栅极 (低电平有效)

    output wire drain      // 漏极
);

        .gate   (n_ctrl),
        .source (in),
        .drain  (nmos_out)

    pmos_transistor pmos_inst (
        .gate   (p_ctrl),
        .source (in),
        .drain  (pmos_out)

    // assign out = (n_ctrl & ~p_ctrl) ? in : 1'bz;
```

### 8.3 功耗估算模型

```verilog
//=============================================================================
// CMOS电路功耗估算模块
// 文件名: power_estimation.v
//=============================================================================

module power_estimator (
    input  wire clk,
    input  wire reset,
    input  wire signal,
    output real power_dynamic,    // 动态功耗 (W)
    output real power_static,     // 静态功耗 (W)
    output real energy_total      // 总能耗 (J)
);
    // 工艺参数 (65nm示例)
    parameter real VDD = 1.0;           // 电源电压 (V)
    parameter real FREQ = 1e9;          // 时钟频率 (Hz)
    parameter real C_LOAD = 10e-15;     // 负载电容 (F)
    parameter real ALPHA = 0.5;         // 活动因子
    parameter real I_LEAK = 1e-9;       // 泄漏电流 (A)

    // 内部寄存器
    reg signal_prev;
    integer toggle_count;
    real time_elapsed;

    // 初始化
    initial begin
        signal_prev = 0;
        toggle_count = 0;
        time_elapsed = 0;
    end

    // 边沿检测与计数
    always @(posedge clk or posedge reset) begin
        if (reset) begin
            signal_prev <= 0;
            toggle_count <= 0;
        end else begin
            signal_prev <= signal;
            if (signal != signal_prev) begin
                toggle_count <= toggle_count + 1;
            end
        end
    end

    // 时间累计
    always @(posedge clk) begin
        if (!reset) begin
            time_elapsed <= time_elapsed + 1.0/FREQ;
        end
    end

    // 功耗计算 (组合逻辑)
    assign power_dynamic = ALPHA * C_LOAD * VDD * VDD * FREQ;
    assign power_static = VDD * I_LEAK;
    assign energy_total = (power_dynamic + power_static) * time_elapsed;

endmodule

//-------------------------
// 功耗分析测试平台
//-------------------------
module power_analysis_tb;
    reg clk, reset;
    reg signal;
    wire real p_dyn, p_static, energy;

    // 实例化功耗估算器
    power_estimator #(
        .VDD(1.0),
        .FREQ(1e9),
        .C_LOAD(10e-15),
        .ALPHA(0.5),
        .I_LEAK(1e-9)
    ) uut (
        .clk(clk),
        .reset(reset),
        .signal(signal),
        .power_dynamic(p_dyn),
        .power_static(p_static),
        .energy_total(energy)
    );

    // 时钟生成 (1GHz)
    initial begin
        clk = 0;
        forever #0.5 clk = ~clk;
    end

    // 测试激励
    initial begin
        $display("========================================");
        $display("    CMOS功耗分析仿真");
        $display("========================================");
        $display("工艺参数: 65nm, VDD=1.0V, f=1GHz");
        $display("----------------------------------------");

        // 复位
        reset = 1;
        signal = 0;
        #10;
        reset = 0;

        // 运行100个时钟周期
        repeat (100) begin
            @(posedge clk);
            signal = $random;
        end

        // 显示结果
        #1;
        $display("仿真完成!");
        $display("----------------------------------------");
        $display("动态功耗:   %0.6f nW", p_dyn * 1e9);
        $display("静态功耗:   %0.6f nW", p_static * 1e9);
        $display("总功耗:     %0.6f nW", (p_dyn + p_static) * 1e9);
        $display("总能量:     %0.9f pJ", energy * 1e12);
        $display("----------------------------------------");

        // 验证计算
        $display("验证:");
        $display("  P_switching = αCV²f");
        $display("              = 0.5 × 10fF × (1V)² × 1GHz");
        $display("              = 5 nW");
        $display("  P_leakage = VDD × I_leak = 1V × 1nA = 1 nW");

        $finish;
    end

    // 实时监控
    always @(posedge clk) begin
        if (!reset && $time % 10 == 0) begin
            $display("时间=%0t: P_dyn=%0.3f nW, P_static=%0.3f nW",
                     $time, p_dyn*1e9, p_static*1e9);
        end
    end
endmodule
```
