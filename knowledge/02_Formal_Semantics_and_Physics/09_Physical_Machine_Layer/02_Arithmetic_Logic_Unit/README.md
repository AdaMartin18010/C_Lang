# 算术逻辑单元

本目录深入讲解CPU核心组件ALU的设计与实现，涵盖从基础算术运算到浮点运算的完整知识体系。

## 内容概览

| 文件 | 描述 |
|------|------|
| `01_Binary_Arithmetic.md` | 二进制算术运算基础 |
| `02_Multiplier_Divider.md` | 乘法器和除法器的设计 |
| `03_ALU_Design.md` | 算术逻辑单元的完整设计 |
| `04_Floating_Point_Unit.md` | 浮点运算单元实现 |

## 核心主题

- **整数运算**: 加法器、减法器的电路实现
- **乘除运算**: Booth算法、恢复/非恢复除法
- **ALU架构**: 多功能运算单元的设计
- **浮点运算**: IEEE 754标准与FPU实现

## 关联内容

与 `01_Digital_Logic_Foundations` 和 `03_Memory_Circuits` 共同构成CPU数据通路的基础。
