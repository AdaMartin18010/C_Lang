## 📖 技术概述

本目录涵盖计算机系统最底层的物理实现基础——**数字逻辑门**。
数字逻辑门是连接布尔代数抽象理论与CMOS晶体管物理实现的桥梁，是理解现代计算机如何从电子信号中产生计算能力的核心知识。

### 核心知识点

| 主题 | 描述 | 重要性 |
|:-----|:-----|:-------|
| 布尔代数 | 公理化定义、完备性定理、德摩根定律 | ⭐⭐⭐⭐⭐ |
| 逻辑门实现 | AND/OR/NOT/XOR/NAND/NOR的真值表与CMOS电路 | ⭐⭐⭐⭐⭐ |
| 组合逻辑 | 半加器、全加器、行波进位与超前进位加法器 | ⭐⭐⭐⭐ |
| 物理约束 | 门延迟、功耗模型、时序分析 | ⭐⭐⭐⭐ |
| 可计算性 | 从有限状态机到图灵完备的过渡 | ⭐⭐⭐⭐ |

### 物理到抽象的层次映射

```
电子物理层 (电子迁移、能带理论)
    ↓
晶体管层 (NMOS/PMOS、CMOS工艺)
    ↓

    ↓

    ↓

    ↓
处理器架构层 (冯诺依曼架构)
```

## ⚠️ 学习建议

### 常见误区

1. **忽视物理延迟**：逻辑门不是瞬时响应的，必须考虑传播延迟和建立时间
2. **混淆抽象层次**：布尔代数的理想假设与实际CMOS实现存在差异
3. **低估功耗问题**：动态功耗与频率成正比，现代设计中的关键约束

### 实践建议

- 使用Logisim仿真验证电路设计
- 编写C程序模拟门延迟和竞争条件
- 阅读开源处理器(如RISC-V)的RTL代码，观察逻辑门级实现

## 🔬 深入主题

### CMOS工艺演进

| 工艺节点 | 特征尺寸 | 典型延迟 | 主要挑战 |
|:---------|:---------|:---------|:---------|
| 45nm | 45nm | ~25ps | 漏电流 |
| 22nm | 22nm | ~15ps | 短沟道效应 |
| 7nm | 7nm | ~5ps | 量子隧穿 |
| 3nm | 3nm | ~3ps | 原子级波动 |

### 低功耗设计技术

- **时钟门控**：关闭不活动模块的时钟
- **电源门控**：关闭不活动模块的电源
- **多阈值单元**：关键路径用低阈值，其他用高阈值
- **动态电压频率调节(DVFS)**：根据负载调整电压频率

### 时序分析基础

```
数据到达时间 = 时钟沿 + 时钟到Q延迟 + 组合逻辑延迟
数据需求时间 = 下一时钟沿 - 建立时间
建立时间裕量 = 数据需求时间 - 数据到达时间
```

保持时间检查确保数据在时钟沿后保持稳定，防止触发器捕获错误值。
