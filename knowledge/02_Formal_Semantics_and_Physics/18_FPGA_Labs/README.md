# FPGA实验

本目录提供完整的FPGA实验指导，从基础门电路到完整的SoC集成，共包含6个递进式实验。

## 实验内容

| 实验 | 文件 | 描述 |
|------|------|------|
| Lab 1 | `01_Lab1_Basic_Gates.md` | 基础门电路实现 |
| Lab 2 | `02_Lab2_ALU_Implementation.md` | 算术逻辑单元设计与实现 |
| Lab 3 | `03_Lab3_Single_Cycle_CPU.md` | 单周期CPU设计与实现 |
| Lab 4 | `04_Lab4_Pipelined_CPU.md` | 流水线CPU设计与实现 |
| Lab 5 | `05_Lab5_Cache_Controller.md` | 缓存控制器设计与实现 |
| Lab 6 | `06_Lab6_SoC_Integration.md` | 片上系统集成 |

## 目录结构

```
18_FPGA_Labs/
├── code/           # 实验代码
│   ├── lab1/       # Lab1代码
│   ├── lab2/       # Lab2代码
│   ├── lab3/       # Lab3代码
│   ├── lab4/       # Lab4代码
│   ├── lab5/       # Lab5代码
│   └── lab6/       # Lab6代码
├── reports/        # 实验报告模板
├── xdc/            # 约束文件
└── *.md            # 实验指导文档
```

## 实验准备

- Xilinx Vivado开发环境
- Basys3或类似FPGA开发板
- 基础的Verilog/VHDL知识
