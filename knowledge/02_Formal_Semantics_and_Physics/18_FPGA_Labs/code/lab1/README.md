# Lab1: 基本逻辑门实现

> **实验**: FPGA基础实验 - 使用Verilog实现基本逻辑门电路，包括与门、或门、非门等组合逻辑，学习数字电路设计的基本流程和方法

## 概述

本实验是FPGA数字电路设计的入门实验，通过实现基本逻辑门电路，掌握Verilog HDL语言基础、数字电路设计流程以及FPGA开发工具的使用。学员将学习如何从真值表出发，使用硬件描述语言描述数字逻辑，并通过仿真和板上验证确保设计的正确性。

基本逻辑门是所有数字电路的基础构建模块。理解这些基础元件的行为特性、时序特性以及硬件实现方式，对于后续复杂的组合逻辑和时序逻辑设计至关重要。

## 实验目标

- **掌握Verilog基础语法**：学习模块定义、端口声明、assign语句等基本语法结构
- **理解逻辑门原理**：深入理解AND、OR、NOT、NAND、NOR、XOR等基本逻辑门的真值表和布尔表达式
- **学习设计流程**：从设计输入、功能仿真、综合实现到板上验证的完整FPGA设计流程
- **熟悉开发工具**：掌握Vivado/Quartus等FPGA开发工具的基本使用方法
- **实践硬件调试**：通过LED和开关进行硬件验证，理解物理世界与数字逻辑的对应关系
- **建立工程思维**：培养硬件描述语言的并行思维，区别于软件顺序执行的思维方式

## 硬件要求

| 项目 | 规格要求 |

| FPGA开发板 | Xilinx Artix-7 / Intel Cyclone IV 或同等性能开发板 |

## 实验步骤

1. **理论学习**：复习基本逻辑门的真值表和布尔代数基础
2. **代码编写**：根据真值表编写Verilog代码，实现多种逻辑门
3. **仿真验证**：编写Testbench进行功能仿真，验证逻辑正确性
4. **综合实现**：使用开发工具进行综合、布局布线
5. **引脚约束**：根据开发板原理图配置引脚约束文件
6. **生成比特流**：生成可下载到FPGA的比特流文件
7. **板上验证**：下载到开发板，使用开关和LED验证功能
8. **撰写报告**：记录设计过程、遇到的问题及解决方案

## 预期成果

- 成功实现基本逻辑门电路并在FPGA上运行
- 完成至少包含4种基本逻辑门的完整设计
- 通过仿真和板上验证证明设计的正确性
- 能够解释Verilog代码与硬件电路的对应关系
