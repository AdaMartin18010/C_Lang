# Lab4: 流水线RISC-V CPU实现

> **实验**: 高级CPU设计实验 - 实现一个5级流水线RISC-V处理器，解决数据冒险、控制冒险和结构冒险，理解现代高性能处理器的设计原理

## 概述

流水线技术是现代处理器提高性能的核心手段。本实验将单周期CPU扩展为5级流水线架构，显著提升指令吞吐率。学员将面临并解决流水线设计中的经典问题：数据冒险、控制冒险和结构冒险。

通过本实验，学员将深入理解现代CPU如何通过指令级并行(ILP)提升性能，掌握前递(Forwarding)、停顿(Stall)、分支预测等关键技术，这些都是现代高性能处理器不可或缺的核心机制。

## 实验目标

- **理解流水线原理**：掌握5级流水线的结构和工作原理(IF/ID/EX/MEM/WB)
- **设计流水线寄存器**：实现各级之间的流水线寄存器，正确传递数据和控制信号
- **解决数据冒险**：实现数据前递(Forwarding)机制和必要的流水线停顿
- **处理控制冒险**：实现分支预测和流水线冲刷(Flush)机制
- **避免结构冒险**：合理设计存储器访问策略，避免资源冲突
- **评估性能提升**：对比单周期与流水线CPU的性能差异

## 实验步骤

1. **流水线设计**：将单周期CPU划分为5个阶段，设计流水线寄存器
2. **基础实现**：先实现无冒险处理的简单流水线，观察问题
3. **数据前递**：实现EX-EX和MEM-EX数据前递，解决数据冒险
4. **Load-Use处理**：实现Load-Use冒险检测和流水线停顿
5. **分支处理**：实现分支预测(静态预测)和错误预测时的流水线冲刷
6. **冒险单元整合**：整合所有冒险处理逻辑到统一的Hazard Unit
7. **仿真验证**：设计包含各种冒险情况的测试程序
8. **性能测试**：运行基准测试程序，对比单周期与流水线的IPC
9. **时序优化**：分析关键路径，优化时序以满足目标频率

## 预期成果

- 实现一个功能正确的5级流水线RISC-V CPU
- 正确解决数据冒险(前递+停顿)和控制冒险(预测+冲刷)
- 流水线CPI接近1，性能显著提升
- 能够运行复杂程序(如快速排序、矩阵乘法)
- 理解流水线设计对时钟频率和指令延迟的权衡
