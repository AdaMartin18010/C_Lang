# Lab6: 完整SoC集成

> **实验**: 系统级设计实验 - 将流水线CPU、Cache、内存控制器、I/O接口集成成一个完整的SoC系统，实现一个可运行操作系统的嵌入式处理器平台

## 概述

本实验是FPGA系列实验的集大成之作，学员将整合之前实验的成果，构建一个完整的片上系统(SoC)。SoC是现代嵌入式系统的核心，集成了处理器、存储器、外设接口等多个功能模块。

学员将学习系统总线设计(如AXI4-Lite)、中断处理、DMA传输、外设接口等关键技术，最终构建一个能够运行简单操作系统或裸机程序的完整嵌入式平台。本实验将检验学员对数字系统设计的综合掌握程度。

## 实验目标

- **系统集成能力**：将CPU、Cache、内存、外设集成成完整SoC
- **总线设计**：实现SoC内部总线互连结构(如AXI4-Lite或自定义总线)
- **外设接口**：实现UART、GPIO、定时器等基本外设
- **中断系统**：设计和实现中断控制器和中断处理机制
- **DMA传输**：实现DMA控制器，支持外设与内存间的直接数据传输
- **系统验证**：在完整SoC上运行程序，验证系统功能

## 硬件要求

| 项目 | 规格要求 |
|------|----------|
| FPGA开发板 | Xilinx Nexys A7 / Intel DE10-Nano 或同等高性能开发板 |
| 逻辑资源 | LUT ≥ 10000，FF ≥ 8000，BRAM ≥ 200KB |
| 外部存储 | DDR3/DDR4 SDRAM ≥ 256MB |
| 通信接口 | UART、USB、以太网接口 |
| 输入输出 | 开关、按键、LED、七段数码管、VGA/HDMI |
| 存储扩展 | SD卡接口或Flash存储 |
| 时钟源 | 多路时钟源，支持PLL动态配置 |
| 开发环境 | Vivado 2020+ / Quartus Prime 20+ |

## 实验步骤

1. **需求分析**：确定SoC的功能规格和性能指标
2. **架构设计**：设计SoC整体架构，划分功能模块，定义接口
3. **总线设计**：实现SoC内部总线互连，包括地址译码和仲裁逻辑
4. **CPU集成**：将流水线CPU与Cache集成，封装成标准接口
5. **内存系统**：实现内存控制器，支持外部DDR和片内BRAM
6. **外设实现**：依次实现UART、GPIO、定时器等基本外设
7. **中断系统**：实现中断控制器，支持中断优先级和嵌套
8. **DMA实现**：实现DMA控制器，支持块数据传输
9. **引导程序**：编写Bootloader，支持程序加载和启动
10. **系统集成**：整合所有模块，进行时序约束和优化
11. **软件测试**：编写测试程序，验证SoC各项功能
12. **性能评估**：运行基准测试，评估SoC整体性能

## 预期成果

- 实现一个功能完整的RISC-V SoC系统
- 成功运行裸机程序和简单操作系统(如FreeRTOS)
- 通过串口与主机通信，实现人机交互
- 能够响应外部中断并执行中断服务程序
- 理解SoC设计的复杂性和系统集成的重要性
- 具备独立设计嵌入式处理器系统的能力
