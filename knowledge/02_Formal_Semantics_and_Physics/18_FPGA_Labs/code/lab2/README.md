# Lab2: 算术逻辑单元(ALU)实现

> **实验**: FPGA进阶实验 - 设计并实现一个4位算术逻辑单元(ALU)，支持基本算术运算(加、减)和逻辑运算(与、或、异或、非)，理解CPU核心组件的工作原理

## 概述

算术逻辑单元(ALU)是中央处理器(CPU)的核心组件，负责执行所有的算术运算和逻辑运算。本实验通过设计一个4位ALU，深入理解CPU内部数据处理的基本原理。

学员将从基本逻辑门出发，逐步构建半加器、全加器，最终组合成完整的ALU单元。实验将涉及组合逻辑设计、多路选择器、运算编码等核心概念，为后续CPU设计实验奠定坚实基础。

## 实验目标

- **理解ALU架构**：掌握ALU的基本结构，包括算术单元、逻辑单元和操作选择器
- **实现加法器链**：从半加器到全加器，再到行波进位加法器的设计过程
- **掌握运算编码**：使用功能选择信号控制ALU执行不同的运算操作
- **理解标志位**：实现零标志、进位标志、溢出标志等状态输出
- **学习模块化设计**：采用自底向上的设计方法，先构建基础模块再组合成系统
- **分析时序特性**：理解组合逻辑的传输延迟和关键路径

## 硬件要求

| 项目 | 规格要求 |
|------|----------|
| FPGA开发板 | Xilinx Artix-7 / Intel Cyclone IV 或更高性能开发板 |
| 输入设备 | 拨码开关 × 8 (操作数A、B各4位) |
| 输出设备 | LED × 8 (结果4位 + 标志位4位) |
| 输入设备 | 按键 × 3 (功能选择) |
| 时钟源 | 板载50MHz晶振 |
| 显示设备 | 七段数码管(可选，用于显示运算结果) |

## 实验步骤

1. **模块设计**：分解ALU功能，设计各个子模块的接口和实现
2. **基础实现**：先实现半加器和全加器，验证正确性
3. **算术单元**：使用全加器构建4位加法器，支持加法和减法
4. **逻辑单元**：实现与、或、异或、非等逻辑运算
5. **操作选择**：使用多路选择器根据操作码选择运算结果
6. **标志生成**：实现零标志、进位标志、符号标志、溢出标志
7. **仿真验证**：编写全面的Testbench验证所有运算功能
8. **综合验证**：在FPGA上实现并验证ALU功能
9. **性能分析**：分析关键路径延迟，评估电路性能

## 预期成果

- 成功实现一个功能完整的4位ALU
- 支持至少8种不同的运算操作
- 正确生成并验证所有状态标志位
- 能够进行补码加减法运算并处理溢出情况
- 理解ALU在CPU中的位置和作用
