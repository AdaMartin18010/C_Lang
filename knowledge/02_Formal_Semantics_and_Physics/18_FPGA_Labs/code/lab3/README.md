# Lab3: 单周期RISC-V CPU实现

> **实验**: CPU设计核心实验 - 实现一个完整的单周期RISC-V RV32I处理器，包含取指、译码、执行、访存、写回五个阶段，理解计算机体系结构的核心原理

## 概述

本实验是计算机体系结构学习的里程碑，学员将亲手构建一个完整的单周期RISC-V处理器。RISC-V是当今最受欢迎的开放指令集架构，理解其设计理念和实现方式对于深入学习现代处理器架构至关重要。

单周期CPU虽然执行效率不高，但结构清晰、易于理解，是学习CPU设计的最佳起点。通过本实验，学员将深入理解指令执行的全过程，包括取指(IF)、译码(ID)、执行(EX)、访存(MEM)和写回(WB)五个阶段的工作机制。

## 实验目标

- **掌握RISC-V架构**：理解RV32I指令集的定义和编码格式
- **设计数据通路**：构建从PC到寄存器堆、ALU、内存的完整数据通路
- **实现控制单元**：根据指令类型生成各类控制信号
- **支持完整指令**：实现RISC-V RV32I整数指令集中的核心指令
- **理解指令周期**：深入理解单周期处理器中一条指令的完整执行过程
- **验证处理器功能**：编写汇编程序并在设计的CPU上正确运行

## 实验步骤

1. **架构学习**：深入学习RISC-V RV32I指令集手册，理解各类指令格式
2. **数据通路设计**：绘制单周期CPU数据通路图，确定各模块连接关系
3. **模块实现**：按照数据通路图依次实现各个功能模块
4. **控制逻辑**：实现控制单元，生成ALU控制、分支控制、写使能等信号
5. **系统集成**：将各模块连接成完整的CPU顶层模块
6. **指令测试**：编写简单的汇编测试程序(如加法、跳转、访存)
7. **仿真验证**：使用Testbench验证各类指令执行的正确性
8. **程序运行**：在FPGA上加载测试程序，观察执行结果
9. **性能评估**：测量CPI(每条指令周期数)，分析单周期设计的特点

## 预期成果

- 实现一个支持RV32I核心指令集的单周期CPU
- 能够正确执行R-Type、I-Type、S-Type、B-Type、U-Type、J-Type指令
- 成功运行简单的汇编程序(如斐波那契数列、冒泡排序)
- 理解单周期CPU的时序特点(CPI=1但时钟周期较长)
- 为后续流水线CPU设计打下理论基础
