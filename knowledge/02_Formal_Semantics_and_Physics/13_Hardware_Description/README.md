# 硬件描述语言

本目录包含硬件描述语言（HDL）相关的学习资料，涵盖从Verilog基础到高级综合（HLS）的完整知识体系。

## 内容概览

| 文件 | 描述 |
|------|------|
| `01_Verilog_Basics.md` | Verilog硬件描述语言基础入门 |
| `01_Vivado_HLS_Introduction.md` | Vivado HLS工具介绍与使用指南 |
| `02_C_to_Hardware_Pragma.md` | C到硬件转换的编译器指令（Pragma）详解 |
| `03_HLS_Optimization.md` | 高级综合优化技术与策略 |
| `04_HLS_with_C` | 使用C语言进行高级综合的子目录 |

## 子目录

- **04_HLS_with_C/** - C语言HLS实践案例

## 学习路径

1. 从 `01_Verilog_Basics.md` 开始了解硬件描述基础
2. 阅读 `01_Vivado_HLS_Introduction.md` 熟悉HLS工具
3. 学习 `02_C_to_Hardware_Pragma.md` 掌握转换指令
4. 深入研究 `03_HLS_Optimization.md` 的优化技术
