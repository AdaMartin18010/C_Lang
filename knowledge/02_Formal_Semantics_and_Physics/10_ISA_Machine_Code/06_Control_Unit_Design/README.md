## 控制单元功能

控制单元是CPU的指挥中心，负责：

- 指令译码
- 控制信号生成
- 执行时序管理

## 设计方法

| 方法 | 特点 | 应用 |
|:-----|:-----|:-----|
| 硬布线 | 速度快、设计复杂 | RISC处理器 |
| 微程序 | 灵活、稍慢 | CISC处理器 |
