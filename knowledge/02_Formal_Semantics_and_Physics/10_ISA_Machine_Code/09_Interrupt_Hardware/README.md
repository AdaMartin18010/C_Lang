# 中断硬件

本目录介绍处理器中断系统的硬件实现，包括中断控制器设计和系统调用硬件支持。

## 内容概览

| 文件 | 描述 |
|------|------|
| `01_Interrupt_Controller.md` | 中断控制器的架构与实现 |
| `02_System_Call_Hardware.md` | 系统调用的硬件支持机制 |

## 核心概念

- **中断类型**: 外部中断、内部异常、软件中断
- **中断向量**: 中断处理程序的地址映射
- **优先级**: 多级中断优先级管理
- **上下文切换**: 硬件保存/恢复执行状态
- **系统调用**: 用户态到内核态的特权转换

## 中断处理流程

1. 中断请求到达中断控制器
2. 控制器进行优先级仲裁
3. CPU响应中断并保存上下文
4. 跳转到对应的中断处理程序
5. 执行中断服务例程
6. 恢复上下文并返回
