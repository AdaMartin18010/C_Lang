## 核心概念

- **DMA**: 无需CPU干预的外设内存访问
- **总线仲裁**: 多设备共享总线的调度机制
- **PCIe**: 高速串行点对点互联技术
- **内存映射I/O**: 设备寄存器的内存地址映射

## DMA工作流程

1. CPU配置DMA控制器（源地址、目的地址、传输长度）
2. DMA控制器向CPU请求总线控制权
3. CPU完成当前总线周期后释放控制权
4. DMA控制器执行数据传输
5. 传输完成后DMA通知CPU（中断）

## 现代I/O架构

```
CPU ←→ 芯片组 ←→ PCIe Root Complex ←→ PCIe设备
        ↓
    DMA控制器 ←→ 内存
```
