# 类型语义 (Type Semantics)

> **层级定位**: 02 Formal Semantics and Physics / 05 Type Semantics
> **难度级别**: L5 专家
> **预估学习时间**: 15-20 小时

## 概述

类型语义研究编程语言类型系统的数学基础，包括类型的形式化定义、类型规则的推导、以及类型安全性的证明。

## 核心主题

### 1. 类型系统基础

- **简单类型 λ 演算**: 函数类型、变量绑定
- **类型推导**: Hindley-Milner 算法 (W 算法)
- **多态性**: 参数多态、子类型多态、特设多态

### 2. C 语言类型系统

- **类型等价**: 结构等价 vs 名等价
- **类型转换**: 隐式提升、显式强制转换规则
- **不完整类型**: `void`, 前向声明结构体
- `_Generic` (C11): 类型泛型选择

### 3. 高级类型理论

- **依赖类型**: 类型依赖于值 (Idris, Agda, Lean)
- **线性类型**: 资源管理 (Rust 所有权系统)
- **效果系统**: 追踪计算效果

### 4. 类型安全

- **进展性 (Progress)**: 良类型的程序不会卡住
- **保持性 (Preservation/Subject Reduction)**: 规约保持类型
- **soundness = progress + preservation**
