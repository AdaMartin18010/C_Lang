## 核心模式

- **封装**: 不透明指针 + 访问函数
- **继承**: 结构体嵌套（组合）
- **多态**: 函数指针表（vtable）
- **状态机**: 函数指针表驱动
- **观察者**: 回调 + 多订阅者
- **策略**: 可替换算法接口
- **迭代器**: 统一遍历接口
- **工厂**: 类型标签创建对象
