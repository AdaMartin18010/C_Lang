## 新增内容

### 1. SEI CERT C Coding Standard (2024 Edition)

**文件**: `CERT_C_2024/README.md`

**覆盖内容**:

- SEI CERT C 2024 最新标准 (14000+ 字)
- C23 语言特性与安全特性整合
- 核心规则详解 (MEM/STR/INT/ENV/FIO/CON)
- 优先级矩阵和严重等级评估
- GitHub CodeQL 集成配置
- 与 ISO/IEC TS 17961 映射关系

**关键规则**:

- MEM30-C: 不要访问已释放的内存
- STR31-C: 确保字符串以 null 结尾
- INT30-C: 确保无符号整数运算不回绕
- FIO30-C: 排除用户输入中的格式字符串
- CON30-C: 正确使用互斥量

**覆盖内容**:

- ISO/IEC TS 17961:2013/COR 1:2016 国际标准 (11000+ 字)
- 46 条核心安全规则详解
- 静态分析器合规性要求
- 与 CWE 漏洞分类映射

**关键规则**:

- 5.1: 小心执行外部程序 (system())

- 5.22: 重复释放内存防护
- 5.36: 污染字符串处理
- 5.37: 正确使用 sizeof()
