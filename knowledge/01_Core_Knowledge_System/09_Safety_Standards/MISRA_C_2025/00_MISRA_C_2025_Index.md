## 快速参考

### 最危险的违反

| 违反 | 风险 | 检测方式 |
|:-----|:-----|:---------|
| 解引用NULL | 崩溃/安全漏洞 | 静态分析 |
| 缓冲区溢出 | 安全漏洞 | ASan + 静态分析 |
| 整数溢出 | 未定义行为 | UBSan |
| 使用已释放内存 | 崩溃/安全漏洞 | ASan |
| 未定义求值顺序 | 难以调试 | -Wsequence-point |

### 必备编译选项

```bash
gcc \
  -std=c23 -pedantic \
  -Wall -Wextra -Werror \
  -Wshadow -Wconversion \
  -Wundef -Wstrict-prototypes \
  -fsanitize=address,undefined \
  -c file.c
```

> **C23支持**: MISRA C:2025预计将提供对ISO C23的合规指导
