## 目录概述

本目录包含C语言的**基础语法元素**，是所有C语言学习者的**起点**。
无论你是编程新手还是有其他语言经验的开发者，都需要从这里开始建立扎实的C语言基础。

基础层涵盖四个核心模块：

- **语法元素**: 字符集、标识符、关键字、注释等词法基础
- **数据类型系统**: 基本类型、修饰符、类型转换、存储类别
- **运算符与表达式**: 算术、关系、逻辑、位运算、赋值等所有运算符
- **控制流**: 条件语句、循环语句、跳转语句的程序流程控制

这些知识点构成了C语言程序的"砖块"，是后续学习指针、内存管理等核心概念的必备前提。

### 详细内容速查

#### 01_Syntax_Elements.md - 语法元素

- **字符集与编码**: ASCII、UTF-8、源字符集与执行字符集
- **标识符规则**: 命名规范、作用域、链接属性
- **关键字列表**: C89/C99/C11/C17关键字演进
- **注释风格**: `/* */` 与 `//` 的区别和使用场景
- **预处理标记**: 翻译阶段中的词法分析

#### 02_Data_Type_System.md - 数据类型系统

- **基本类型**: `int`、`float`、`double`、`char`的存储细节
- **类型修饰符**: `signed`/`unsigned`、`short`/`long`、`long long`
- **整数类型**: `stdint.h`中的固定宽度类型(`int32_t`等)
- **类型转换**: 隐式转换规则、显式强制转换、整数提升
- **存储类别**: `auto`、`static`、`extern`、`register`、`_Thread_local`

#### 03_Operators_Expressions.md - 运算符与表达式

- **算术运算符**: `+`、`-`、`*`、`/`、`%`的整数/浮点行为
- **关系运算符**: `==`、`!=`、`<`、`>`、`<=`、`>=`
- **逻辑运算符**: `&&`、`||`、`!`的短路求值特性
- **位运算符**: `&`、`|`、`^`、`~`、`<<`、`>>`
- **赋值运算符**: 简单赋值与复合赋值(`+=`、`-=`等)
- **其他运算符**: 条件运算符`?:`、逗号运算符、sizeof
- **优先级与结合性**: 完整运算符优先级表

#### 04_Control_Flow.md - 控制流

- **条件语句**: `if-else`链、嵌套、`switch-case`跳转表
- **循环语句**: `for`循环的三个表达式、`while`、`do-while`
- **跳转语句**: `break`、`continue`、`goto`、`return`
- **结构化编程**: Dijkstra原则、避免滥用`goto`

## 学习路径建议

### 推荐学习顺序

```
第1周: 语法元素 → 数据类型系统
    ↓
第2周: 运算符与表达式 → 控制流
    ↓
第3周: 综合练习与项目
```

### 分阶段学习目标

| 阶段 | 目标 | 验证标准 |
|:-----|:-----|:---------|
| **第1阶段** (5h) | 掌握语法元素 | 能正确编写标识符、注释，理解关键字含义 |
| **第2阶段** (8h) | 理解类型系统 | 能选择合适的数据类型，理解类型转换规则 |
| **第3阶段** (7h) | 熟练使用运算符 | 能正确运用各类运算符，掌握优先级规则 |
| **第4阶段** (5h) | 掌握控制流 | 能编写嵌套条件、多重循环，避免死循环 |

### 实践项目建议

| 项目 | 涉及知识点 | 难度 |
|:-----|:-----------|:----:|
| 计算器程序 | 运算符、控制流 | ⭐⭐ |
| 九九乘法表 | 嵌套循环 | ⭐ |
| 猜数字游戏 | 随机数、条件判断、循环 | ⭐⭐ |
| 学生成绩统计 | 数组、循环、条件判断 | ⭐⭐ |

### 常见学习陷阱

1. **整数除法陷阱**: `5/2` 结果是 `2` 而不是 `2.5`
2. **赋值与比较混淆**: `if (a = 5)` 与 `if (a == 5)` 的区别
3. **运算符优先级错误**: `a & b == c` 实际是 `a & (b == c)`
4. **无符号整数溢出**: 无符号整数下溢变为大正数
5. **switch中的fall-through**: 忘记`break`导致意外贯穿
