## 循环优化

```c
// 原始版本
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        sum += a[i][j];
    }
}

// 优化: 循环展开
for (int i = 0; i < n; i += 4) {
    sum += a[i][0] + a[i][1] + a[i][2] + a[i][3];
}
```

---

## 缓存优化

```c
// 缓存友好 (按行访问)
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        a[i][j] = 0;  // 连续内存访问
    }
}

// 缓存不友好 (按列访问)
for (int j = 0; j < n; j++) {
    for (int i = 0; i < n; i++) {
        a[i][j] = 0;  // 跳跃式访问
    }
}
```

---

## 编译器优化

```bash
# 启用优化
gcc -O3 -march=native program.c

# 向量化
gcc -O3 -ftree-vectorize -msse4.2 program.c
```
