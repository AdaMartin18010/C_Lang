# C 语言声明解析：螺旋法则（The Spiral Rule）

> **参考**: Expert C Programming (Peter van der Linden), c-faq.com

## 核心法则

```
                  +-------+
                 /         \
                |   标识符  |
                 \         /
                  +-------+
                       |
         从标识符开始，螺旋向外（顺时针），
         遇到运算符时按优先级解析
```

**解析顺序**：
1. 找到**标识符**（变量名/函数名/类型名）
2. **优先处理括号内的内容**
3. **向右**遇到 `()` → 函数；遇到 `[]` → 数组
4. **向左**遇到 `*` → 指针；遇到 `const` → 常量限定
5. 重复 2-4 直到解析完毕

## 渐进示例

### Level 1: 基础声明

```c
int foo;
```

```
foo 是 ... int
     ↑
   标识符
```

**读法**：`foo` 是一个 `int`。

---

### Level 2: 指针

```c
int *foo;
```

```
     foo 是 ... * ... int
      ↑          ↑
    标识符    向左遇到 *
```

**读法**：`foo` 是一个**指针**，指向 `int`。

---

### Level 3: 数组

```c
int foo[5];
```

```
     foo 是 ... [5] ... int
      ↑           ↑
    标识符     向右遇到 []
```

**读法**：`foo` 是一个**数组**，包含 5 个 `int`。

---

### Level 4: 指针数组 vs 数组指针

```c
int *foo[5];      /* 指针数组 */
int (*foo)[5];    /* 数组指针 */
```

**指针数组** `int *foo[5]`：
```
     foo 是 ... [5] ... * ... int
      ↑           ↑        ↑
    标识符     向右[]    向左*
```
**读法**：`foo` 是一个数组，包含 5 个**指针**，每个指向 `int`。

**数组指针** `int (*foo)[5]`：
```
     foo 是 ... (*) ... [5] ... int
      ↑          ↑         ↑
    标识符    括号优先   向右[]
```
**读法**：`foo` 是一个**指针**，指向一个数组，该数组包含 5 个 `int`。

> ⚠️ **关键区别**：`[]` 优先级高于 `*`。`int *foo[5]` 等价于 `int *(foo[5])`。

---

### Level 5: 函数指针

```c
int *foo(void);
```

```
     foo 是 ... () ... * ... int
      ↑          ↑       ↑
    标识符    向右()   向左*
```

**读法**：`foo` 是一个**函数**，参数为 `void`，返回**指针**（指向 `int`）。

```c
int (*foo)(void);
```

```
     foo 是 ... (*) ... () ... int
      ↑          ↑        ↑
    标识符    括号优先  向右()
```

**读法**：`foo` 是一个**指针**，指向一个函数（参数 `void`），该函数返回 `int`。

---

### Level 6: 复杂声明

```c
int (*(*foo)[5])(int);
```

**逐步解析**：

```
Step 1: 找到标识符 foo
Step 2: 括号优先: (*foo) → foo 是指针
Step 3: 向右: [5] → 指向一个包含 5 个元素的数组
Step 4: 向左: * → 数组元素是指针
Step 5: 向右: (int) → 这些指针指向函数，参数为 int
Step 6: 向左: int → 函数返回 int
```

**读法**：`foo` 是一个**指针**，指向一个包含 5 个元素的**数组**，数组元素是**指针**，每个指针指向一个**函数**（参数为 `int`），这些函数返回 `int`。

---

### Level 7: 函数返回函数指针

```c
int (*foo(void))(int);
```

```
     foo 是 ... () ... * ... (int) ... int
      ↑          ↑       ↑        ↑
    标识符    向右()   向左*   向右(int)
```

**读法**：`foo` 是一个**函数**（参数 `void`），返回一个**指针**，该指针指向一个**函数**（参数 `int`），返回 `int`。

等价于 `typedef int (*FuncPtr)(int); FuncPtr foo(void);`

---

### Level 8: 信号处理函数

```c
void (*signal(int sig, void (*func)(int)))(int);
```

**逐步解析**：

```
signal 是 ... (int sig, void (*func)(int)) ... * ... (int) ... void
```

**读法**：`signal` 是一个**函数**，参数为 `int sig` 和一个**函数指针** `func`（参数 `int`，返回 `void`）；`signal` 返回一个**指针**，该指针指向一个**函数**（参数 `int`），返回 `void`。

> 这正是 `signal(2)` 的原型！

---

## 速查对照表

| 声明 | 读法 |
|------|------|
| `int a;` | a 是 int |
| `int *a;` | a 是指向 int 的指针 |
| `int a[5];` | a 是包含 5 个 int 的数组 |
| `int *a[5];` | a 是包含 5 个指针的数组，指针指向 int |
| `int (*a)[5];` | a 是指向数组的指针，数组包含 5 个 int |
| `int a(void);` | a 是函数（无参数），返回 int |
| `int *a(void);` | a 是函数（无参数），返回指向 int 的指针 |
| `int (*a)(void);` | a 是指向函数的指针，函数无参数返回 int |
| `int (*a[5])(void);` | a 是包含 5 个指针的数组，每个指针指向函数（无参数返回 int） |
| `int (*(*a)[5])(void);` | a 是指向数组的指针，数组包含 5 个指针，每个指针指向函数（无参数返回 int） |

## typedef 解药

复杂声明应使用 `typedef` 分解：

```c
/* 原始复杂声明 */
int (*(*signal(int, void (*)(int)))(int);

/* 分解为 typedef */
typedef void (*SignalHandler)(int);
SignalHandler signal(int sig, SignalHandler handler);
```

---

## 常见错误

| 错误声明 | 意图 | 正确写法 |
|---------|------|---------|
| `int* a, b;` | a 和 b 都是指针 | `int *a, *b;` |
| `const int *p;` | p 指向的 int 不可变 | ✅ 正确（p 可变，*p 不可变） |
| `int *const p;` | p 本身不可变 | ✅ 正确（p 不可变，*p 可变） |
| `const int *const p;` | p 和 *p 都不可变 | ✅ 正确 |
| `void (*fp)();` | fp 指向函数 | ✅ 正确（但 () 参数未指定，C23 废弃） |
| `void *fp();` | fp 是返回指针的函数 | 如要函数指针需加括号 |

## 练习

尝试解析以下声明：

```c
char * const *(*next)(void);
void (*(*f[])())();
int (*(*foo())[5])();
```

<details>
<summary>点击查看答案</summary>

1. `next` 是指针，指向函数（无参数），返回指针，该指针指向 `char * const`（指向 char 的常量指针）。
2. `f` 是数组，元素是指针，指向函数（无参数），返回指针，指向函数（无参数），返回 void。
3. `foo` 是函数（无参数），返回指针，指向数组（5 个元素），元素是指针，指向函数（无参数），返回 int。

</details>
