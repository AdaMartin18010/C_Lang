## 五、实际案例

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│                    性能对比实际案例                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  案例 1: 矩阵乘法                                                            │
│  ─────────────────────────────────────────                                   │
│                                                                              │
│  C (gcc -O3):                                                                │
│  ┌─────────────────────────────────────────┐                                 │
│  │ void matmul(float* C, float* A,         │                                 │
│  │             float* B, int n) {          │                                 │
│  │     for (int i = 0; i < n; i++)         │                                 │
│  │         for (int j = 0; j < n; j++)     │                                 │
│  │             for (int k = 0; k < n; k++) │                                 │
│  │                 C[i*n+j] += A[i*n+k]    │                                 │
│  │                          * B[k*n+j];    │                                 │
│  │ }                                       │                                 │
│  └─────────────────────────────────────────┘                                 │
│                                                                              │
│  Zig (ReleaseFast):                                                          │
│  ┌─────────────────────────────────────────┐                                 │
│  │ pub fn matmul(                          │                                 │
│  │     C: []f32, A: []const f32,           │                                 │
│  │     B: []const f32, n: usize            │                                 │
│  │ ) void {                                │                                 │
│  │     for (0..n) |i| {                    │                                 │
│  │         for (0..n) |j| {                │                                 │
│  │             for (0..n) |k| {            │                                 │
│  │                 C[i*n+j] +=             │                                 │
│  │                     A[i*n+k] * B[k*n+j];│                                 │
│  │             }                           │                                 │
│  │         }                               │                                 │
│  │     }                                   │                                 │
│  │ }                                       │                                 │
│  └─────────────────────────────────────────┘                                 │
│                                                                              │
│  结果 (1024x1024):                                                            │
│  • C: 2.5s                                                                   │
│  • Zig: 2.6s (104% of C)                                                     │
│  • 结论: 性能相当                                                            │
│                                                                              │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                              │
│  案例 2: 内存分配器                                                          │
│  ─────────────────────────────────────────                                   │
│                                                                              │
│  C (malloc/free):                                                            │
│  • 100万次分配/释放: ~150ms                                                  │
│                                                                              │
│  Zig (GeneralPurposeAllocator):                                              │
│  • Debug: ~250ms (安全检查开销)                                              │
│  • ReleaseFast: ~140ms (比 C 更快!)                                          │
│                                                                              │
│  结论: Zig 的 GPA 在发布模式下具有竞争力，且提供更多安全功能                 │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
