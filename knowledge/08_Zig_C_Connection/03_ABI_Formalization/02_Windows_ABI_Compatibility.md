## 一、Windows x64 ABI 总览

### 1.1 与 System V AMD64 ABI 的核心差异

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│              Windows x64 ABI vs System V AMD64 ABI 对比                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  特性                    │  Windows x64          │  System V AMD64          │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  整数参数寄存器          │  RCX, RDX, R8, R9     │  RDI, RSI, RDX,          │
│  (前4个)                 │  (仅4个)              │  RCX, R8, R9 (6个)       │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  浮点参数寄存器          │  XMM0-XMM3            │  XMM0-XMM7               │
│  (前4个)                 │  (仅4个)              │  (最多8个)               │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  额外参数传递            │  栈 (RSP+0x28开始)    │  栈 (RSP+8开始)          │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  调用者栈空间            │  必须分配 32 字节     │  无强制要求              │
│  (Shadow Space)          │  (Shadow Space)       │                          │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  结构体分类              │  简化分类             │  复杂8类分类系统         │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  变参函数                │  所有参数必须         │  寄存器参数不保存        │
│                          │  保存在栈上           │                          │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  栈对齐                  │  16 字节对齐          │  16 字节对齐             │
│  ────────────────────────┼───────────────────────┼────────────────────────  │
│  红区 (Red Zone)         │  不存在               │  128 字节                │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 寄存器使用约定

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Windows x64 寄存器分类                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  参数传递寄存器 (Volatile):                                                   │
│  ─────────────────────────────────────────────────────────────────────────  │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 整数:  RCX, RDX, R8, R9                                              │   │
│  │ 浮点:  XMM0, XMM1, XMM2, XMM3                                        │   │
│  │ 向量:  YMM0-YMM3 (低128位 = XMM), ZMM0-ZMM3 (如果支持 AVX-512)       │   │
│  ├─────────────────────────────────────────────────────────────────────┤   │
│  │ 规则: 被调用者可以随意修改，调用者负责保存                           │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
│  被调用者保存寄存器 (Non-volatile):                                           │
│  ─────────────────────────────────────────────────────────────────────────  │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 整数:  RDI, RSI, RBX, RBP, R12, R13, R14, R15, RSP                   │   │
│  │ 浮点:  XMM4-XMM15 (以及对应的 YMM/ZMM)                               │   │
│  ├─────────────────────────────────────────────────────────────────────┤   │
│  │ 规则: 如果被修改，必须恢复原始值                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
│  特殊寄存器:                                                                  │
│  ─────────────────────────────────────────────────────────────────────────  │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ RAX    - 返回值 (整数)                                               │   │
│  │ XMM0   - 返回值 (浮点/向量)                                          │   │
│  │ R10    - 用于间接调用 (如虚函数表)                                   │   │
│  │ R11    - 临时/未使用                                                 │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 二、参数传递机制

### 2.1 整数与指针参数

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│                    Windows x64 整数参数传递                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  寄存器顺序 (前4个参数):                                                      │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                              │
│  参数位置   │  64位     │  32位     │  16位    │  8位                      │
│  ──────────┼───────────┼───────────┼──────────┼──────────────────────────  │
│  第1个     │  RCX      │  ECX      │  CX      │  CL                       │
│  第2个     │  RDX      │  EDX      │  DX      │  DL                       │
│  第3个     │  R8       │  R8D      │  R8W     │  R8B                      │
│  第4个     │  R9       │  R9D      │  R9W     │  R9B                      │
│  第5+个    │  栈       │  栈       │  栈      │  栈                       │
│                                                                              │
│  栈参数位置:                                                                 │
│  ─────────────────────────────────────────────────────────────────────────  │
│  • Shadow Space: RSP+0x00 到 RSP+0x1F (32字节)                              │
│  • 第5个参数:    RSP+0x28 (第5个整数或浮点)                                  │
│  • 第6个参数:    RSP+0x30                                                    │
│  • ...以此类推，每个参数 8 字节                                              │
│                                                                              │
│  重要规则:                                                                   │
│  ─────────────────────────────────────────────────────────────────────────  │
│  • 即使参数少于4个，也必须分配 Shadow Space                                 │
│  • 32位参数在寄存器中零扩展到 64 位                                          │
│  • 16位和8位参数在寄存器中零扩展到 64 位                                     │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

```c
// C 代码: Windows x64 整数参数传递示例
// 编译: cl.exe /FA /O2 test_int.c

#include <stdint.h>

// 前4个参数使用寄存器
void four_args(int a, int b, int c, int d) {
    // a: ECX (零扩展到 RCX)
    // b: EDX
    // c: R8D
    // d: R9D
}

// 第5个参数使用栈
void five_args(int a, int b, int c, int d, int e) {
    // a-d: ECX, EDX, R8D, R9D
    // e: [RSP+0x28] (在 Shadow Space 之后)
}

// 混合类型
void mixed_types(int64_t big, int32_t small, int16_t tiny, int8_t mini) {
    // big:   RCX
    // small: EDX (零扩展到 RDX)
    // tiny:  R8W (零扩展到 R8)
    // mini:  R9B (零扩展到 R9)
}
```

```zig
// Zig 代码: 对应的 Windows x64 函数声明

// 前4个参数使用寄存器
extern "C" fn fourArgs(a: c_int, b: c_int, c: c_int, d: c_int) void;
// 与 C 代码完全相同的寄存器分配

// 第5个参数使用栈
extern "C" fn fiveArgs(a: c_int, b: c_int, c: c_int, d: c_int, e: c_int) void;
// a-d: RCX, RDX, R8, R9
// e: [RSP+0x28]

// 混合类型
extern "C" fn mixedTypes(big: i64, small: i32, tiny: i16, mini: i8) void;
// 与 C 代码完全相同的寄存器分配
```

### 2.2 浮点参数

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│                    Windows x64 浮点参数传递                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  寄存器顺序 (前4个浮点参数):                                                  │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                              │
│  参数位置   │  寄存器   │  类型                                          │
│  ──────────┼───────────┼─────────────────────────────────────────────── │
│  第1个     │  XMM0     │  float / double / __m128 / __m256              │
│  第2个     │  XMM1     │  float / double / __m128 / __m256              │
│  第3个     │  XMM2     │  float / double / __m128 / __m256              │
│  第4个     │  XMM3     │  float / double / __m128 / __m256              │
│  第5+个    │  栈       │  (在 Shadow Space 之后)                        │
│                                                                              │
│  关键差异 (vs System V):                                                     │
│  ─────────────────────────────────────────────────────────────────────────  │
│  • Windows: 整数和浮点参数共享同一个 "槽位" 计数                           │
│  • System V: 整数和浮点参数独立计数                                          │
│                                                                              │
│  示例:                                                                       │
│  ─────────────────────────────────────────────────────────────────────────  │
│  void f(int a, double b, int c, float d);                                   │
│                                                                              │
│  Windows x64:                                                                │
│  • a: ECX (整数槽位1)                                                        │
│  • b: XMM1 (浮点槽位2, 与整数共享计数)                                      │
│  • c: R8D  (整数槽位3)                                                       │
│  • d: XMM3 (浮点槽位4)                                                       │
│                                                                              │
│  System V AMD64:                                                             │
│  • a: EDI (整数1)                                                            │
│  • b: XMM0 (浮点1, 独立计数)                                                │
│  • c: ESI (整数2)                                                            │
│  • d: XMM1 (浮点2)                                                           │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

```c
// C 代码: Windows x64 浮点参数示例

void float_args(float a, double b, float c, double d) {
    // a: XMM0 (32位)
    // b: XMM1 (64位)
    // c: XMM2 (32位)
    // d: XMM3 (64位)
}

void mixed_int_float(int a, double b, int c, float d) {
    // Windows x64: 共享槽位计数
    // a: ECX  (槽位1)
    // b: XMM1 (槽位2)
    // c: R8D  (槽位3)
    // d: XMM3 (槽位4)
}
```

### 2.3 结构体参数

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│                  Windows x64 结构体参数传递                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  结构体分类规则:                                                             │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                              │
│  1. 可放入寄存器的结构体 (整数或浮点类):                                      │
│     ─────────────────────────────────────                                    │
│     • 大小 <= 8 字节                                                         │
│     • 只包含整数/指针 或 只包含浮点                                          │
│     • 混合类型结构体（整数+浮点）必须使用栈                                  │
│                                                                              │
│  2. 必须传递指针的结构体:                                                     │
│     ─────────────────────────────────────                                    │
│     • 大小 > 8 字节                                                          │
│     • 混合整数和浮点成员                                                     │
│     • 包含数组的结构体                                                       │
│                                                                              │
│  与 System V 的主要区别:                                                     │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                              │
│  Windows x64                              │  System V AMD64                 │
│  ─────────────────────────────────────────┼──────────────────────────────── │
│  • 最多使用2个寄存器传递结构体             │  • 最多使用2个寄存器            │
│  • 混合类型结构体强制使用栈                │  • 使用复杂分类算法             │
│  • 不存在 SSEUP/X87UP 概念                 │  • 可以部分使用寄存器           │
│  • 返回值 > 8字节使用隐藏指针              │  • 类似规则但实现不同           │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

```c
// C 代码: Windows x64 结构体传递

// 8字节整数结构体 - 使用 RCX
struct IntPair {
    int a, b;
};
void pass_int_pair(struct IntPair p) {
    // p: RCX (打包两个32位整数)
}

// 8字节浮点结构体 - 使用 XMM0
struct FloatPair {
    float a, b;
```

```zig
    lParam: LPARAM,

fn windowProc(
    hwnd: HWND,
    uMsg: u32,
    wParam: WPARAM,
    lParam: LPARAM,
```
