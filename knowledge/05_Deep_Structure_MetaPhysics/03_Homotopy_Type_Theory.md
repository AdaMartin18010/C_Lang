﻿# 同伦类型理论

## 概述

**同伦类型理论**（Homotopy Type Theory, HoTT）是2010年代初期发展起来的一种新型数学基础理论，由 Vladimir Voevodsky 等人创立。它将同伦论（代数拓扑学的一个分支）与类型论（数理逻辑的基础）统一起来，提出了一种全新的数学基础范式。

HoTT 的核心洞见是：类型可以被解释为空间，类型的元素可以被视为空间中的点，而同一性证明则对应于点之间的路径。这种对应关系开启了数学基础研究的新纪元。

---

## 1. 同伦论基础

### 1.1 同伦的概念

同伦（Homotopy）是拓扑学中描述连续形变的核心概念。

**定义**：设 $X$ 和 $Y$ 是拓扑空间，$f, g: X \to Y$ 是连续映射。如果存在连续映射 $H: X \times [0,1] \to Y$ 使得：

- $H(x, 0) = f(x)$
- $H(x, 1) = g(x)$

则称 $f$ 和 $g$ 是**同伦的**，记作 $f \simeq g$。

**直观理解**：同伦表示一个映射可以连续地变形为另一个映射，就像拉伸橡皮筋一样，不允许切断或粘合。

### 1.2 同伦等价

两个空间 $X$ 和 $Y$ 称为**同伦等价**的，如果存在映射 $f: X \to Y$ 和 $g: Y \to X$ 使得：

- $g \circ f \simeq \text{id}_X$
- $f \circ g \simeq \text{id}_Y$

同伦等价是比同胚（homeomorphism）更弱的关系，它允许空间有"洞"或不同的维度特征，只要它们的"形状"在连续变形意义下相同。

### 1.3 同伦群

**基本群**（第一同伦群）$\pi_1(X, x_0)$ 描述基于点 $x_0$ 的环路在同伦意义下的等价类。

**高阶同伦群** $\pi_n(X, x_0)$ 推广到 $n$ 维球面 $S^n$ 到空间 $X$ 的映射。

## 2. 路径类型

### 2.1 路径类型的定义

在同伦类型理论中，给定类型 $A$ 和元素 $a, b : A$，**路径类型**（Path Type）记为：

$$\text{Path}_A(a, b) \quad \text{或} \quad a =_A b$$

路径类型的元素是连接 $a$ 和 $b$ 的"路径"或"证明"。

### 2.2 路径的操作

#### 路径的复合（Concatenation）

给定路径 $p: a = b$ 和 $q: b = c$，可以定义复合路径：

$$p \cdot q : a = c$$

#### 路径的逆（Inverse）

给定路径 $p: a = b$，其逆路径为：

$$p^{-1} : b = a$$

#### 常值路径（Reflexivity）

对于任意 $a : A$，存在常值路径：

$$\text{refl}_a : a = a$$

### 2.3 路径的空间结构

关键洞见：路径类型本身又是一个类型，因此路径之间也可以有路径（称为**同伦**或**2-路径**）：

$$\alpha : p =_{a=b} q$$

这形成了无限层次的层级结构：

- 0-路径：点（元素）
- 1-路径：点之间的路径
- 2-路径：路径之间的路径
- ...
- $n$-路径：$(n-1)$-路径之间的路径

---

## 3. 同一性类型

### 3.1 归纳定义

同一性类型（Identity Type）通过归纳原理定义。对于类型 $A : \mathcal{U}$，同一性类型族：

$$=_A : A \to A \to \mathcal{U}$$

由以下构造子生成：

- **自反性**：$\text{refl} : \prod_{a:A} (a =_A a)$

### 3.2 基于路径的归纳原理

同一性类型的归纳原理（J-规则）表述为：

给定类型族 $C : \prod_{a,b:A} (a =_A b) \to \mathcal{U}$ 和函数 $c : \prod_{a:A} C(a, a, \text{refl}_a)$，存在唯一的 $J_{C,c}$ 满足：

$$J_{C,c}(a, a, \text{refl}_a) = c(a)$$

### 3.3 同一性 vs 等价性

传统数学中，相等关系是命题性的（要么成立要么不成立）。在 HoTT 中：

- **同一性类型**是结构性的，包含证明信息
- 不同的证明（路径）可以不同
- 同一性类型可以有非平凡的结构（高阶路径）

## 4. Univalence 公理

### 4.1 等价的概念

在类型论中，函数 $f: A \to B$ 称为**等价**，如果：

- 它存在拟逆（quasi-inverse）：存在 $g: B \to A$ 使得 $f \circ g \sim \text{id}_B$ 且 $g \circ f \sim \text{id}_A$

等价类型记为 $A \simeq B$。

### 4.2 Univalence Axiom

**Voevodsky 的 Univalence 公理**（UA）是 HoTT 的核心创新：

$$(A =_\mathcal{U} B) \simeq (A \simeq B)$$

即：
> 类型的相等等价于类型的等价

更精确地说，存在规范映射 $\text{idtoeqv} : (A =_\mathcal{U} B) \to (A \simeq B)$，Univalence 断言这是一个等价。

### 4.3 Univalence 的推论

#### 结构不变性

若 $A \simeq B$，则 $A$ 和 $B$ 具有完全相同的结构性质。即对于任意类型族 $P : \mathcal{U} \to \mathcal{U}$：

$$P(A) \simeq P(B)$$

#### 同构即相等

在数学实践中，数学家常说"同构的对象是相同的"。Univalence 公理将这一直观形式化。

### 4.4 数学结构传输

Univalence 允许通过等价自动传输结构：

```
如果 A ≅ B（同构），且 A 上有群结构
则该结构可以自动传输到 B 上
```

这在形式化数学中极大地简化了证明。

## 5. 高阶归纳类型

### 5.1 传统归纳类型

传统归纳类型（如自然数、列表）由**点构造子**定义：

- 自然数：$0 : \mathbb{N}$ 和 $\text{succ} : \mathbb{N} \to \mathbb{N}$

### 5.2 高阶归纳类型的定义

**高阶归纳类型**（Higher Inductive Types, HITs）允许同时引入：

- **0-维构造子**（点构造子）：生成类型的元素
- **n-维构造子**（路径构造子）：生成元素之间的路径

### 5.3 典型示例

#### 区间类型（Interval Type）

```agda
data Interval : Set where
  zero  : Interval
  one   : Interval
  seg   : zero ≡ one   -- 路径构造子
```

区间类型有两个端点和一个连接它们的路径。

#### 圆（Circle）

```agda
data S¹ : Set where
  base : S¹
  loop : base ≡ base   -- 非平凡环路
```

圆有一个基点和一个非平凡的环路。其基本群 $\pi_1(S¹) = \mathbb{Z}$。

#### 球面（Sphere）

```agda
data S² : Set where
  base : S²
  surf : refl ≡ refl   -- 2-维表面
```

#### 推出（Pushout）

给定 $f: A \to B$ 和 $g: A \to C$，推出是：

```agda
data Pushout (f : A → B) (g : A → C) : Set where
  inl  : B → Pushout f g
  inr  : C → Pushout f g
  glue : (a : A) → inl (f a) ≡ inr (g a)
```

### 5.4 应用：代数拓扑的形式化

HITs 允许直接在类型论中形式化拓扑构造：

- 同伦群 $\pi_n(X)$
- 纬垂（Suspension）
- 上纤维（Cofiber）
- 同调论

### 5.5 商类型

HITs 提供了构造**商类型**的通用方法：

```agda
data Quotient (A : Set) (R : A → A → Set) : Set where
  proj  : A → Quotient A R
  ident : (a b : A) → R a b → proj a ≡ proj b
  trunc : (x y : Quotient A R) → (p q : x ≡ y) → p ≡ q  -- 集截断
```

## 6. HoTT 的数学意义

### 6.1 构造性数学

HoTT 是**构造性**的：每个存在性命题都有计算内容，每个证明都可以被执行。

### 6.2 证明的相关性

传统命题逻辑中，证明是无关的（proof irrelevant）：$p = q$ 对所有证明 $p, q$ 成立。

HoTT 允许**证明相关**的数学，不同的证明可以携带不同的计算信息。

### 6.3 合成同伦论

HoTT 提供了"合成"的同伦论方法，不依赖于点集拓扑的复杂构造，直接从类型论的规则推导拓扑结果。

### 7.1 证明助手

- **Coq**（带 HoTT 库）

### 7.2 Cubical Type Theory

为了解决 Univalence 的计算问题，发展出了**立方类型论**（Cubical Type Theory）：

- 引入区间 $[0,1]$ 作为原生概念
- 提供计算意义的 Univalence
- 实现了计算完备的路径操作

## 总结

同伦类型理论统一了：

| 类型论语义 | 拓扑语义 | 逻辑语义 |
|-----------|---------|---------|
| 类型 $A$ | 空间 $X$ | 命题 $P$ |
| 元素 $a : A$ | 点 $x \in X$ | 证明 $p$ |
| 路径 $p : a = b$ | 路径 $p$ | 同一性证明 |
| 路径类型 | 路径空间 | 等价关系 |
| HIT | CW-复形 | 归纳定义 |

Univalence 公理将数学结构的"等价即相等"原则形式化，高阶归纳类型则将拓扑构造引入类型论。HoTT 代表了数学基础研究的一次范式转变。
