## 1. 集合论基础 (Set Theory Foundations)

### 1.1 基本定义

**定义 1.1.1** (集合)：集合是对象的汇集，记作：

```
A = {x | P(x)}
```

其中 P(x) 是定义成员关系的谓词。

**定义 1.1.2** (隶属关系)：

```
x ∈ A  ⇔  x 是 A 的元素
x ∉ A  ⇔  ¬(x ∈ A)
```

**定义 1.1.3** (子集)：

```
A ⊆ B  ⇔  ∀x(x ∈ A → x ∈ B)
A ⊂ B  ⇔  A ⊆ B ∧ A ≠ B
```

### 1.2 集合运算

**定义 1.2.1** (基本运算)：

```
并集:    A ∪ B = {x | x ∈ A ∨ x ∈ B}
交集:    A ∩ B = {x | x ∈ A ∧ x ∈ B}
差集:    A \ B = {x | x ∈ A ∧ x ∉ B}
补集:    A^c = {x | x ∉ A}  (相对于全集 U)
幂集:    P(A) = {B | B ⊆ A}
```

**定理 1.2.1** (幂集基数)：

```
|A| = n  ⇒  |P(A)| = 2^n
```

*证明*：A 的每个子集对应于一个 n 位二进制数，共有 2^n 个。 ∎

### 1.3 映射 (Functions/Mappings)

**定义 1.3.1** (映射)：从集合 A 到集合 B 的映射 f 是 A × B 的子集，满足：

```
f: A → B  ⇔  f ⊆ A × B ∧ ∀a∈A.∃!b∈B.(a,b) ∈ f
```

记作 f(a) = b 当 (a,b) ∈ f。

**定义 1.3.2** (映射类型)：

```
单射 (Injective):   f(a₁) = f(a₂) → a₁ = a₂
满射 (Surjective):  ∀b∈B.∃a∈A.f(a) = b
双射 (Bijective):   单射 ∧ 满射
```

**定义 1.3.3** (复合映射)：

```
给定 f: A → B, g: B → C
g ∘ f: A → C,  (g ∘ f)(a) = g(f(a))
```

**定理 1.3.1** (复合结合律)：

```
h ∘ (g ∘ f) = (h ∘ g) ∘ f
```

*证明*：对于任意 a ∈ A：

```
(h ∘ (g ∘ f))(a) = h((g ∘ f)(a)) = h(g(f(a)))
((h ∘ g) ∘ f)(a) = (h ∘ g)(f(a)) = h(g(f(a)))
```

两边相等。 ∎

### 1.4 关系 (Relations)

**定义 1.4.1** (关系)：集合 A 上的二元关系 R 是 A × A 的子集：

```
R ⊆ A × A,  记 aRb 当 (a,b) ∈ R
```

**定义 1.4.2** (关系性质)：

```
自反性:    ∀a∈A.aRa
对称性:    ∀a,b∈A.aRb → bRa
传递性:    ∀a,b,c∈A.aRb ∧ bRc → aRc
反对称性:  ∀a,b∈A.aRb ∧ bRa → a = b
```

**定义 1.4.3** (等价关系)：

```
R 是等价关系  ⇔  R 是自反的、对称的、传递的
等价类: [a]_R = {b ∈ A | aRb}
商集:   A/R = {[a]_R | a ∈ A}
```

**定理 1.4.1** (等价类划分)：

```
R 是 A 上的等价关系 ⇒ A/R 构成 A 的划分
```

即：(1) ∀a.[a]_R ≠ ∅, (2) [a]_R ∩ [b]_R ≠ ∅ → [a]_R = [b]_R, (3) ∪[a]_R = A

## 2. 序理论 (Order Theory)

### 2.1 偏序与全序

**定义 2.1.1** (偏序)：集合 P 上的偏序 ≤ 是满足以下条件的关系：

```
自反性:    ∀x∈P.x ≤ x
反对称性:  ∀x,y∈P.x ≤ y ∧ y ≤ x → x = y
传递性:    ∀x,y,z∈P.x ≤ y ∧ y ≤ z → x ≤ z
```

称 (P, ≤) 为**偏序集** (Poset)。

**定义 2.1.2** (全序)：

```
≤ 是全序  ⇔  ≤ 是偏序 ∧ ∀x,y∈P.x ≤ y ∨ y ≤ x
```

**定义 2.1.3** (上下界)：

```
上界:  u 是 S ⊆ P 的上界  ⇔  ∀x∈S.x ≤ u
下界:  l 是 S ⊆ P 的下界  ⇔  ∀x∈S.l ≤ x

最小上界 (LUB/Supremum):  ⊔S = min{u | u 是 S 的上界}
最大下界 (GLB/Infimum):   ⊓S = max{l | l 是 S 的下界}
```

### 2.2 格 (Lattice)

**定义 2.2.1** (格)：偏序集 (L, ≤) 是格，如果：

```
∀x,y∈L.  x ⊔ y (join) 和 x ⊓ y (meet) 都存在
```

**定义 2.2.2** (代数格)：代数结构 (L, ∨, ∧) 满足：

```
交换律:  x ∨ y = y ∨ x,    x ∧ y = y ∧ x
结合律:  (x ∨ y) ∨ z = x ∨ (y ∨ z),    (x ∧ y) ∧ z = x ∧ (y ∧ z)
吸收律:  x ∨ (x ∧ y) = x,    x ∧ (x ∨ y) = x
```

**定理 2.2.1** (序格与代数格等价)：

```
(L, ≤) 是序格  ⇔  (L, ∨, ∧) 是代数格
其中 x ∨ y = ⊔{x,y}, x ∧ y = ⊓{x,y}
      x ≤ y ⇔ x ∨ y = y ⇔ x ∧ y = x
```

**定义 2.2.3** (完全格)：

```
(L, ≤) 是完全格  ⇔  ∀S⊆L. ⊔S 和 ⊓S 都存在
```

**定理 2.2.2** (Knaster-Tarski 不动点定理)：

```
设 (L, ≤) 是完全格，f: L → L 是单调函数
则 f 有最小不动点和最大不动点：
  μf = ⊓{x | f(x) ≤ x}   (最小不动点)
  νf = ⊔{x | x ≤ f(x)}   (最大不动点)
```

*证明概要*：
设 A = {x | f(x) ≤ x}（后不动点集），a = ⊓A。
对任意 x ∈ A，有 a ≤ x，由单调性 f(a) ≤ f(x) ≤ x。
因此 f(a) ≤ ⊓A = a，即 a ∈ A。
又 f(a) ≤ a ⇒ f(f(a)) ≤ f(a)（单调性）⇒ f(a) ∈ A ⇒ a ≤ f(a)。
故 f(a) = a。最小性由构造保证。 ∎

### 2.3 链与完备性

**定义 2.3.1** (链)：

```
C ⊆ P 是链  ⇔  ∀x,y∈C.x ≤ y ∨ y ≤ x
```

**定义 2.3.2** (完全偏序 - CPO)：

```
(D, ≤) 是 CPO  ⇔
  (1) D 有最小元 ⊥
  (2) D 中每条链都有最小上界
```

**定义 2.3.3** (Scott 连续)：

```
f: D → E 是 Scott 连续的  ⇔  f(⊔C) = ⊔f(C) 对非空链 C 成立
```

## 3. 范畴论基础 (Category Theory Basics)

### 3.1 范畴的定义

**定义 3.1.1** (范畴)：范畴 **C** 由以下组成：

```
- 对象类: Ob(C)
- 态射类: 对任意 A,B ∈ Ob(C)，有 Hom_C(A,B)（A 到 B 的态射集）
- 复合运算: ∘ : Hom(B,C) × Hom(A,B) → Hom(A,C)
- 单位态射: 对每个 A，有 id_A ∈ Hom(A,A)
```

满足公理：

```
结合律:  h ∘ (g ∘ f) = (h ∘ g) ∘ f
单位律:  id_B ∘ f = f = f ∘ id_A  对 f: A → B
```

### 3.2 态射类型

**定义 3.2.1** (态射分类)：

```
单态射 (Monomorphism):  f ∘ g₁ = f ∘ g₂ → g₁ = g₂
满态射 (Epimorphism):   g₁ ∘ f = g₂ ∘ f → g₁ = g₂
同构 (Isomorphism):     ∃g. g ∘ f = id_A ∧ f ∘ g = id_B
```

### 3.3 函子 (Functor)

**定义 3.3.1** (协变函子)：函子 F: **C** → **D** 满足：

```
对象映射: A ∈ Ob(C) ↦ F(A) ∈ Ob(D)
态射映射: f: A → B ↦ F(f): F(A) → F(B)

保持复合: F(g ∘ f) = F(g) ∘ F(f)
保持单位: F(id_A) = id_{F(A)}
```

**定义 3.3.2** (逆变函子)：

```
F(f: A → B): F(B) → F(A)
F(g ∘ f) = F(f) ∘ F(g)
```

**定义 3.3.3** (遗忘函子)：

```
U: Grp → Set,  将群映射到其底层集合
U: Top → Set,  将拓扑空间映射到其底层集合
```

### 3.4 自然变换

**定义 3.4.1** (自然变换)：给定函子 F,G: **C** → **D**，自然变换 η: F ⇒ G 由以下组成：

```
对每个 A ∈ Ob(C)，有分量 η_A: F(A) → G(A)

满足自然性条件（交换图）：

    F(A) ──F(f)──> F(B)
     |               |
   η_A|               |η_B
     ↓               ↓
    G(A) ──G(f)──> G(B)

即: G(f) ∘ η_A = η_B ∘ F(f)  对所有 f: A → B
```

## 4. 类型理论 (Type Theory)

### 4.1 简单类型 λ 演算

**定义 4.1.1** (类型)：

```
τ ::= B | τ → τ | τ × τ | 1 | 0

B: 基本类型 (Bool, Nat, ...)
τ₁ → τ₂: 函数类型
τ₁ × τ₂: 积类型
1: 单位类型
0: 空类型
```

**定义 4.1.2** (上下文与类型判断)：

```
上下文 Γ ::= ∅ | Γ, x:τ

类型判断规则:

(变量)  ─────────────  x:τ ∈ Γ
        Γ ⊢ x : τ

(抽象)  Γ, x:τ₁ ⊢ e : τ₂
        ─────────────────
        Γ ⊢ λx.e : τ₁ → τ₂

(应用)  Γ ⊢ e₁ : τ₁ → τ₂    Γ ⊢ e₂ : τ₁
        ─────────────────────────────
        Γ ⊢ e₁ e₂ : τ₂
```

### 4.2 多态类型

**定义 4.2.1** (System F / 多态 λ 演算)：

```
τ ::= α | τ → τ | ∀α.τ
e ::= x | λx:τ.e | e e | Λα.e | e[τ]

类型抽象: Λα.e  表示类型为 ∀α.τ 的项
类型应用: e[τ]  将多态类型实例化
```

**类型规则**：

```
(类型抽象)  Γ ⊢ e : τ    α ∉ FV(Γ)
           ─────────────────────
           Γ ⊢ Λα.e : ∀α.τ

(类型应用)  Γ ⊢ e : ∀α.τ₁
           ─────────────────
           Γ ⊢ e[τ] : τ₁[τ/α]
```

### 4.3 依赖类型

**定义 4.3.1** (依赖函数类型 - Π 类型)：

```
Π(x:A).B(x) 表示依赖函数类型

f : Π(x:A).B(x) 意味着 ∀x:A. f(x) : B(x)
```

**定义 4.3.2** (依赖对类型 - Σ 类型)：

```
Σ(x:A).B(x) 表示依赖对类型

(a, b) : Σ(x:A).B(x) 意味着 a : A 且 b : B(a)
```

**示例**：向量类型

```
Vec : Type → ℕ → Type
Vec A n = 长度为 n 的 A 向量

append : Π(n,m:ℕ). Vec A n → Vec A m → Vec A (n+m)
```

### 4.4 Curry-Howard 同构

**定理 4.4.1** (Curry-Howard 同构)：

```
类型     ⟷    命题
项       ⟷    证明
函数类型  ⟷    蕴含
积类型    ⟷    合取
和类型    ⟷    析取
多态类型  ⟷    全称量词
依赖类型  ⟷    量词
```

## 5. 指称语义数学基础 (Denotational Semantics)

### 5.1 域理论 (Domain Theory)

**定义 5.1.1** (域)：域 (D, ⊑) 是完全偏序，满足：

```
(1) 有向完备性: 每个有向子集有上确界
(2) 连续性: 函数保持上确界
```

**定义 5.1.2** (提升域 - Lifted Domain)：

```
D⊥ = D ∪ {⊥}

⊥ ⊑ d 对所有 d ∈ D
⊥ 表示未定义/发散
```

**定义 5.1.3** (平坦域)：

```
对于离散集合 A，A⊥ 的序关系为：
  ⊥ ⊑ a 对所有 a ∈ A
  a ⊑ b ⇔ a = b 对 a,b ∈ A
```

### 5.2 连续函数

**定义 5.2.1** (连续函数)：

```
f: D → E 连续  ⇔  f(⊔S) = ⊔f(S) 对所有有向集 S
```

**定理 5.2.1** (连续函数空间形成域)：

```
[D → E] = {f: D → E | f 连续}

序关系: f ⊑ g ⇔ ∀d∈D. f(d) ⊑ g(d)
```

是域当 D,E 是域。

### 5.3 不动点语义

**定义 5.3.1** (不动点算子)：

```
fix: [D → D] → D
fix(f) = ⊔{fⁿ(⊥) | n ≥ 0}
```

**定理 5.3.1** (不动点定理)：

```
设 f: D → D 连续，则：
(1) fix(f) 是 f 的最小不动点
(2) fix(f) = f(fix(f))
(3) 若 d = f(d)，则 fix(f) ⊑ d
```

*证明*：

```
(1) 链 ⊥ ⊑ f(⊥) ⊑ f²(⊥) ⊑ ... 的上确界存在
(2) f(fix(f)) = f(⊔fⁿ(⊥)) = ⊔f(fⁿ(⊥)) = ⊔fⁿ⁺¹(⊥) = fix(f)
(3) 由归纳: ⊥ ⊑ d, fⁿ(⊥) ⊑ d → fⁿ⁺¹(⊥) = f(fⁿ(⊥)) ⊑ f(d) = d
```

∎

### 5.4 指称语义示例

**示例 5.4.1** (While 语言表达式语义)：

```
E: Exp → (State → ℤ⊥)

⟦n⟧σ = n
⟦x⟧σ = σ(x)
⟦e₁ + e₂⟧σ = let n₁ = ⟦e₁⟧σ, n₂ = ⟦e₂⟧σ in n₁ + n₂
```

**示例 5.4.2** (While 语言语句语义)：

```
C: Com → (State → State⊥)

⟦skip⟧σ = σ
⟦x := e⟧σ = σ[x ↦ ⟦e⟧σ]
⟦c₁; c₂⟧σ = ⟦c₂⟧(⟦c₁⟧σ)

⟦if b then c₁ else c₂⟧σ =
  if ⟦b⟧σ then ⟦c₁⟧σ else ⟦c₂⟧σ

⟦while b do c⟧ = fix(λφ.λσ.
  if ⟦b⟧σ then φ(⟦c⟧σ) else σ)
```

### 5.5 完全偏序构造

**定义 5.5.1** (积域与和域)：

```
积域:  D × E = {(d,e) | d∈D, e∈E}
       (d₁,e₁) ⊑ (d₂,e₂) ⇔ d₁ ⊑ d₂ ∧ e₁ ⊑ e₂

和域:  D + E = ({0}×D) ∪ ({1}×E) ∪ {⊥}
       inₗ(d) = (0,d),  inᵣ(e) = (1,e)
       inₗ(d₁) ⊑ inₗ(d₂) ⇔ d₁ ⊑ d₂
       inᵣ(e₁) ⊑ inᵣ(e₂) ⇔ e₁ ⊑ e₂
       ⊥ ⊑ x 对所有 x
```

**定理 5.5.1** (域的代数性质)：

```
[D × E → F] ≅ [D → [E → F]]  (Currying)
[D → E × F] ≅ [D → E] × [D → F]
```
