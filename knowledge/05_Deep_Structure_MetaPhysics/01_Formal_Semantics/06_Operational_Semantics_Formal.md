## 1. 大步操作语义 (Big-step)

### 1.1 基本思想

大步语义（自然语义）直接描述表达式的最终求值结果或语句的最终状态转换。

```
⟨表达式, 状态⟩ ⇓ 值
⟨语句, 状态⟩ ⇓ 新状态
```

### 1.2 算术表达式语义

**语法**：

```
a ::= n | x | a₁ + a₂ | a₁ - a₂ | a₁ * a₂
```

**语义规则**：

```
─────────── (Num)
⟨n, σ⟩ ⇓ n

σ(x) = n
─────────── (Var)
⟨x, σ⟩ ⇓ n

⟨a₁, σ⟩ ⇓ n₁    ⟨a₂, σ⟩ ⇓ n₂    n = n₁ + n₂
─────────────────────────────────────────── (Add)
           ⟨a₁ + a₂, σ⟩ ⇓ n

⟨a₁, σ⟩ ⇓ n₁    ⟨a₂, σ⟩ ⇓ n₂    n = n₁ - n₂
─────────────────────────────────────────── (Sub)
           ⟨a₁ - a₂, σ⟩ ⇓ n

⟨a₁, σ⟩ ⇓ n₁    ⟨a₂, σ⟩ ⇓ n₂    n = n₁ × n₂
─────────────────────────────────────────── (Mult)
           ⟨a₁ * a₂, σ⟩ ⇓ n
```

### 1.3 布尔表达式语义

**语法**：

```
b ::= true | false | a₁ = a₂ | a₁ ≤ a₂ | ¬b | b₁ ∧ b₂
```

**语义规则**：

```
──────────────── (True)
⟨true, σ⟩ ⇓ true

────────────────── (False)
⟨false, σ⟩ ⇓ false

⟨a₁, σ⟩ ⇓ n₁    ⟨a₂, σ⟩ ⇓ n₂
──────────────────────────────── (Eq)
     ⟨a₁ = a₂, σ⟩ ⇓ (n₁ = n₂)

⟨a₁, σ⟩ ⇓ n₁    ⟨a₂, σ⟩ ⇓ n₂
──────────────────────────────── (Leq)
     ⟨a₁ ≤ a₂, σ⟩ ⇓ (n₁ ≤ n₂)

⟨b, σ⟩ ⇓ t
──────────────── (Not)
⟨¬b, σ⟩ ⇓ ¬t

⟨b₁, σ⟩ ⇓ t₁    ⟨b₂, σ⟩ ⇓ t₂
──────────────────────────────── (And)
     ⟨b₁ ∧ b₂, σ⟩ ⇓ (t₁ ∧ t₂)
```

### 1.4 语句语义

**语法**：

```
S ::= skip | x := a | S₁; S₂ | if b then S₁ else S₂ | while b do S
```

**语义规则**：

```
──────────────── (Skip)
⟨skip, σ⟩ ⇓ σ

⟨a, σ⟩ ⇓ n
──────────────────────── (Assign)
⟨x := a, σ⟩ ⇓ σ[x ↦ n]

⟨S₁, σ⟩ ⇓ σ₁    ⟨S₂, σ₁⟩ ⇓ σ₂
─────────────────────────────── (Seq)
      ⟨S₁; S₂, σ⟩ ⇓ σ₂

⟨b, σ⟩ ⇓ true    ⟨S₁, σ⟩ ⇓ σ'
────────────────────────────────── (If-True)
⟨if b then S₁ else S₂, σ⟩ ⇓ σ'

⟨b, σ⟩ ⇓ false    ⟨S₂, σ⟩ ⇓ σ'
────────────────────────────────── (If-False)
⟨if b then S₁ else S₂, σ⟩ ⇓ σ'

⟨b, σ⟩ ⇓ false
──────────────────────────────── (While-False)
⟨while b do S, σ⟩ ⇓ σ

⟨b, σ⟩ ⇓ true    ⟨S, σ⟩ ⇓ σ₁    ⟨while b do S, σ₁⟩ ⇓ σ₂
──────────────────────────────────────────────────────── (While-True)
              ⟨while b do S, σ⟩ ⇓ σ₂
```

### 1.5 非终止的表示

```
若程序不终止，则无推导树：
⟨while true do skip, σ⟩ 无 ⇓ 结果
```

## 2. 小步操作语义 (Small-step)

### 2.1 基本思想

小步语义描述单个计算步骤，展示计算的逐步演化。

```
⟨表达式, 状态⟩ → ⟨表达式', 状态'⟩   或   → 值
```

### 2.2 表达式语义

**规则**：

```
⟨n₁, σ⟩ → n₁    ⟨n₂, σ⟩ → n₂
────────────────────────────────
⟨n₁ + n₂, σ⟩ → n₁ + n₂

⟨a₁, σ⟩ → ⟨a₁', σ⟩
────────────────────────────
⟨a₁ + a₂, σ⟩ → ⟨a₁' + a₂, σ⟩

⟨a₁, σ⟩ → n₁    ⟨a₂, σ⟩ → ⟨a₂', σ⟩
────────────────────────────────────
⟨n₁ + a₂, σ⟩ → ⟨n₁ + a₂', σ⟩
```

### 2.3 语句语义

```
⟨x := n, σ⟩ → σ[x ↦ n]

⟨a, σ⟩ → ⟨a', σ⟩
────────────────────────────
⟨x := a, σ⟩ → ⟨x := a', σ⟩

⟨S₁, σ⟩ → ⟨S₁', σ'⟩        (S₁ 未终止)
────────────────────────────
⟨S₁; S₂, σ⟩ → ⟨S₁'; S₂, σ'⟩

⟨S₁, σ⟩ → σ₁               (S₁ 终止)
────────────────────────────
⟨S₁; S₂, σ⟩ → ⟨S₂, σ₁⟩

⟨b, σ⟩ → true
────────────────────────────────────
⟨if b then S₁ else S₂, σ⟩ → ⟨S₁, σ⟩

⟨b, σ⟩ → false
────────────────────────────────────
⟨if b then S₁ else S₂, σ⟩ → ⟨S₂, σ⟩

──────────────────────────────── (While)
⟨while b do S, σ⟩ →
  ⟨if b then (S; while b do S) else skip, σ⟩
```

### 2.4 反射传递闭包

```
→* 表示零步或多步：

────────── (Refl)
⟨e, σ⟩ →* ⟨e, σ⟩

⟨e, σ⟩ → ⟨e', σ'⟩    ⟨e', σ'⟩ →* ⟨e'', σ''⟩
───────────────────────────────────────────── (Trans)
         ⟨e, σ⟩ →* ⟨e'', σ''⟩
```

### 2.5 大小步等价性

**定理 2.5.1**：

```
⟨e, σ⟩ ⇓ v    ⇔    ⟨e, σ⟩ →* v
⟨S, σ⟩ ⇓ σ'   ⇔    ⟨S, σ⟩ →* σ'
```

## 3. 结构化操作语义 (SOS)

### 3.1 Plotkin 风格 SOS

由 Gordon Plotkin 发展的标准形式，使用推导规则：

```
前提₁ ... 前提ₙ
──────────────── (规则名)
    结论
```

### 3.2 配置 (Configurations)

```
配置类型:
- ⟨S, σ⟩: 待执行语句与当前状态
- σ: 终止状态
- ⟨E, σ⟩: 待求值表达式
- v: 值
- 错误: 运行时错误
```

### 3.3 推导树示例

计算 `⟨x := 1; y := 2, σ⟩`：

```
──────────── (Num)     ──────────── (Num)
⟨1, σ⟩ ⇓ 1             ⟨2, σ[x↦1]⟩ ⇓ 2
────────────           ───────────────────
⟨x:=1, σ⟩ ⇓ σ[x↦1]     ⟨y:=2, σ[x↦1]⟩ ⇓ σ[x↦1, y↦2]
──────────────────────────────────────────────────── (Seq)
      ⟨x:=1; y:=2, σ⟩ ⇓ σ[x↦1, y↦2]
```

### 3.4 并发扩展

```
⟨S₁, σ⟩ → ⟨S₁', σ'⟩
──────────────────────────── (Par-1)
⟨S₁ || S₂, σ⟩ → ⟨S₁' || S₂, σ'⟩

⟨S₂, σ⟩ → ⟨S₂', σ'⟩
──────────────────────────── (Par-2)
⟨S₁ || S₂, σ⟩ → ⟨S₁ || S₂', σ'⟩

──────────────────────────── (Par-Skip)
⟨skip || skip, σ⟩ → σ
```

## 4. 归约语义 (Reduction Semantics)

### 4.1 求值上下文

**定义**：

```
E ::= [] | E + a | n + E | E - a | ...  (表达式上下文)
C ::= [] | C; S | x := E | if E then S₁ else S₂ | ...  (语句上下文)

[] 表示"洞"(hole)，即可归约位置
```

### 4.2 归约规则

```
E[a] → E[a']  当 a → a' 是基本归约
```

**基本归约**：

```
n₁ + n₂ → n  其中 n = n₁ + n₂
(λx.e) v → e[v/x]
```

### 4.3 策略

```
调用按值 (Call-by-Value):
  参数求值后再代入

调用按名 (Call-by-Name):
  参数不求值直接代入

调用按需 (Call-by-Need):
  第一次求值后缓存结果
```

### 4.4  Church-Rosser 性质

**定理 4.4.1** (合流性)：

```
若 e →* e₁ 且 e →* e₂，则 ∃e'. e₁ →* e' 且 e₂ →* e'
```

## 5. 环境语义 (Environment Semantics)

### 5.1 环境定义

```
环境 ρ : Var → Loc    (变量到存储位置的映射)
状态/存储 σ : Loc → Val  (位置到值的映射)

或者合并为:
环境 env : Var ⇀ Val   (偏函数)
```

### 5.2 词法作用域规则

```
⟨x, ρ, σ⟩ ⇓ σ(ρ(x))

⟨let x = e₁ in e₂, ρ, σ⟩ ⇓ v₂
  其中 ⟨e₁, ρ, σ⟩ ⇓ v₁
        l = newloc(σ)
        ⟨e₂, ρ[x↦l], σ[l↦v₁]⟩ ⇓ v₂
```

### 5.3 闭包 (Closure)

```
函数值表示为闭包 ⟨λx.e, ρ⟩:

⟨λx.e, ρ, σ⟩ ⇓ ⟨λx.e, ρ⟩

⟨e₁ e₂, ρ, σ⟩ ⇓ v
  其中 ⟨e₁, ρ, σ⟩ ⇓ ⟨λx.e, ρ'⟩
        ⟨e₂, ρ, σ⟩ ⇓ v₂
        l = newloc(σ)
        ⟨e, ρ'[x↦l], σ[l↦v₂]⟩ ⇓ v
```

### 5.4 递归环境

```
⟨rec f = λx.e, ρ⟩ ⇓ ⟨λx.e, ρ[f↦⟨λx.e, ρ⟩]⟩

或通过不动点:
ρ_rec = μρ'. ρ[f ↦ ⟨λx.e, ρ'⟩]
```

## 6. 存储语义 (Store Semantics)

### 6.1 显式存储模型

```
存储 S : Loc →_{fin} Val  (有限偏函数)
位置 l ∈ Loc
下一个空闲位置: next(S)
```

### 6.2 引用操作

```
分配:
⟨ref v, S⟩ ⇓ ⟨l, S[l ↦ v]⟩  其中 l = next(S)

解引用:
⟨!l, S⟩ ⇓ ⟨S(l), S⟩

赋值:
⟨l := v, S⟩ ⇓ ⟨(), S[l ↦ v]⟩
```

### 6.3 带有存储的表达式语义

```
配置: ⟨e, S⟩ 或 ⟨v, S⟩

⟨!e, S⟩ → ⟨!e', S'⟩    若 ⟨e, S⟩ → ⟨e', S'⟩
⟨!l, S⟩ → ⟨S(l), S⟩

⟨ref e, S⟩ → ⟨ref e', S'⟩  若 ⟨e, S⟩ → ⟨e', S'⟩
⟨ref v, S⟩ → ⟨l, S[l↦v]⟩  l ∉ dom(S)

⟨e₁ := e₂, S⟩ → ⟨e₁' := e₂, S'⟩  若 ⟨e₁, S⟩ → ⟨e₁', S'⟩
⟨v := e₂, S⟩ → ⟨v := e₂', S'⟩    若 ⟨e₂, S⟩ → ⟨e₂', S'⟩
⟨l := v, S⟩ → ⟨(), S[l↦v]⟩
```

### 6.4 存储关系

```
存储等价: S₁ ≈ S₂  如果它们仅在未使用位置不同

存储扩展: S₁ ≤ S₂  如果 S₂ 是 S₁ 的扩展

垃圾回收: 移除不可达位置
```

## 7. 类型环境的操作语义

### 7.1 带类型的配置

```
类型环境 Γ : Var → Type

配置:
- Γ ⊢ ⟨e, σ⟩ : τ    (表达式 e 在状态 σ 下有类型 τ)
- Γ ⊢ ⟨S, σ⟩ : cmd  (语句 S 是良类型的命令)
```

### 7.2 类型保持性 (Preservation)

**定理 7.2.1** (Subject Reduction)：

```
若 Γ ⊢ e : τ 且 ⟨e, σ⟩ → ⟨e', σ'⟩，则 Γ ⊢ e' : τ
```

**定理 7.2.2** (Progress)：

```
若 Γ ⊢ e : τ，则 e 是值或 ∃e'. ⟨e, σ⟩ → ⟨e', σ'⟩
```

### 7.3 类型化的操作规则

```
Γ(x) = τ
────────────────
Γ ⊢ ⟨x, σ⟩ → ⟨σ(x), σ⟩ : τ

Γ ⊢ ⟨e₁, σ⟩ : int    Γ ⊢ ⟨e₂, σ⟩ : int
────────────────────────────────────────
Γ ⊢ ⟨e₁ + e₂, σ⟩ : int

Γ ⊢ ⟨e₁, σ⟩ → ⟨e₁', σ'⟩ : τ₁
────────────────────────────────
Γ ⊢ ⟨e₁ + e₂, σ⟩ → ⟨e₁' + e₂, σ'⟩ : int

n₁ + n₂ = n
────────────────────────
Γ ⊢ ⟨n₁ + n₂, σ⟩ → ⟨n, σ⟩ : int
```

### 7.4 类型环境与存储一致性

```
存储 σ 与类型环境 Γ 一致 (记作 Γ ⊢ σ):

∀x ∈ dom(Γ). σ(x) 是类型 Γ(x) 的值

即: Γ(x) = int ⇒ σ(x) ∈ ℤ
    Γ(x) = bool ⇒ σ(x) ∈ {true, false}
```

## 附录：操作语义对比

| 特性 | 大步语义 | 小步语义 | 归约语义 |
|------|----------|----------|----------|
| 描述粒度 | 最终结果 | 单步演化 | 上下文+归约 |
| 非终止 | 无推导 | 无限归约序列 | 无限归约序列 |
| 并发 | 难以描述 | 容易交错 | 需扩展 |
| 调试性 | 差 | 好 | 好 |
| 实现 | 解释器 | 抽象机 | 优化编译 |
