## 概述

计算复杂性理论是理论计算机科学的核心分支，研究计算问题所需的资源（时间、空间等）与问题难度之间的关系。复杂性类别（Complexity Classes）将计算问题按照解决它们所需的资源量进行分类，为理解计算的本质极限提供了严谨的数学框架。

---

## 1. 基本复杂性类别

### 1.1 P 类（多项式时间）

**定义**：P（Polynomial Time）是所有可以被确定性图灵机在多项式时间内解决的决策问题的集合。

**形式化定义**：

```
P = ∪_{k≥1} DTIME(n^k)
```

**关键特征**：

- **高效可解**：P类问题被认为是"实际可解的"或"高效的"
- **封闭性**：对并、交、补集、多项式时间归约封闭
- **机器无关性**：在不同的合理计算模型下保持一致

**典型问题**：

- 图连通性检测
- 最短路径问题（Dijkstra算法）
- 线性规划（单纯形法、内点法）
- 素性测试（AKS算法）

### 1.2 NP 类（非确定性多项式时间）

**定义**：NP（Nondeterministic Polynomial Time）是所有可以被非确定性图灵机在多项式时间内解决的决策问题的集合，等价于解可以被确定性图灵机在多项式时间内验证的问题集合。

**形式化定义**：

```
NP = ∪_{k≥1} NTIME(n^k)
```

**关键特征**：

- **验证易性**：给定一个候选解，可以在多项式时间内验证其正确性
- 包含P类：P ⊆ NP（是否相等是千禧年难题之一）
- **搜索与验证**：搜索问题可能比验证困难得多

**典型问题**：

- 布尔可满足性问题（SAT）
- 旅行商问题（TSP）
- 图着色问题
- 子集和问题
- 背包问题

### 1.3 PSPACE 类（多项式空间）

**定义**：PSPACE是所有可以被确定性图灵机使用多项式数量空间解决的决策问题的集合。

**形式化定义**：

```
PSPACE = ∪_{k≥1} DSPACE(n^k)
```

**关键特征**：

- **空间资源**：以空间而非时间为主要复杂度度量
- **层次关系**：NP ⊆ PSPACE（因为非确定性多项式时间算法可用多项式空间模拟）
- **完备问题**：TQBF（真值量词布尔公式）是PSPACE完全的

**典型问题**：

- 量化布尔公式（QBF）
- 博弈问题（如国际象棋、围棋的广义版本）
- 正则表达式等价性
- 上下文无关语言成员关系

### 1.4 EXPTIME 类（指数时间）

**定义**：EXPTIME是所有可以被确定性图灵机在指数时间内解决的决策问题的集合。

**形式化定义**：

```
EXPTIME = ∪_{k≥1} DTIME(2^{n^k})
```

**关键特征**：

- **严格包含**：PSPACE ⊆ EXPTIME
- **实际难解**：EXPTIME问题通常被认为在实际中不可解
- **博弈完备性**：许多组合博弈的决策问题EXPTIME完全

**时间层次定理**：
对于时间可构造函数 f(n)，有：

```
DTIME(f(n)) ⊊ DTIME(f(n)^2)
```

这表明复杂性类别形成严格的层次结构。

## 2. 其他重要复杂性类别

### 2.1 概率复杂性类别

| 类别 | 定义 | 特性 |
|------|------|------|
| **BPP** | 有界误差概率多项式时间 | 错误概率 ≤ 1/3，可通过重复降低 |
| **RP** | 随机多项式时间（单侧误差） | 接受时总是正确，拒绝时可能错误 |
| **coRP** | RP的补集 | 拒绝时总是正确，接受时可能错误 |
| **ZPP** | 零误差概率多项式时间 | RP ∩ coRP，期望多项式时间 |

**关系链**：

```
ZPP = RP ∩ coRP
RP ⊆ NP, coRP ⊆ coNP
BPP ⊆ Σ₂^P ∩ Π₂^P
```

### 2.2 计数复杂性类别

**#P（Sharp-P）**：

- 计算NP问题的解的数量
- #P完全问题比NP完全问题更困难
- 典型问题：#SAT（计算满足赋值的数量）

### 2.3 交互式证明系统

| 类别 | 定义 | 能力 |
|------|------|------|
| **IP** | 交互式多项式时间证明 | IP = PSPACE（Shamir 1992） |
| **MIP** | 多证明者交互式证明 | MIP = NEXP |
| **PCP** | 概率可检查证明 | PCP定理：NP = PCP[O(log n), O(1)] |

## 3. 归约理论

### 3.1 多项式时间归约

**定义**：问题A可归约到问题B（记作 A ≤_p B），如果存在多项式时间可计算函数f，使得：

```
x ∈ A ⇔ f(x) ∈ B
```

**归约类型**：

1. **Karp归约（多项式时间映射归约）**：上述标准定义
2. **Cook归约（图灵归约）**：使用B作为预言机解决A
3. **对数空间归约**：使用对数空间计算归约函数

**归约的应用**：

- **难度比较**：若 A ≤_p B，则B至少与A一样难
- **完备性定义**：类别中最难的问题
- **算法设计**：将新问题转化为已知问题

### 3.2 归约示例

**3-SAT ≤_p 独立集**：

- 将布尔公式转化为图
- 每个子句创建三角形，变量创建边
- 公式可满足当且仅当图有大小为m的独立集

**独立集 ≤_p 顶点覆盖**：

- 利用互补关系：S是独立集 ⇔ V\S是顶点覆盖
- 线性时间归约

**哈密顿路径 ≤_p TSP**：

- 构造完全图，原图边权重为1，其他为2
- 存在哈密顿路径 ⇔ TSP最优解为n

## 4. NP完全性理论

### 4.1 Cook-Levin 定理

**定理**：布尔可满足性问题（SAT）是NP完全的。

**证明思路**：

1. SAT ∈ NP（显然，给定赋值可多项式时间验证）
2. 对于任意L ∈ NP，构造从L到SAT的多项式时间归约
3. 利用非确定性图灵机的接受计算历史构造布尔公式

### 4.2 NP完全问题谱系

```
                SAT (Cook-Levin)
                   |
        +----------+----------+
        |          |          |
       3-SAT    图着色    哈密顿路径
        |          |          |
   独立集      顶点覆盖    子集和
        |          |          |
   团问题    支配集     背包问题
        |          |          |
   集合覆盖   分区问题   旅行商问题
```

### 4.3 NP难问题与NP完全问题

| 特征 | NP完全 | NP难 |
|------|--------|------|
| 属于NP | 是 | 不一定 |
| 至少与NP最难问题一样难 | 是 | 是 |
| 示例 | SAT, TSP(决策版) | 停机问题, TSP(优化版) |

### 4.4 常见NP完全问题详解

**3-SAT**：

- 每个子句恰好3个文字的合取范式可满足性
- 是证明其他问题NP完全的常用起点

**顶点覆盖**：

- 给定图G和整数k，是否存在大小不超过k的顶点覆盖
- 与独立集问题互补

**哈密顿回路**：

- 图中是否存在经过每个顶点恰好一次的回路
- 与TSP密切相关

**图着色**：

- 给定图G和k种颜色，是否可用k种颜色正常着色
- 三色问题是NP完全，二色问题在P中

## 5. 复杂性层次结构

### 5.1 多项式层次（Polynomial Hierarchy）

**定义**：

```
Δ₀^P = Σ₀^P = Π₀^P = P
Σ_{k+1}^P = NP^{Σ_k^P}
Π_{k+1}^P = coNP^{Σ_k^P}
Δ_{k+1}^P = P^{Σ_k^P}
PH = ∪_{k≥0} Σ_k^P
```

**层次关系**：

```
P = Δ₁^P ⊆ Σ₁^P = NP ⊆ Δ₂^P ⊆ Σ₂^P ⊆ ... ⊆ PH ⊆ PSPACE
```

**若P = NP，则PH塌陷到P**：
这是复杂性理论中一个重要的条件结果。

### 5.2 空间复杂性层次

```
L ⊆ NL ⊆ P ⊆ NP ⊆ PSPACE = NPSPACE ⊆ EXPTIME ⊆ NEXPTIME ⊆ EXPSPACE
```

**Savitch定理**：

```
NSPACE(f(n)) ⊆ DSPACE(f(n)²)
```

特别地，NPSPACE = PSPACE。

### 5.3 时间-空间权衡

**关键包含关系**：

```
DTIME(f(n)) ⊆ NTIME(f(n)) ⊆ DSPACE(f(n)) ⊆ NSPACE(f(n)) ⊆ DTIME(2^{O(f(n))})
```

**分离结果**：

- L ≠ PSPACE（对角化论证）
- P ≠ EXPTIME（时间层次定理）
- NL ≠ PSPACE

## 6. 电路复杂性

### 6.1 布尔电路模型

**定义**：布尔电路是有向无环图，其中：

- 源点（输入节点）：标记为变量或常量
- 内部节点：布尔门（AND、OR、NOT等）
- 汇点（输出节点）：计算结果

**电路族**：{C_n}_{n≥1}，其中C_n处理n位输入

### 6.2 电路复杂性类别

| 类别 | 定义 | 门限制 |
|------|------|--------|
| **AC⁰** | 常数深度、多项式大小、无界扇入AND/OR/NOT | 深度O(1) |
| **NC¹** | 对数深度、多项式大小、有界扇入 | 深度O(log n) |
| **NC** | Nick's Class，高效并行算法 | 深度O(log^k n)，大小O(n^k) |
| **P/poly** | 多项式大小电路族 | 无深度限制 |

**关系**：

```
AC⁰ ⊊ NC¹ ⊆ NC² ⊆ ... ⊆ NC ⊆ P/poly
```

### 6.3 电路下界结果

**AC⁰下界（Furst-Saxe-Sipser, Ajtai）**：

- PARITY ∉ AC⁰
- AC⁰电路无法计算奇偶校验函数

**电路深度-大小权衡**：

- 深度2电路需要指数大小计算某些函数
- 这暗示了P ≠ NP的可能性

**自然证明障碍（Razborov-Rudich）**：

- 某些证明技术无法分离P和NP
- 需要新的证明技术突破

### 6.4 电路与图灵机的关系

**定理**：P ⊆ P/poly

**逆否命题**：若NP ⊄ P/poly，则P ≠ NP

**Karp-Lipton定理**：
若NP ⊆ P/poly，则PH = Σ₂^P

## 7. 复杂性类别的精细划分

### 7.1 次指数时间

| 类别 | 定义 | 典型算法 |
|------|------|----------|
| **SUBEXP** | ∩_{ε>0} DTIME(2^{n^ε}) | 某些数论算法 |
| **QP** | ∪_{c>0} DTIME(2^{(log n)^c}) | 拟多项式时间 |

### 7.2 参数化复杂性

**FPT（固定参数可解）**：

- 时间复杂度：f(k)·n^c
- k是参数，n是输入大小
- 典型问题：顶点覆盖（FPT），独立集（W[1]-难）

## 8. 现代研究方向

### 8.1 精细复杂性（Fine-Grained Complexity）

研究P类问题之间的相对难度：

- 3SUM猜想
- 强指数时间假设（SETH）
- 全对最短路径猜想（APSP）

### 8.2 量子复杂性

| 类别 | 定义 | 关系 |
|------|------|------|
| **BQP** | 有界误差量子多项式时间 | BPP ⊆ BQP ⊆ PSPACE |
| **QMA** | 量子NP | NP ⊆ QMA ⊆ PSPACE |

Shor算法：整数分解 ∈ BQP，对传统密码学构成威胁。

### 8.3 平均情况复杂性

**DistNP**：分布问题的NP类
研究问题在随机输入上的难度，而非最坏情况。

## 9. 总结

计算复杂性类别构成了理解计算问题难度的系统框架：

1. **基本类别**：P、NP、PSPACE、EXPTIME形成核心层次
2. **归约理论**：建立问题间的相对难度关系
3. **完全性理论**：识别每个类别中的"最难"问题
4. **电路复杂性**：提供非uniform计算模型视角
5. **开放问题**：P vs NP等千禧年难题仍有待解决
