## 核心概念

### 复杂性类别层次

```text
P ⊆ NP ⊆ PSPACE ⊆ EXPTIME ⊆ NEXPTIME
│     │       │          │
│     └─ NP完全(NPC)     │
│            │           │
└─ 多项式时间可解   └─ 指数时间问题
```

### 关键问题

| 问题 | 类别 | 状态 |
|:-----|:-----|:-----|
| P vs NP | 决定性开放问题 | 千禧年大奖难题 |
| 快速排序 | O(n log n) | 最优比较排序 |
| 矩阵乘法 | O(n^2.373) | 理论最优待突破 |
| 布尔可满足性(SAT) | NP完全 | 首个NP完全问题 |
