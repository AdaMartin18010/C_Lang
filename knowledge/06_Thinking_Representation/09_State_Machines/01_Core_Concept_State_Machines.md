## 1. 内存对象生命周期状态机

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        内存对象生命周期状态机                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│                              ┌─────────┐                                     │
│                    ┌────────▶│ 未分配  │◀────────┐                          │
│                    │         │Unallocated       │                          │
│                    │         └────┬────┘         │                          │
│                    │              │ malloc/calloc│                          │
│                    │              ▼              │                          │
│              free  │         ┌─────────┐         │  free                    │
│              释放  │  ┌──────│  已分配 │──────┐  │  后使用                  │
│                    │  │      │Allocated│      │  │  (UAF!)                  │
│                    │  │      └────┬────┘      │  │                          │
│                    │  │           │ 初始化    │  │                          │
│                    │  │           ▼           │  │                          │
│                    │  │      ┌─────────┐      │  │                          │
│                    │  │      │  已初始化│      │  │                          │
│                    │  │      │Initialized     │  │                          │
│                    │  │      └────┬────┘      │  │                          │
│                    │  │           │ 使用      │  │                          │
│                    │  │           ▼           │  │                          │
│                    │  │      ┌─────────┐      │  │                          │
│                    │  └─────▶│  使用中 │◀─────┘  │                          │
│                    │         │In Use   │         │                          │
│                    │         └────┬────┘         │                          │
│                    │              │ free         │                          │
│                    │              ▼              │                          │
│                    │         ┌─────────┐         │                          │
│                    └─────────│  已释放 │─────────┘                          │
│                              │Freed    │                                    │
│                              └────┬────┘                                    │
│                                   │                                          │
│                                   ▼                                          │
│                              ┌─────────┐                                     │
│                              │ 悬挂指针│ (危险状态!)                         │
│                              │Dangling │                                    │
│                              └─────────┘                                     │
│                                                                              │
│  状态转换表:                                                                 │
│  ────────────────────────────────────────────────────────────────────────   │
│  当前状态        │ 事件          │ 下一状态       │ 条件/动作                │
│  ────────────────────────────────────────────────────────────────────────   │
│  未分配          │ malloc        │ 已分配         │ size > 0                 │
│  已分配          │ 初始化        │ 已初始化       │ 赋值操作                 │
│  已初始化        │ 使用          │ 使用中         │ 读写操作                 │
│  使用中          │ free          │ 已释放         │ ptr != NULL              │
│  已释放          │ 访问          │ 悬挂指针(UB)   │ 任何解引用               │
│  ────────────────────────────────────────────────────────────────────────   │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 2. 线程生命周期状态机

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         POSIX线程生命周期状态机                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│                         ┌───────────┐                                       │
│                    ┌───▶│  新建     │                                       │
│                    │    │  NEW      │                                       │
│                    │    └─────┬─────┘                                       │
│                    │          │ pthread_create()                            │
│         pthread_exit│         ▼                                             │
│         或取消     │    ┌───────────┐        ┌───────────┐                  │
│                    └───┬│  就绪     │◀───────│  等待     │                  │
│                        │  READY    │ 调度   │  WAITING  │                  │
│                        └─────┬─────┘        └─────┬─────┘                  │
│                              │                    ▲                         │
│                              │ 获得CPU            │ 等待条件/锁              │
│                              ▼                    │                          │
│                        ┌───────────┐              │                          │
│                        │  运行     │──────────────┘                          │
│                        │ RUNNING   │ 阻塞操作                                 │
│                        └─────┬─────┘                                         │
│                              │                                               │
│              ┌───────────────┼───────────────┐                              │
│              │               │               │                              │
│              ▼               ▼               ▼                              │
│        ┌───────────┐  ┌───────────┐  ┌───────────┐                         │
│        │ 终止      │  │ 等待      │  │ 分离      │                         │
│        │TERMINATED │  │           │  │DETACHED   │                         │
│        └───────────┘  └───────────┘  └───────────┘                         │
│                                                                              │
│  关键转换:                                                                   │
│  • pthread_create(): NEW → READY                                             │
│  • 调度器: READY → RUNNING → READY (时间片用完)                              │
│  • pthread_mutex_lock(): RUNNING → WAITING (锁被占用)                        │
│  • I/O操作: RUNNING → WAITING → READY (I/O完成)                              │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 4. 信号处理状态机

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         信号处理状态机                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│                         ┌──────────┐                                         │
│                         │  默认    │                                         │
│                         │ DEFAULT  │                                         │
│                         └────┬─────┘                                         │
│                              │ signal() / sigaction()                        │
│                              ▼                                               │
│  信号产生              ┌──────────┐                                          │
│       │                │  捕获    │                                          │
│       │                │  HANDLER │                                          │
│       │                └────┬─────┘                                          │
│       │                     │ 执行信号处理函数                               │
│       ▼                     ▼                                               │
│  ┌─────────┐           ┌──────────┐          ┌──────────┐                   │
│  │  挂起   │──────────▶│  执行中  │─────────▶│  返回    │                   │
│  │ PENDING │  递送     │ RUNNING  │ 处理完成 │ RESUME   │                   │
│  └─────────┘           └──────────┘          └────┬─────┘                   │
│       ▲                                           │                          │

│                                                                              │
│  关键注意事项:                                                               │

│                                                                              │
```

## 6. 快速参考

| 状态机 | 关键状态 | 常见错误 |
|:-------|:---------|:---------|
| 内存对象 | 未分配/已分配/使用中/已释放/悬挂 | Use-After-Free |
| 线程 | 新建/就绪/运行/等待/终止 | 死锁、竞态条件 |
| 变量 | 不存在/未初始化/已初始化/使用中/销毁 | 未初始化使用 |
| 信号 | 默认/捕获/挂起/执行中 | 非信号安全函数 |
| TCP连接 | CLOSED/LISTEN/ESTABLISHED/TIME_WAIT | 端口耗尽 |
