﻿# 存储期与生命周期概念映射

> **主题**: 存储期、生命周期与作用域的关系 | **难度**: L3

---

## 概念概述

C语言中变量的三个核心属性：**作用域(Scope)**、**链接属性(Linkage)**、**存储期(Storage Duration)**。

```text
┌─────────────────────────────────────────────────────────────┐
│                    变量声明的三维属性                         │
├─────────────┬─────────────┬─────────────────────────────────┤
│   作用域     │  链接属性    │           存储期                │
├─────────────┼─────────────┼─────────────────────────────────┤
│ 代码可见范围 │ 跨文件可见性 │        内存存在时间              │
├─────────────┼─────────────┼─────────────────────────────────┤
│ 块作用域     │ 外部链接     │  自动存储期(Automatic)          │
│ 文件作用域   │ 内部链接     │  静态存储期(Static)             │
│ 函数作用域   │ 无链接       │  线程存储期(Thread)             │
│ 函数原型作用域│            │  动态分配存储期(Allocated)       │
└─────────────┴─────────────┴─────────────────────────────────┘
```

## 存储期类型映射

| 存储期类型 | 生命周期 | 存储位置 | 初始化 | 关键字 |
|:-----------|:---------|:---------|:-------|:-------|
| **自动存储期** | 块执行期间 | 栈 | 不确定 | `auto` |
| **静态存储期** | 程序运行期 | 数据段 | 零初始化 | `static` |
| **线程存储期** | 线程执行期 | TLS段 | 零初始化 | `_Thread_local` |
| **动态分配期** | 分配至释放 | 堆 | 手动 | `malloc/free` |

## 声明关键字映射

```c
// 存储类别说明符与存储期关系
auto           → 自动存储期 (默认)
static         → 静态存储期
extern         → 静态存储期 (声明外部定义)
register       → 自动存储期 (建议寄存器存储)
_Thread_local  → 线程存储期 (C11)
```

## 完整映射矩阵

| 声明位置 | 存储类别 | 作用域 | 链接属性 | 存储期 | 生命周期 |
|:---------|:---------|:-------|:---------|:-------|:---------|
| 块内，无static | auto | 块 | 无 | 自动 | 块执行期 |
| 块内，static | static | 块 | 无 | 静态 | 程序运行期 |
| 文件内，无static | 隐式extern | 文件 | 外部 | 静态 | 程序运行期 |
| 文件内，static | static | 文件 | 内部 | 静态 | 程序运行期 |
| 函数参数 | auto | 块(函数) | 无 | 自动 | 函数执行期 |

## 常见误区

| 误区 | 正确理解 |
|:-----|:---------|
| `static` 只影响生命周期 | 也影响链接属性(内部链接) |
| `extern` 定义变量 | 仅声明，不分配存储 |
| 局部static初始化为0 | 仅显式初始化为0的会 |
| 所有static都在数据段 | 局部static也在数据段，只是作用域受限 |
