## 二、指针与数组关系

```
数组退化为指针:
┌─────────┐         ┌─────────────────────────────┐
│ int arr │    →    │ arr[0] │ arr[1] │ ... │     │
│  [5];   │         └─────────────────────────────┘
└────┬────┘                      ↑
     └───────────────────────────┘
```

| 表达式 | 类型 | 含义 |
|:-------|:-----|:-----|
| `arr[i]` | int | 第i个元素 |
| `*(arr+i)` | int | 同上 |
| `&arr[i]` | int* | 第i个元素地址 |
| `&arr` | int[*](5) | 整个数组地址 |

**关键区别**: `sizeof(arr)` = 总字节数, `sizeof(ptr)` = 指针大小(4/8)

## 三、指针运算规则

| 运算 | 结果 | 说明 |
|:-----|:-----|:-----|
| `p + n` | 地址 | `p + n * sizeof(*p)` |
| `p - n` | 地址 | 反向移动 |
| `p1 - p2` | 整数 | 元素个数差（需同类型）|
| `p[i]` | 值 | `*(p + i)` 语法糖 |
| `*p` | 值 | 解引用访问 |
| `&x` | 地址 | 取变量地址 |

**非法运算**: `p1 + p2`, `p * n`, `p << n`

## 四、多级指针说明

```
二级指针 (T**):
┌────────┐     ┌────────┐     ┌────────┐
│  pp    │────→│   p    │────→│  data  │
│ int**  │     │ int*   │     │  int   │
└────────┘     └────────┘     └────────┘

常见用途:
- T**: 动态二维数组、修改指针参数、字符串数组
- void**: 泛型句柄、回调函数上下文
```

## 五、函数指针语法

```c
// 语法解析（从右向左读）
int (*fp)(int, int)
// fp是指针，指向接受两个int、返回int的函数

// typedef简化
typedef int (*BinaryOp)(int, int);
BinaryOp ops[] = {add, sub, mul, div};
result = ops[0](a, b);  // 调用add
```

## 六、void*与泛型

| 特性 | 说明 |
|:-----|:-----|
| 与任何指针兼容 | 可隐式转换为/从任何对象指针类型 |
| 不能解引用 | 需要强制类型转换后才能解引用 |
| 不能算术运算 | `void*`不知道指向对象的大小 |

```c
// 泛型交换
void swap(void *a, void *b, size_t size) {
    char temp[size];
    memcpy(temp, a, size);
    memcpy(a, b, size);
    memcpy(b, temp, size);
}
```

## 七、常见错误模式

| 错误类型 | 示例 | 后果 |
|:---------|:-----|:-----|
| 未初始化指针 | `int *p; *p = 10;` | 野指针，UB |
| 空指针解引用 | `*NULL` | 段错误 |
| 悬挂指针 | `free(p); *p = 10;` | 使用已释放内存 |
| 内存泄漏 | 丢失指针未free | 内存耗尽 |
| 重复释放 | `free(p); free(p);` | 双重释放，UB |
| 越界访问 | `p[n]` 超出分配 | UB |
| 返回栈地址 | `return &local;` | 悬挂指针 |

**防御建议**: 初始化置NULL、释放后置NULL、使用前检查、用sizeof
