## 四、能力自测检查点

### 4.1 各阶段自测题

**阶段1自测:**

- [ ] 能独立编写100行以内的程序
- [ ] 理解变量作用域
- [ ] 能使用基本控制结构

**阶段2自测:**

- [ ] 能解释指针与数组的区别
- [ ] 能正确使用malloc/free
- [ ] 能排查简单内存泄漏

**阶段3自测:**

- [ ] 能设计模块化程序结构
- [ ] 能编写可复用的头文件
- [ ] 理解链接过程

**阶段4自测:**

- [ ] 能使用GDB调试复杂问题
- [ ] 能使用Valgrind检测内存错误
- [ ] 能识别常见的UB

**阶段5自测:**

- [ ] 能编写线程安全的代码
- [ ] 能进行基本的性能优化
- [ ] 理解内存模型

**阶段6自测:**

- [ ] 能阅读Linux内核源码
- [ ] 能设计系统级架构
- [ ] 能贡献开源项目
