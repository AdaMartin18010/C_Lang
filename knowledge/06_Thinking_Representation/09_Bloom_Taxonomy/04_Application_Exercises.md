## 1. 编程练习 (20题)

### 1.1 基础编程 (难度 ⭐)

#### 练习 1-5

```
E1: 实现字符串反转函数
    函数原型: char *str_reverse(char *str);
    要求:
    - 原地反转，不使用额外数组
    - 时间复杂度O(n)，空间复杂度O(1)
    - 处理空字符串和NULL输入

评分标准:
    - 优秀: 代码简洁，边界处理完善，有文档注释
    - 良好: 功能正确，有少量边界问题
    - 及格: 基本功能实现
```

```
E2: 实现整数转字符串函数
    函数原型: char *itoa(int value, char *str, int base);
    要求:
    - 支持2-36进制
    - 处理负数和0
    - base为10时处理符号

评分标准:
    - 优秀: 支持所有进制，正确处理INT_MIN
    - 良好: 支持2/8/10/16进制
    - 及格: 仅支持10进制
```

```
E3: 实现安全的字符串拷贝函数
    函数原型: size_t strlcpy(char *dst, const char *src, size_t size);
    要求:
    - 总是以'\0'结尾
    - 返回src的长度(便于检查是否截断)
    - 不同于strncpy的行为

评分标准:
    - 优秀: 完全符合BSD strlcpy语义
    - 良好: 基本安全，返回值可能有差异
    - 及格: 功能类似，有安全漏洞
```

```
E4: 实现动态数组(Vector)
    接口要求:
    - vector_create(), vector_destroy()
    - vector_push_back(), vector_pop_back()
    - vector_get(), vector_set()
    - vector_size(), vector_capacity()

    要求:
    - 自动扩容(通常2倍)
    - 存储任意类型(void*或宏模板)
    - 内存安全，无泄漏

评分标准:
    - 优秀: 完整实现，支持泛型，有迭代器
    - 良好: 基本功能完整，有少量缺陷
    - 及格: 基本功能实现，可能有内存问题
```

```
E5: 实现链表操作
    实现以下函数:
    - list_create(), list_destroy()
    - list_insert_head(), list_insert_tail()
    - list_delete(), list_find()
    - list_reverse()

    要求:
    - 使用双向链表
    - 考虑空链表和单节点情况
    - 所有操作O(1)或O(n)

评分标准:
    - 优秀: 完整双向链表，有哨兵节点，代码优雅
    - 良好: 基本功能正确，可能有边界问题
    - 及格: 单向链表实现，功能基本正确
```
