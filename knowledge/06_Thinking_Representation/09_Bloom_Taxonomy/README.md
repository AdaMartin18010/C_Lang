# 布鲁姆分类学在C语言学习中的应用

> **层级定位**: 06_Thinking_Representation > 09_Bloom_Taxonomy
> **用途**: 基于布鲁姆分类学的学习目标设计

---

## 布鲁姆分类学六层次

```
创建 (Creation)
    ↓
评价 (Evaluation)
    ↓
分析 (Analysis)
    ↓
应用 (Application)
    ↓
理解 (Understanding)
    ↓
记忆 (Remembering)
```

---

## C语言学习层次映射

| 层次 | C语言示例 | 学习活动 |
|:-----|:----------|:---------|
| **记忆** | 记住关键字、语法规则 | 背诵、识别 |
| **理解** | 解释指针概念 | 描述、比较 |
| **应用** | 使用指针解决实际问题 | 编码练习 |
| **分析** | 分析内存泄漏原因 | 调试、分解 |
| **评价** | 评估代码质量 | 代码审查 |
| **创建** | 设计新数据结构 | 项目开发 |

---

## 各层次内容分布

### 记忆层 (L1基础)

- 32个关键字
- 运算符优先级
- 标准库函数原型

### 理解层 (L2)

- 指针与数组关系
- 内存模型
- 编译过程

### 应用层 (L3)

- 链表实现
- 文件操作
- 网络编程基础

### 分析层 (L4)

- 性能分析
- 内存分析
- 代码审查

### 评价层 (L5)

- 架构设计评审
- 代码重构决策
- 技术选型

### 创建层 (L6专家)

- 设计新算法
- 创建工具库
- 系统架构设计
