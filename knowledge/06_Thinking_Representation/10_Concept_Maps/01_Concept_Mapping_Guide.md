## 1. 概念图概述

### 1.1 什么是概念图？

概念图（Concept Map）是由Joseph D. Novak开发的认知工具，用于表示知识概念及其关系：

```
概念图 = 概念节点 + 关系连线 + 连接词

示例:
[指针] ──是──> [变量类型]
  |
  └──存储──> [内存地址]
  |
  └──通过──> [解引用操作] ──访问──> [内存数据]
```

### 1.2 概念图 vs 思维导图

| 特性 | 思维导图 | 概念图 |
|------|---------|--------|
| 结构 | 树状，中心发散 | 网状，多中心 |
| 关系 | 层次关系 | 多种关系类型 |
| 连接词 | 通常无 | 必须有连接词 |
| 目的 | 头脑风暴 | 深度理解 |
| 焦点 | 一个中心主题 | 多个相关概念 |

```
                        [虚拟地址空间]
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
        ▼                     ▼                     ▼
   [内核空间]           [用户空间]            [保留区域]
        │                     │
        │        ┌────────────┼────────────┐
        │        │            │            │
        │        ▼            ▼            ▼
        │   [代码段]    [数据段]     [堆区]
        │   (.text)     (.data)      (heap)
        │      │        /     \         │
        │      │       /       \        │
        │      ▼      ▼         ▼       ▼
        │  [指令] [已初始化] [未初始化] [动态分配]
        │         (.data)   (.bss)   [malloc/free]
        │                                 │
        │        ┌────────────────────────┘
        │        │
        ▼        ▼
   [系统调用] [栈区]
    接口     (stack)
                │
        ┌───────┴───────┐
        │               │
        ▼               ▼
   [局部变量]    [函数参数]
   [返回地址]    [栈帧信息]
```

### 4.3 关键命题

| 概念A | 连接词 | 概念B | 说明 |
|-------|--------|-------|------|
| 代码段 | 存储 | 程序指令 | 只读 |
| 数据段 | 分为 | 已初始化/未初始化 | .data/.bss |
| 堆区 | 通过 | malloc分配 | 向上增长 |
| 栈区 | 存储 | 局部变量 | 向下增长 |
| 堆和栈 | 之间是 | 空闲空间 | 动态调整 |

## 6. 构建方法

### 6.1 构建步骤

```
步骤1: 确定焦点问题
  └─> 选择一个具体的、有意义的问题

步骤2: 头脑风暴概念
  └─> 列出15-25个相关概念
  └─> 使用便签纸便于移动

步骤3: 排序分层
  └─> 从最一般到最具体排列
  └─> 形成层级结构

步骤4: 建立连接
  └─> 用连线连接相关概念
  └─> 添加连接词

步骤5: 寻找交叉连接
  └─> 连接不同分支的概念
  └─> 显示复杂关系

步骤6: 审查完善
  └─> 检查逻辑一致性
  └─> 补充遗漏概念
```

### 6.2 连接词库

| 类别 | 连接词 | 示例 |
|------|--------|------|
| 分类 | 是, 包括, 分为 | [指针]是[变量类型] |
| 组成 | 包含, 由...组成 | [结构体]包含[成员] |
| 过程 | 导致, 产生, 转换为 | [编译]导致[机器码] |
| 属性 | 有, 具有, 特征为 | [数组]有[固定大小] |
| 关系 | 依赖于, 影响, 使用 | [指针运算]依赖于[类型] |
