| B3 | char | 本质上是 | 整数类型 | 存储字符的ASCII码 |
| B4 | int | 通常与 | 机器字长 | 32位系统32位，64位系统32位 |
| B5 | float | 精度约为 | 6-7位有效数字 | 单精度浮点 |
| B6 | double | 精度约为 | 15-16位有效数字 | 双精度浮点 |

| 编号 | 概念A | 连接词 | 概念B | 说明 |

| U3 | enum | 定义 | 命名常量集合 | enum Color {RED, GREEN} |
| U4 | typedef | 创建 | 类型别名 | typedef unsigned int uint |

| C3 | 显式转换 | 使用 | 强制类型转换运算符 | (type)expression |

## 6. 常见误区澄清

| 误区 | 正确理解 |
|------|---------|
| 数组是指针 | 数组名在表达式中退化为指针，但数组不是指针 |
| 指针大小与类型有关 | 指针大小固定(4/8字节)，与指向类型无关 |
| void*可以解引用 | void*必须先转换为具体类型指针 |
| typedef创建新类型 | typedef只创建别名，不是新类型 |
| union同时存储多个值 | union同一时刻只能使用一个成员 |
| enum是整数类型 | enum本质是int，不是独立类型 |
| sizeof是函数 | sizeof是运算符，不是函数 |
