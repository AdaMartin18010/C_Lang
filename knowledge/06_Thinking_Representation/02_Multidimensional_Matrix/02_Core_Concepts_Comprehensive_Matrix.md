```
├──────────┬──────────┬──────────┬──────────┬──────────┬──────────┬───────────┤
│ 函数     │ 初始化   │ 对齐保证 │ 数组支持 │ 错误处理 │ 可重入性 │ 性能特征  │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼───────────┤
│ malloc   │ No       │ No       │ No       │ NULL     │ Yes      │ 快        │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼───────────┤
│ calloc   │ Zero     │ No       │ Yes      │ NULL     │ Yes      │ 较慢      │

├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼───────────┤

│ alloc    │          │          │          │          │          │           │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼───────────┤
│ posix_   │ No       │ Yes      │ No       │ NULL     │ Yes      │ 页对齐    │
│ memalign │          │          │          │          │          │           │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼───────────┤
│ alloca   │ No       │ No       │ No       │ 栈溢出   │ No       │ 极快      │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼───────────┤
│ mmap     │ No       │ Yes      │ No       │ MAP_FAIL │ Yes      │ 大块内存  │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┴───────────┘

选择决策树:
小对象(<4KB) → malloc
需要清零 → calloc
数组(n*size) → calloc(n, size)
需要特定对齐 → aligned_alloc
栈上小对象 → alloca(注意栈溢出!)
大内存映射 → mmap
详细解析: [内存分配对比矩阵](../02_Comparison_Matrices/04_Memory_Allocation_Matrix.md)
```

## 4. 并发同步原语对比

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                      同步原语八维对比矩阵 (POSIX)                              │
├─────────────┬──────────┬──────────┬──────────┬──────────┬────────────────────┤
│ 原语        │ 类型     │ 阻塞方式 │ 超时支持 │ 进程共享 │ 典型应用场景       │
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ mutex       │ 互斥     │ 阻塞     │ 是       │ 可选     │ 临界区保护         │
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ rwlock      │ 读写     │ 阻塞     │ 否       │ 可选     │ 读多写少场景       │
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ spinlock    │ 自旋     │ 忙等     │ N/A      │ 否       │ 短临界区/中断上下文│
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ semaphore   │ 计数     │ 阻塞     │ 是       │ 是       │ 资源池管理         │
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ cond_var    │ 条件     │ 阻塞     │ 是       │ 可选     │ 等待条件满足       │
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ barrier     │ 屏障     │ 阻塞     │ 是       │ 可选     │ 阶段同步点         │
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ atomic_ops  │ 原子     │ 无       │ N/A      │ 部分     │ 计数器/标志位      │
├─────────────┼──────────┼──────────┼──────────┼──────────┼────────────────────┤
│ RCU         │ 读拷贝   │ 非阻塞   │ N/A      │ 内核     │ 读多写少+高性能    │
└─────────────┴──────────┴──────────┴──────────┴──────────┴────────────────────┘

关键选择因素:
• 临界区长度: 短→spinlock, 长→mutex
• 读写比例: 读>>写→rwlock/RCU
• 上下文: 中断处理→spinlock(非阻塞)
• 进程间同步→semaphore(命名)/共享mutex
详细解析: [同步机制对比矩阵](../02_Comparison_Matrices/02_Synchronization_Matrix.md)
```

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                      字符串函数五维安全对比矩阵                                │
├──────────┬──────────┬──────────┬──────────┬──────────┬──────────────────────┤
│ 函数     │ 安全等级 │ 目标缓冲 │ 源限制   │ 返回值   │ 推荐使用场景         │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strcpy   │ ❌ 危险  │ 无边界   │ 无       │ dest     │ 永不使用             │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strncpy  │ ⚠️ 中等 │ 有       │ 有       │ dest     │ 需手动补0, 易错      │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strcpy_s │ ✅ 安全  │ 有       │ 有       │ errno    │ C11边界检查, 推荐    │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strcat   │ ❌ 危险  │ 无边界   │ 无       │ dest     │ 永不使用             │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strncat  │ ⚠️ 中等 │ 有       │ 有       │ dest     │ 需计算剩余空间       │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strlcpy  │ ✅ 安全  │ 有       │ 有       │ src_len  │ BSD风格, 推荐        │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ snprintf │ ✅ 安全  │ 有       │ 格式化   │ 长度     │ 格式化输出, 最推荐   │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strdup   │ ⚠️ 中等 │ 动态分配 │ 无       │ ptr/NULL │ 需检查返回值并释放   │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ strndup  │ ✅ 安全  │ 动态分配 │ 有       │ ptr/NULL │ 有限长度复制, 推荐   │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────────────────┘

安全使用原则:
1. 优先使用 _s 后缀函数或 snprintf
2. 始终检查返回值
3. 确保目标缓冲区足够大
4. 动态分配后记得释放
```
