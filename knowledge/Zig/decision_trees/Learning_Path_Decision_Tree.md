### 1.1 C 程序员快速路径 (2周)

```text
第1周：差异对比
├── 语法对比 (1天)
│   ├── 变量声明：int x = 5; → const x: i32 = 5;
│   ├── 指针：int* p → *i32 p
│   ├── 数组：int arr[10] → var arr: [10]i32
│   └── 结构体：struct Point { ... } → const Point = struct { ... }
├── 错误处理 (1天)
│   ├── 错误码 → !i32 错误联合
│   ├── 返回码检查 → try/catch
│   └── 资源清理 → defer/errdefer
├── 内存管理 (2天)
│   ├── malloc/free → Allocator
│   ├── 显式分配器传递
│   └── 内存安全检查
└── 构建系统 (1天)
    ├── Makefile → build.zig
    ├── 交叉编译简化
    └── 包管理

第2周：项目迁移
├── 选择一个C项目
├── 逐步用Zig替换模块
└── 混合项目配置
```

第1周：概念映射

    ├── 声明宏 → comptime函数

## 3. 错误处理策略决策树

```text
┌─────────────────────────────────────────────────────────────────────┐
│                    函数可能失败？                                    │
└─────────────────────────────────────────────────────────────────────┘
                              │
                    ┌─────────┴─────────┐
                    │                   │
                   是                  否
                    │                   │
                    ▼                   ▼
        ┌──────────────────┐   ┌──────────────┐
        │ 定义错误集       │   │ 返回普通类型 │
        │ const MyError =  │   └──────────────┘
        │   error{A, B};  │
        └──────────────────┘
                    │
        ┌───────────┴───────────┐
        │                       │
   调用者需要处理？        调用者只传播？
        │                       │
   ┌────┴────┐            ┌────┴────┐
   │         │            │         │
  是        否            是        否
   │         │            │         │
   ▼         ▼            ▼         ▼
┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│使用    │ │使用    │ │使用    │ │使用    │
│try     │ │catch   │ │try     │ │try     │
│传播    │ │默认值  │ │返回    │ │+       │
│错误    │ │处理    │ │错误    │ │errdefer│
└────────┘ └────────┘ └────────┘ └────────┘
```

### 详细决策规则

| 场景 | 推荐写法 | 示例 |
|------|----------|------|
| 简单传播 | `try` | `const f = try openFile(path);` |
| 默认值 | `catch` | `const port = getPort() catch 8080;` |
| 多错误处理 | `switch` | `catch \|err\| switch(err) {...}` |
| 资源清理 | `errdefer` | `errdefer file.close();` |
| 可选错误 | `if` | `if (mayFail()) \|val\| {...} else \|e\| {...}` |
| 断言成功 | `catch unreachable` | `try init() catch unreachable;` |
