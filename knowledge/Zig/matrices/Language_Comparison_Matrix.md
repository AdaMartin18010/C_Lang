## 10. Zig vs 其他语言：特定场景推荐矩阵

| 场景 | 首选 | 次选 | 不推荐 | 理由 |
|------|------|------|--------|------|
| **操作系统内核** | Zig | C | Rust(学习曲线) | 裸机支持、简单语义 |
| **嵌入式 MCU** | Zig | C | C++(复杂) | 无运行时、精确控制 |
| **系统工具/CLI** | Zig | Go | Python(慢) | 单二进制、快速启动 |
| **WebAssembly** | Zig | Rust | C++(体积) | 体积小、直接编译 |
| **游戏引擎核心** | Zig | C++ | Rust(借用检查) | 性能、C++生态 |
| **网络服务** | Go | Zig | C++(开发效率) | 并发模型、库生态 |
| **加密/安全** | Rust | Zig | C(不安全) | 内存安全优先 |
| **快速原型** | Python | Go | Zig(显式) | 开发速度 |
| **教学/学习** | Zig | Python | C++(复杂) | 概念清晰 |
| **C代码重构** | Zig | Rust | Go(差异大) | C兼容性 |
