## 7. 统计方法说明

### 数据收集方法

```
数据来源类型:
├── 官方安全公告 (Android Security Bulletin, MSRC)
├── CVE数据库 (NVD, CVE.org)
├── 项目安全博客和报告
├── 学术研究论文
└── 第三方安全分析机构

统计口径说明:
• 内存安全漏洞定义: 与内存管理相关的安全缺陷
  - 缓冲区溢出/下溢
  - Use-After-Free
  - 双重释放
  - 未初始化内存使用
  - 空指针解引用
  - 内存泄漏 (仅当导致DoS时计入)

• 排除项:
  - 逻辑错误
  - 配置问题
  - 权限绕过
  - 信息泄露 (非内存相关)
```
