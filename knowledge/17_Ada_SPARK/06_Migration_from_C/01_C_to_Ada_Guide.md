﻿# C到Ada迁移指南

## 1. 迁移策略概述

### 1.1 为什么要迁移？

```
从C迁移到Ada的典型驱动力:

安全关键项目升级:
├── 需要DO-178C/ISO 26262认证
├── 形式化验证需求
├── 降低运行时错误风险
└── 长期维护成本优化

技术债务偿还:
├── 消除缓冲区溢出漏洞
├── 改进代码可维护性
├── 增强类型安全
└── 简化并发编程

新项目选择:
├── 安全关键新系统
├── 长期生命周期产品
└── 高可靠性要求
```

### 1.2 迁移策略选择

| 策略 | 描述 | 适用场景 | 风险 | 时间 |
|------|------|----------|------|------|
| **大爆炸** | 完全重写 | 小型项目 (<50K LOC) | 高 | 短 |
| **分模块** | 逐个模块替换 | 中型项目 | 中 | 中 |
| **分层迁移** | C→Ada→SPARK | 大型项目 | 低 | 长 |
| **绿场开发** | 仅新功能用Ada | 遗留系统 | 低 | 持续 |

**推荐策略：分层渐进式迁移**

```
阶段1: 封装与互操作 (1-3个月)
├── 保持C代码运行
├── 添加Ada绑定层
└── 建立混合编译环境

阶段2: 边缘模块替换 (3-6个月)
├── 工具函数迁移
├── 数据结构重构
└── 单元测试建立

阶段3: 核心模块迁移 (6-12个月)
├── 业务逻辑重写
├── SPARK验证引入
└── 系统集成测试

阶段4: 优化与验证 (3-6个月)
├── 性能调优
├── 形式化验证
└── 认证准备
```

## 2. 语法结构迁移

### 2.1 基本结构对比

| C结构 | Ada等效 | 说明 |
|-------|---------|------|
| `#include` | `with` | 包导入 |
| `#define` | `constant` | 常量定义 |
| `int main()` | `procedure Main` | 程序入口 |
| `void func()` | `procedure Proc` | 过程 |
| `int func()` | `function Func return Int` | 函数 |
| `struct` | `record` | 结构体 |
| `enum` | `type is (...)` | 枚举 |
| `typedef` | `type is new` | 类型别名 |

### 2.2 Hello World对比

**C版本**:

```c
#include <stdio.h>

#define GREETING "Hello, World!"

int main(void) {
    printf("%s\n", GREETING);
    return 0;
}
```

**Ada版本**:

```ada
with Ada.Text_IO;

procedure Hello is
   Greeting : constant String := "Hello, World!";
begin
   Ada.Text_IO.Put_Line (Greeting);
end Hello;
```

### 2.3 控制流迁移

**C控制流**:

```c
// if-else
if (x > 0) {
    printf("positive");
} else if (x < 0) {
    printf("negative");
} else {
    printf("zero");
}

// for循环
for (int i = 0; i < 10; i++) {
    process(i);
}

// while循环
while (count < 100) {
    count++;
}

// switch
case (cmd) {
    case 'A': action_a(); break;
    case 'B': action_b(); break;
    default: error();
}
```

**Ada等效**:

```ada
-- if-elsif-else
if X > 0 then
   Put_Line ("positive");
elsif X < 0 then
   Put_Line ("negative");
else
   Put_Line ("zero");
end if;

-- for循环（更安全）
for I in 0 .. 9 loop
   Process (I);
end loop;

-- while循环
while Count < 100 loop
   Count := Count + 1;
end loop;

-- case（必须穷尽所有可能）
case Cmd is
   when 'A' => Action_A;
   when 'B' => Action_B;
   when others => Error;  -- others必须
end case;
```

## 3. 类型系统迁移

### 3.1 基本类型映射

| C类型 | Ada类型 | 注意 |
|-------|---------|------|
| `char` | `Character` | 或 `Interfaces.C.char` |
| `short` | `Interfaces.C.short` | 精确匹配 |
| `int` | `Interfaces.C.int` | 精确匹配 |
| `long` | `Interfaces.C.long` | 精确匹配 |
| `float` | `Interfaces.C.C_float` | 精确匹配 |
| `double` | `Interfaces.C.double` | 精确匹配 |
| `unsigned` | `Interfaces.C.unsigned` | 精确匹配 |
| `size_t` | `Interfaces.C.size_t` | 精确匹配 |
| `void*` | `System.Address` | 地址类型 |
| `bool` | `Boolean` | 预定义类型 |

### 3.2 强类型转换

**C（弱类型，危险）**:

```c
typedef float Meters;
typedef float Seconds;

Meters distance = 100.0f;
Seconds time = 10.0f;

// 危险！单位不匹配但编译通过
float speed = distance / time;  // 单位混乱
```

**Ada（强类型，安全）**:

```ada
type Meters is new Float;
type Seconds is new Float;
type Meters_Per_Second is new Float;

Distance : Meters := 100.0;
Time     : Seconds := 10.0;

-- 编译错误！类型不匹配
-- Speed : Meters_Per_Second := Distance / Time;

-- 正确做法：显式转换
Speed : Meters_Per_Second :=
   Meters_Per_Second (Float (Distance) / Float (Time));
```

### 3.3 范围约束迁移

**C（无范围检查）**:

```c
// 只能依赖命名约定和注释
typedef unsigned char Percentage;  // 希望是0-100

Percentage p = 150;  // 允许！危险
```

**Ada（自动范围检查）**:

```ada
-- 编译器强制执行范围
subtype Percentage is Integer range 0 .. 100;

P : Percentage := 50;     -- OK
-- P := 150;              -- 编译错误！
-- P := P + 60;           -- 运行时错误（可能溢出）
```

### 3.4 结构体迁移

**C结构体**:

```c
typedef struct {
    int x;
    int y;
    char name[32];
} Point;

Point p = {10, 20, "origin"};
printf("Point %s at (%d,%d)\n", p.name, p.x, p.y);
```

**Ada记录**:

```ada
type Point is record
   X, Y : Integer;
   Name : String (1 .. 32);
end record;

P : Point := (X => 10, Y => 20, Name => (others => ' '));
-- 或使用位置参数
P2 : Point := (10, 20, "origin              ");

Put_Line ("Point " & P.Name & " at (" & P.X'Image & "," & P.Y'Image & ")");
```

## 4. 指针到访问类型的转换

### 4.1 基本概念对比

| C指针 | Ada访问类型 | 说明 |
|-------|-------------|------|
| `int *p` | `access Integer` | 基本访问 |
| `const int *p` | `access constant Integer` | 只读访问 |
| `int *const p` | 不适用 | 访问常量（少用） |
| `void *p` | `System.Address` | 泛型地址 |
| `NULL` | `null` | 空访问值 |
| `p->field` | `P.Field` | 成员访问 |
| `*p` | `P.all` | 解引用 |
| `p++` | 不支持 | Ada禁止指针运算 |

### 4.2 简单指针迁移

**C代码**:

```c
int value = 42;
int *ptr = &value;
printf("Value: %d\n", *ptr);
*ptr = 100;
```

**Ada代码**:

```ada
declare
   Value : aliased Integer := 42;  -- aliased允许取地址
   Ptr   : access Integer := Value'Access;
begin
   Put_Line ("Value: " & Ptr.all'Image);
   Ptr.all := 100;
end;
```

### 4.3 动态内存分配迁移

**C代码**:

```c
int* create_array(size_t n) {
    int* arr = malloc(n * sizeof(int));
    if (arr == NULL) return NULL;
    for (size_t i = 0; i < n; i++) {
        arr[i] = 0;
    }
    return arr;
}

void destroy_array(int* arr) {
    free(arr);
}
```

**Ada代码（使用池化访问类型）**:

```ada
package Dynamic_Arrays is
   type Int_Array is array (Natural range <>) of Integer;
   type Int_Array_Access is access Int_Array;

   function Create_Array (N : Natural) return Int_Array_Access;
   -- 自动内存管理或显式释放
end Dynamic_Arrays;

package body Dynamic_Arrays is
   function Create_Array (N : Natural) return Int_Array_Access is
   begin
      return new Int_Array (1 .. N);  -- 自动初始化为0
   exception
      when Storage_Error =>
         return null;
   end Create_Array;
end Dynamic_Arrays;
```

### 4.4 链表迁移

**C链表**:

```c
typedef struct Node {
    int data;
    struct Node* next;
} Node;

Node* create_node(int data) {
    Node* node = malloc(sizeof(Node));
    node->data = data;
    node->next = NULL;
    return node;
}

void insert(Node** head, int data) {
    Node* new_node = create_node(data);
    new_node->next = *head;
    *head = new_node;
}
```

**Ada链表（使用受控类型自动管理）**:

```ada
with Ada.Finalization;

package Linked_Lists is
   type Node is private;
   type List is tagged private;

   procedure Insert (L : in out List; Data : Integer);
   function Head (L : List) return Integer;
   function Is_Empty (L : List) return Boolean;

private
   type Node_Record;
   type Node_Access is access Node_Record;

   type Node_Record is record
      Data : Integer;
      Next : Node_Access;
   end record;

   type List is new Ada.Finalization.Controlled with record
      Head : Node_Access;
      Size : Natural := 0;
   end record;

   overriding procedure Finalize (L : in out List);  -- 自动释放
end Linked_Lists;
```

### 4.5 函数指针迁移

**C函数指针**:

```c
typedef int (*Comparator)(const void*, const void*);

int compare_int(const void* a, const void* b) {
    int ia = *(const int*)a;
    int ib = *(const int*)b;
    return (ia > ib) - (ia < ib);
}

void sort(void* arr, size_t n, size_t size, Comparator cmp);
```

**Ada（使用泛型替代）**:

```ada
generic
   type Element_Type is private;
   with function "<" (L, R : Element_Type) return Boolean is <>;
package Generic_Sort is
   type Array_Type is array (Positive range <>) of Element_Type;

   procedure Sort (Arr : in out Array_Type);
end Generic_Sort;

-- 实例化
package Int_Sort is new Generic_Sort (
   Element_Type => Integer
);

Numbers : Int_Sort.Array_Type := (3, 1, 4, 1, 5);
Int_Sort.Sort (Numbers);  -- 类型安全
```

## 5. 宏到泛型的转换

### 5.1 常量宏迁移

**C宏**:

```c
#define MAX_SIZE 100
#define PI 3.14159
#define VERSION "1.0.0"
```

**Ada常量**:

```ada
Max_Size : constant := 100;
Pi       : constant := 3.14159;
Version  : constant String := "1.0.0";
```

### 5.2 函数式宏迁移

**C宏（危险）**:

```c
#define MAX(a,b) ((a) > (b) ? (a) : (b))
#define SQUARE(x) ((x) * (x))

int m = MAX(5, 3);        // OK
int s = SQUARE(i++);      // 危险！i++执行两次
```

**Ada泛型函数**:

```ada
generic
   type Element_Type is private;
   with function ">" (L, R : Element_Type) return Boolean is <>;
function Generic_Max (A, B : Element_Type) return Element_Type;

function Generic_Max (A, B : Element_Type) return Element_Type is
begin
   if A > B then
      return A;
   else
      return B;
   end if;
end Generic_Max;

-- 实例化
function Int_Max is new Generic_Max (Integer);
function Float_Max is new Generic_Max (Float);

M : Integer := Int_Max (5, 3);  -- 安全，无重复求值
```

### 5.3 泛型容器迁移

**C宏容器（代码膨胀，类型不安全）**:

```c
#define DEFINE_STACK(TYPE, NAME, SIZE) \
    typedef struct { \
        TYPE data[SIZE]; \
        int top; \
    } NAME##_Stack; \
    void NAME##_push(NAME##_Stack* s, TYPE val) { \
        s->data[s->top++] = val; \
    }

DEFINE_STACK(int, Int, 100)
DEFINE_STACK(float, Float, 100)
```

**Ada泛型包（类型安全，无重复代码）**:

```ada
generic
   type Element_Type is private;
   Max_Size : in Positive;
package Generic_Stack is
   type Stack is limited private;

   Overflow  : exception;
   Underflow : exception;

   procedure Push (S : in out Stack; Item : Element_Type)
      with Pre => not Is_Full (S) or else raise Overflow;

   procedure Pop (S : in out Stack; Item : out Element_Type)
      with Pre => not Is_Empty (S) or else raise Underflow;

   function Is_Empty (S : Stack) return Boolean;
   function Is_Full (S : Stack) return Boolean;

private
   type Element_Array is array (1 .. Max_Size) of Element_Type;

   type Stack is record
      Data : Element_Array;
      Top  : Natural range 0 .. Max_Size := 0;
   end record;
end Generic_Stack;

-- 实例化
package Int_Stack is new Generic_Stack (Integer, 100);
package Float_Stack is new Generic_Stack (Float, 100);
```

## 6. 逐步迁移方法

### 6.1 建立互操作层

**C头文件（保持不变）**:

```c
// legacy_math.h
#ifndef LEGACY_MATH_H
#define LEGACY_MATH_H

double compute_average(const double* values, size_t count);
int validate_input(int min, int max, int value);

#endif
```

**Ada绑定（自动或手写）**:

```ada
-- legacy_math_binding.ads
with Interfaces.C;

package Legacy_Math_Binding is
   use Interfaces.C;

   function Compute_Average (
      Values : access double;
      Count  : size_t
   ) return double
   with
      Import => True,
      Convention => C,
      External_Name => "compute_average";

   function Validate_Input (
      Min_Value : int;
      Max_Value : int;
      Value     : int
   ) return int
   with
      Import => True,
      Convention => C,
      External_Name => "validate_input";

   -- Ada友好包装
   function Safe_Average (Values : Double_Array) return Double;
end Legacy_Math_Binding;
```

### 6.2 模块替换流程

```
单模块迁移步骤:

第1步: 分析
├── 识别模块边界
├── 列出外部依赖
├── 确定接口契约
└── 评估迁移难度

第2步: 创建绑定
├── 生成Ada规范文件(.ads)
├── 测试绑定正确性
└── 建立回归测试

第3步: 重写实现
├── 编写Ada包体(.adb)
├── 保持接口不变
├── 添加类型安全
└── 增加合约注解

第4步: 验证
├── 单元测试通过
├── 集成测试通过
├── SPARK验证（可选）
└── 性能基准通过

第5步: 切换
├── 移除C绑定
├── 直接使用Ada实现
└── 监控运行情况
```

### 6.3 混合代码组织

```
项目结构示例:

my_project/
├── src/
│   ├── c_legacy/           # 未迁移C代码
│   │   ├── utils.c
│   │   └── io.c
│   │
```

## 7. 常见问题与解决

### 7.1 数组索引差异

**问题**: C从0开始，Ada通常从1开始

**C代码**:

```c
int arr[10];
for (int i = 0; i < 10; i++) {
    arr[i] = i;  // 0到9
}
```

**Ada代码（可选从0开始）**:

```ada
-- 方案1: 保持0索引
Arr : array (0 .. 9) of Integer;
for I in 0 .. 9 loop
   Arr (I) := I;
end loop;

-- 方案2: Ada惯用法（推荐）
Arr : array (1 .. 10) of Integer;
for I in Arr'Range loop
   Arr (I) := I - 1;  -- 或调整逻辑
end loop;
```

### 7.2 字符串处理

**C代码**:

```c
char name[32];
strcpy(name, "John");
printf("Hello %s\n", name);
```

**Ada代码**:

```ada
-- 定长字符串
Name : String (1 .. 32) := (others => ' ');
Name (1 .. 4) := "John";
Put_Line ("Hello " & Name);

-- 或使用无界字符串
with Ada.Strings.Unbounded;
use Ada.Strings.Unbounded;

Name : Unbounded_String := To_Unbounded_String ("John");
Put_Line ("Hello " & To_String (Name));
```

### 7.3 错误处理

**C代码**:

```c
FILE* fp = fopen("data.txt", "r");
if (fp == NULL) {
    fprintf(stderr, "Cannot open file\n");
    return -1;
}
// 使用文件...
fclose(fp);
```

**Ada代码**:

```ada
with Ada.Text_IO;

procedure Read_File is
   File : Ada.Text_IO.File_Type;
begin
   Ada.Text_IO.Open (File, Ada.Text_IO.In_File, "data.txt");
   -- 使用文件...
   Ada.Text_IO.Close (File);
exception
   when Ada.Text_IO.Name_Error =>
      Put_Line (Standard_Error, "Cannot open file");
      -- 处理错误
end Read_File;
```
