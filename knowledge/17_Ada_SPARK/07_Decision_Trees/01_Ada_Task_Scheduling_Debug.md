### 4.2 Ravenscar设计模式

```
Ravenscar 兼容设计

┌──────────────┐
│ Periodic_Task│

│     Next_Time := Next_Time + Period;
│   end loop;  │

事件驱动任务模式:
┌──────────────┐
│ Event_Task   │
│   loop       │
│     PO.Entry_Call;  -- 阻塞等待
│     Handle_Event;
│   end loop;  │
└──────────────┘

┌──────────────┐
│ Protected_PO │
│   entry Safe_Entry
│     when Condition is

│   end Safe_Entry;
│              │
│   procedure Proc
│     -- 互斥操作
│ end Protected_PO;
```

## 7. 快速决策参考表

### 7.1 调度问题速查

| 症状 | 可能原因 | 解决方案 |
|------|----------|----------|
| 任务从不执行 | 优先级太低/被饿死 | 调整优先级，检查高优先级任务是否无限循环 |
| 任务执行不规律 | 优先级反转 | 使用 Priority Ceiling |
| 任务响应延迟 | 上下文切换过多 | 合并任务或调整周期 |
| 死锁 | 锁顺序不一致 | 实施全局锁顺序 |
| 数据损坏 | 缺少同步 | 使用 Protected Object |

### 7.2 工具选择速查

| 问题类型 | 推荐工具 | 替代方案 |
|----------|----------|----------|
| 静态分析 | gnatprove | CodePeer |
| 运行时跟踪 | GNAT.Traceback | 自定义日志 |
| 性能分析 | gprof/gnatmetric | AdaProfiler |
| 内存调试 | gnatmem | Valgrind |
| 任务可视化 | GPS Tasks视图 | 文本输出 |
