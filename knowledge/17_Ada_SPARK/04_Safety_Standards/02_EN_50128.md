﻿# EN 50128 铁路应用-通信、信号和处理系统

## 1. 标准概述

### 1.1 什么是EN 50128？

EN 50128是欧洲铁路信号系统软件安全标准，是IEC 61508在铁路领域的应用：

| 属性 | 说明 |
|------|------|
| **全称** | Railway applications - Communication, signalling and processing systems - Software for railway control and protection systems |
| **版本** | EN 50128:2011 (当前) / EN 50716:2023 (更新) |
| **适用范围** | 铁路控制和保护系统软件 |
| **SIL等级** | SIL0-SIL4 (SIL4为最高安全完整性等级) |
| **与CENELEC关系** | 与EN 50129 (系统)、EN 50159 (通信) 配套使用 |

### 1.2 SIL等级定义

| SIL等级 | 每小时危险失效概率 | 应用示例 |
|---------|-------------------|----------|
| **SIL 0** | - | 非安全相关软件 |
| **SIL 1** | 10⁻⁶ to 10⁻⁵ | 乘客信息系统 |
| **SIL 2** | 10⁻⁷ to 10⁻⁶ | 列车监控系统 |
| **SIL 3** | 10⁻⁸ to 10⁻⁷ | 联锁系统 |
| **SIL 4** | 10⁻⁹ to 10⁻⁸ | 列车控制(ETCS) |


### 4.2 SPARK应用示例

```ada
-- EN 50128 SIL 4 联锁系统示例

package Interlocking with SPARK_Mode is
   -- 轨道区段状态
   type Track_Section is (Clear, Occupied, Reserved);

   -- 信号状态
   type Signal_Aspect is (Red, Yellow, Green);

   -- 道岔位置
   type Point_Position is (Normal, Reverse, Moving);

   -- 设置信号 (SIL 4关键功能)
   procedure Set_Signal (
      Signal_Id : in Signal_ID;
      Aspect    : in Signal_Aspect
   ) with
      Global => (Input  => (Track_State, Point_State),
                 Output => Signal_State),
      Pre => Aspect /= Green or else
             (Is_Route_Clear (Signal_Id) and
              Are_Points_Locked (Signal_Id)),
      Post => Get_Signal_Aspect (Signal_Id) = Aspect;

   -- 检查区段空闲
   function Is_Section_Clear (Section : Track_Section_ID) return Boolean
      with Global => Track_State;

private
   type Track_Array is array (Track_Section_ID) of Track_Section;
   type Signal_Array is array (Signal_ID) of Signal_Aspect;
   type Point_Array is array (Point_ID) of Point_Position;

   Track_State  : Track_Array := (others => Clear);
   Signal_State : Signal_Array := (others => Red);
   Point_State  : Point_Array := (others => Normal);
end Interlocking;

package body Interlocking with SPARK_Mode is
   procedure Set_Signal (
      Signal_Id : in Signal_ID;
      Aspect    : in Signal_Aspect
   ) is
   begin
      -- 前置条件确保只有安全状态才能设置绿灯
      Signal_State (Signal_Id) := Aspect;
   end Set_Signal;

   function Is_Section_Clear (Section : Track_Section_ID) return Boolean is
   begin
      return Track_State (Section) = Clear;
   end Is_Section_Clear;
end Interlocking;
```
