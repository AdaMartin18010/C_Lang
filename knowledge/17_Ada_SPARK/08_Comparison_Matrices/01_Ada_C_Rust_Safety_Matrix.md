## 2. 具体安全机制对比

### 2.1 内存安全机制

| 机制 | C | Ada | SPARK | Rust |
|------|---|-----|-------|------|
| **边界检查** | ❌ 无 | ✅ 数组索引检查 | ✅ 证明无越界 | ✅ 索引检查 |
| **缓冲区溢出防护** | ❌ 无 | ✅ Range检查 | ✅ 完全消除 | ✅ Slice检查 |
| **释放后使用防护** | ❌ 无 | ⚠️ 访问检查 | ✅ 无访问类型 | ✅ 生命周期检查 |
| **双重释放防护** | ❌ 无 | ⚠️ 可控类型 | ✅ 无堆分配 | ✅ 所有权转移 |
| **内存泄漏防护** | ❌ 无 | ✅ 垃圾回收可选 | ✅ 静态分配 | ✅ RAII+所有权 |
| **栈溢出防护** | ⚠️ 栈保护页 | ⚠️ 栈检查 | ✅ 证明栈使用 | ⚠️ 栈保护页 |

**代码示例对比：数组安全**

```c
// C - 不安全
int arr[10];
int x = arr[15];  // 未定义行为，无错误
```

```ada
-- Ada - 运行时检查
Arr : array (1 .. 10) of Integer;
X : Integer := Arr (15);  -- Constraint_Error raised
```

```ada
-- SPARK - 证明安全
procedure Access_Array (I : Index_Range) with
  Pre => I in Index_Range
is
   X : Integer := Arr (I);  -- 证明无运行时错误
begin
   null;
end;
```

```rust
// Rust - 运行时检查（或编译时如可能）
let arr = [0; 10];
let x = arr[15];  // panic: index out of bounds
```

### 2.2 类型系统安全

| 特性 | C | Ada | SPARK | Rust |
|------|---|-----|-------|------|
| **强类型** | ❌ 弱类型 | ✅ 强类型 | ✅ 强类型 | ✅ 强类型 |
| **类型推导** | ❌ 无 | ⚠️ 有限 | ⚠️ 有限 | ✅ 强大 |
| **子类型** | ❌ 无 | ✅ 范围子类型 | ✅ 约束子类型 | ⚠️ 新类型模式 |
| **泛型/模板** | ❌ 宏不安全 | ✅ 泛型安全 | ✅ 泛型安全 | ✅ 泛型+trait |
| **代数类型** | ❌ 无 | ⚠️ 变体记录 | ⚠️ 受限 | ✅ 强大enum |
| **空值安全** | ❌ NULL | ⚠️ 访问可为null | ✅ 无null | ✅ Option<T> |

**类型安全代码对比**

```c
// C - 类型不安全
typedef float Meters;
typedef float Seconds;
Meters m = 100.0;
Seconds s = m;  // 允许！单位混淆
```

```ada
-- Ada - 强类型分离
type Meters is new Float;
type Seconds is new Float;
M : Meters := 100.0;
S : Seconds := M;  -- 编译错误！
```

```rust
// Rust - 新类型模式
type Meters = f32;
type Seconds = f32;
let m: Meters = 100.0;
let s: Seconds = m;  // 编译错误！
```

### 2.3 并发安全机制

| 机制 | C | Ada | SPARK | Rust |
|------|---|-----|-------|------|
| **语言级并发** | ❌ 库实现 | ✅ 任务内建 | ✅ Ravenscar | ✅ 标准库 |
| **数据竞争防护** | ❌ 无 | ⚠️ 需正确使用 | ✅ 可证明 | ✅ 编译时检查 |
| **死锁检测** | ❌ 无 | ❌ 运行时 | ⚠️ 可分析 | ❌ 运行时 |
| **优先级继承** | ❌ 无 | ✅ 支持 | ✅ 支持 | ⚠️ 库支持 |
| **实时确定性** | ⚠️ 依赖实现 | ✅ Ravenscar | ✅ Ravenscar | ⚠️ 非确定性GC |

**并发代码对比**

```c
// C - pthread，无安全保证
pthread_mutex_t lock;
int counter = 0;

void* thread_fn(void* arg) {
    pthread_mutex_lock(&lock);
    counter++;  // 忘记解锁会怎样？
    pthread_mutex_unlock(&lock);
}
```

```ada
-- Ada - Protected Object
protected Counter is
   procedure Increment;
   function Get return Natural;
private
   Value : Natural := 0;
end Counter;

protected body Counter is
   procedure Increment is
   begin
      Value := Value + 1;  -- 自动互斥
   end Increment;
   -- 自动解锁
end Counter;
```

```rust
// Rust - 编译时数据竞争检查
use std::sync::Mutex;

let counter = Mutex::new(0);
{
    let mut num = counter.lock().unwrap();
    *num += 1;
} // 自动解锁

// 以下代码编译错误：
// let mut num = counter.lock().unwrap();
// let mut num2 = counter.lock().unwrap(); // 死锁！编译器警告
```
