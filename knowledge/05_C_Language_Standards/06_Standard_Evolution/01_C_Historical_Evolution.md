# C 语言历史演进

> **层级**: L3 (核心层)  
> **目标**: 了解 C 语言从 K&R 到 C23 的发展历程

---

## 1. K&R C (1978)

原始 C 语言、无函数原型、隐式 int。

---

## 2. C89/C90

ANSI 标准化、函数原型、标准库、void、const。

---

## 3. C99

变长数组、单行注释、stdint、复数、内联。

---

## 4. C11/C17/C23

多线程、原子操作、Unicode、constexpr、安全函数。

---

## 5. 总结

C 语言在保持简洁的同时不断现代化。
