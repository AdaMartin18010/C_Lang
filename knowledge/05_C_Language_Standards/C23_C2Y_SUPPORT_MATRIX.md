## 使用建议

### 编译器选择指南

| 场景 | 推荐编译器 | 标准选项 | 原因 |
|------|------------|----------|------|
| 最新C23特性 | GCC 15 | `-std=c23` | 完整支持 |
| C2y实验特性 | GCC 15 或 Clang 20 | `-std=c2y` | 两者都支持 |
| 跨平台兼容性 | Clang 20 | `-std=c17` | 最广泛支持 |
| 最大性能 | GCC 15 | `-O3 -march=native` | 优化更强 |
| 最佳诊断 | Clang 20 | `-std=c23 -Wall -Wextra` | 错误信息更清晰 |

### 特性检测宏参考

```c
/* C标准版本检测 */
#if __STDC_VERSION__ >= 202311L
    // C23+ 代码
#elif __STDC_VERSION__ >= 201710L
    // C17 代码
#elif __STDC_VERSION__ >= 201112L
    // C11 代码
#elif __STDC_VERSION__ >= 199901L
    // C99 代码
#else
    // C89/C90 代码
#endif

/* 特定特性检测 */
#ifdef __has_feature
    #if __has_feature(c_auto_type)
        // 支持 auto
    #endif
    #if __has_feature(c_defer)
        // 支持 defer
    #endif
#endif

/* GCC 特性 */
#ifdef __GNUC__
    #if __GNUC__ >= 15
        // GCC 15+ 特性
    #endif
#endif

/* Clang 特性 */
#ifdef __clang__
    #if __clang_major__ >= 20
        // Clang 20+ 特性
    #endif
#endif
```
