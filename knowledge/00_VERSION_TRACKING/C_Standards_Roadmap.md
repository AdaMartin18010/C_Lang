## C2y (未来标准) 预览

### 提案追踪

| 提案 | 作者 | 状态 | 描述 |
|------|------|------|------|
| N3226 | WG14 | Active | 标准草案工作文件 |
| Enhanced constexpr | Various | 讨论中 | 允许更多编译时计算 |
| Memory safety | Various | 早期 | 借鉴 Rust/Zig 的安全特性 |
| Modules | Various | 早期 | 替代头文件系统 |

### 与 Zig 的趋同

```
C2y 可能借鉴的 Zig 特性:
├── 显式错误处理 (类似 Zig 的错误联合)
├── 更好的可选类型支持
├── 编译时计算增强 (类似 comptime)
├── 分配器显式传递模式
└── 与 C++ 模块的互操作
```
