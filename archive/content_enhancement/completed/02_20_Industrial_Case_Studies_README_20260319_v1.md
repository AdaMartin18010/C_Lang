# 工业案例研究

本目录收集真实的工业级案例研究，展示C语言在高端硬件项目中的实际应用。

## 案例概览

| 文件 | 案例 | 描述 |
|------|------|------|
| `01_Rocket_Chip_Case_Study.md` | Rocket Chip | UC Berkeley开发的RISC-V处理器生成器 |
| `02_Linux_on_RISCV.md` | Linux on RISC-V | RISC-V架构上的Linux操作系统移植 |
| `03_Baremetal_Embedded.md` | 裸机嵌入式 | 无操作系统嵌入式C编程实践 |
| `04_SIMD_Optimization_Case.md` | SIMD优化 | 向量化指令集性能优化案例 |
| `05_Cache_Optimization_Real_World.md` | 缓存优化 | 实际项目中的缓存性能优化 |

## 学习目标

- 了解开源RISC-V生态系统的架构
- 学习操作系统移植的核心技术
- 掌握裸机编程的最佳实践
- 理解SIMD指令在性能优化中的应用
- 分析真实场景中的缓存优化策略

## 前置知识

建议先完成 `10_ISA_Machine_Code` 和 `18_FPGA_Labs` 目录的学习。
