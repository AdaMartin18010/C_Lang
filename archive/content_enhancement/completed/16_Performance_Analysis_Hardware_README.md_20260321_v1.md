# 硬件性能分析

本目录介绍硬件层面的性能分析技术，包括性能监控单元（PMU）架构、Linux perf工具使用、自顶向下分析方法等。

## 内容概览

| 文件 | 描述 |
|------|------|
| `01_PMU_Architecture.md` | CPU性能监控单元架构详解 |
| `02_Linux_Perf_Tutorial.md` | Linux perf性能分析工具完整教程 |
| `03_Top_Down_Analysis_Method.md` | Intel自顶向下性能分析方法 |
| `04_Microarchitecture_Benchmarking.md` | 微架构级基准测试技术 |

## 学习目标

- 理解硬件性能计数器的工作原理
- 掌握Linux perf工具的实际使用
- 学会使用自顶向下方法定位性能瓶颈
- 能够设计微架构级基准测试
